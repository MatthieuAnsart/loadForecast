CPLEX Optimization Studio accessibility

Lists keyboard shortcuts in CPLEX Optimization Studio to provide accessibility to those who cannot easily use a mouse or other pointing device.

Accessibility features help a user who has a physical disability, such as restricted mobility or limited vision, to use software products successfully.

CPLEX Optimization Studio provides accessibility features though its user interface, which provides screen-reader accessible textual displays and a keyboard-only interface.

There are many keyboard shortcuts in CPLEX Optimization Studio to improve accessibility for those who cannot easily use a mouse or other pointing device. This section lists some of them.

Note:

You can get a complete list of all keyboard shortcuts in the IDE by typing Crtl+Shift+L.
