Accessibility in the online IBM Information Center documentation

This section describes some of the accessibility features available when using the online IBM Information Center documentation.

Using a screen reader with the information center

The information center has been tested using the screen reader JAWS 4.5 with Internet Explorer 6.

Font sizes

The font sizes in the information center are set using relative font sizes. You can change them using functions available in your browser.

Keyboard shortcuts

Use the following key combinations to navigate the help system by keyboard. Note that there might be a slight variation in supported key combinations, depending on which browser you are using. Always check the browser help information if you find that the combinations below do not work.

  • To go to the next link, button or topic from inside a frame (page), press Tab.

  • To expand and collapse a node in the navigation tree, press the Right and Left arrows.

  • To move to the next node, press the Down arrow or Tab.

  • To move to the previous node, press the Up arrow or Shift+Tab.

  • To scroll all the way up or down, press Home or End.

  • To go back, press Alt+Left arrow; to go forward press Alt+Right arrow.

  • To go to the next frame, press Ctrl+Tab.

  • To move to previous frame, press Shift+Ctrl+Tab.

  • To print the current page or active frame, press Ctrl+P.

Alternate text for images

All images in the information center that convey additional information have alternate text. If you move your mouse pointer over the image, the alternate text can be viewed or read by a screen reader.
