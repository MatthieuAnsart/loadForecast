Uninstalling CPLEX Optimization Studio

Describes how to remove CPLEX Optimization Studio from your system.

The preferred method to remove IBM ILOG CPLEX Optimization Studio on Windows is to use the Windows control panel. For all other platforms, you must remove the application by directly running the uninstallation feature provided by the installer.

Uninstalling using the Windows control panel

During installation, the install wizard registers with Windows such that the application can be removed using the Windows control panel. If multiple copies or versions of CPLEX Optimization Studio are present on your system, this procedure will only remove the most recently installed copy; other copies can be removed by directly running their respective installers.

How to remove CPLEX Optimization Studio using the Windows control panel

  1. Exit the CPLEX Optimization Studio application and any other applications that might be using files in the CPLEX Optimization Studio installation directory.

  2. Click Start > Control Panel.

  3. In the Windows Control Panel, click Add or Remove Programs.

  4. In the Add or Remove Programs panel, select the entry for IBM ILOG CPLEX Optimization Studio and click the Change/Remove button to start the uninstallation wizard.

  5. Select Uninstall, click Next and follow the instructions in the wizard.

Uninstalling by directly running the installer

Use this procedure to uninstall on non-Windows platforms or to uninstall additional copies or versions of the application on Windows.

  1. Exit the CPLEX Optimization Studio application and any other applications that might be using files in the CPLEX Optimization Studio installation directory.

  2. Locate the uninstallation command:

    • On UNIX:

      <CPLEX_OPTIMIZATION_STUDIO_HOME>/Uninstall/Uninstall

      where <CPLEX_OPTIMIZATION_STUDIO_HOME> is the directory in which the application is installed.

    • On Windows:

      <CPLEX_OPTIMIZATION_STUDIO_HOME>/Uninstall/Uninstall.exe

      where <CPLEX_OPTIMIZATION_STUDIO_HOME> is the directory in which the application is installed.

      A shortcut is available using the Start menu: Start > All Programs > IBM ILOG > CPLEX Optimization Studio X.x > Uninstall > Uninstall

  3. If you intend to run the uninstallation process from a command prompt, make sure that your current directory is not within the <CPLEX_OPTIMIZATION_STUDIO_HOME> directory tree as this might interfere with the uninstallation process.

  4. Uninstall the application by running the Uninstall command.

  5. Select Uninstall, click Next and follow the instructions in the wizard.
