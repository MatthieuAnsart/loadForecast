Enumeration SolutionTargetType

Definition file: ilcplex/ilocplexi.h
Deprecated: Since V12.6.3

The enumeration IloCplex::SolutionTargetType lists the values that the solution target parameter IloCplex::SolutionTarget can assume in an instance of IloCplex. Use these values with the method IloCplex::setParam(IloCplex::SolutionTarget, value) when you set solution target.

With the default setting of IloCplex::SolutionTarget, IloCplex tries to compute an optimal solution of a convex problem. For a discussion about various settings, refer to the CPLEX User's Manual.

See the CPLEX Parameters Reference Manual and the information about these parameters. Also see the CPLEX User's Manual for examples of their use.
