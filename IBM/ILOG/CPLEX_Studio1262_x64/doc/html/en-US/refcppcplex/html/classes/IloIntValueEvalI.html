Class IloIntValueEvalI

Definition file: ilcp/cp.h
The implementation class for a value evaluator in a search strategy.

A evaluator of integer values is an object that is used by value selectors (IloValueSelector) to define instantiation strategies. An evaluator of integer variable value assignments is defined by implementing the pure virtual member function IloNum IloIntValueEvalI::eval(IloCP cp, IloIntVar x, IloInt value) that returns an evaluation of the instantiation of x to value.
