Class IloCplex::Parameter

Definition file: ilcplex/ilocplexi.h

The class IloCplex::Parameter lists the types of parameters available (Boolean, integer, long integer, numeric, string). For constructors and methods to manipulate parameters, see the documentation of each of those types in this reference manual. For documentation of the use and values of a parameter, see the CPLEX Parameter Reference Manual. For examples of the use of parameters, see the CPLEX User's Manual.
