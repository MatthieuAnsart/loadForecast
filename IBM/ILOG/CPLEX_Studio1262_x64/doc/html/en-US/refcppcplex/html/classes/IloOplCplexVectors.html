Class IloOplCplexVectors

Definition file: ilopl/ilooplmodel.h
Utility for CPLEX warm start of MIP problems.

An instance of this class can be used to save MIP start information for one instance of IloCplex and restore it on other instances of IloCplex.

See Also:

IloCplex::setStart, IloAlgorithm::getValue
