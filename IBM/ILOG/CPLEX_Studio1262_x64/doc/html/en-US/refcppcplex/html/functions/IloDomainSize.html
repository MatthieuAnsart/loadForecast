IloDomainSize

public IloIntVarEval IloDomainSize(IloEnv env)
Definition file: ilcp/cp.h
Returns an evaluator which evaluates the number of possible values of a domain.

This function returns a variable evaluator. This evaluator returns the number of elements in the current domain of the evaluated variable.
