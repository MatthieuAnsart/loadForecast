012: // From Bradley, Hax, Magnanti, Applied Mathematical Programming, 
013: // Chapter 9, Exercise 24
014: 
015: //  - A Custom Pilot Chemical Company is a chemical manufacturer that produces
016: // batches of specialty chemicals to order. Principal equipment consists of
017: // eight interchangable reactor vessels, five interchangeable distillation
018: // columns, four large interchangeable centrifuges, and a network of switchable
019: // piping and storage tanks. Customer demand comes in the form of orders for
020: // batches of one or more specialty chemicals, normally to be delivered
021: // simultaneously for further use by the customer.
022: 
023: // - An order consists of a set of jobs. Each job has an optional precedence
024: // requirement, arrival week of the job, duration of the job in weeks, the week
025: // that the job is due, the number of reactors required, distillation columns
026: // required, and centrifuges required.
027: 
028: // - Find a schedule of the orders and jobs to minimize the completion time of all
029: // orders
030: 
031: 
032: using CP;
033: 
034: tuple paramsT{
035:         int NbReactors;
036:         int NbColumns;
037:         int NbCentrifuges;
038: };
039: paramsT Params = ...;
040: 
041: int NbReactors    = Params.NbReactors;
042: int NbColumns     = Params.NbColumns;
043: int NbCentrifuges = Params.NbCentrifuges;
044: 
045: tuple JobIndex {
046:   string ordernumber;
047:   int    jobnum;
048: };
049: 
050: tuple JobInfo {
051:   int jobprec;
052:   int arrival;
053:   int duration;
054:   int weekdue;
055:   int reactors;
056:   int columns;
057:   int centrifuges;
058: };
059: 
060: tuple JobData {
061:   JobIndex ind;
062:   JobInfo  info;
063: };
064: {JobData} jobs = ...;
065: 
066: {JobIndex} joblist = { i | <i,j> in jobs };
067: assert ( card(joblist) == card(jobs) );
068: 
069: JobInfo datarray[joblist];
070: 
071: execute {
072:   for(var j in jobs)
073:     datarray[j.ind] = j.info;
074: };
075: 
076: dvar interval a[j in joblist] 
077:   in datarray[j].arrival..datarray[j].weekdue 
078:   size datarray[j].duration;
079: 
080: cumulFunction reactors    = sum (j in joblist) pulse(a[j], datarray[j].reactors);
081: cumulFunction columns     = sum (j in joblist) pulse(a[j], datarray[j].columns);
082: cumulFunction centrifuges = sum (j in joblist) pulse(a[j], datarray[j].centrifuges);
083: 
084: minimize max(j in joblist) endOf(a[j]);
085: subject to {
086:   forall (j in joblist) {
087:     if (datarray[j].jobprec > 0) {
088:       endBeforeStart(a[<j.ordernumber,datarray[j].jobprec>], a[j]);
089:     }
090:   }
091:   reactors    <= NbReactors;
092:   columns     <= NbColumns;
093:   centrifuges <= NbCentrifuges;
094: };
095: 
096: execute {
097:   for(var j in joblist) {
098:     writeln(j + " on [" + a[j].start + "," + a[j].end + ")");
099:   }
100: }
101: 
102: tuple solutionT{
103:         JobIndex idx;
104:         int start;
105:         int end;
106: };
107: {solutionT} solution = {<j, startOf(a[j]), endOf(a[j])> | j in joblist};
108: execute{
109:         writeln(solution);
110: }
