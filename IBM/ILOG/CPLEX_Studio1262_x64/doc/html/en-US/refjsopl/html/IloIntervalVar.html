Class IloIntervalVar 

IloIntervalVar
Example:
dvar interval x in 2..20 size 10;

subject to {
  executed( x ) == 1;
}

execute {
  writeln(x);  
  writeln(x.start," ",x.end);
}
After execution|the Scripting log shows:
<1 2 12 10 10>
2 12
