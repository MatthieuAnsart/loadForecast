Class IloTuple 

IloTuple
Example:
tuple T {
  int i;
  string s;
}

T t1=<1,"one">;
execute {
  t1.i=2;
  t1.s="two";
  writeln("t1 = ",t1);
}
After execution the Scripting log shows:
t1 =  <2 "two">

Default behavior.
