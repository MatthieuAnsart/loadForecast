Class Opl 

Opl
Description:
Using the Opl namespace you can have access to OPL functions (such as item(), ord(), ...) in a scripting block. You can find more information on OPL modelling functions in the OPL document
Language Quick Reference.
Example:
int i = 10;
execute {
   var resu = Opl.abs( i );
}
