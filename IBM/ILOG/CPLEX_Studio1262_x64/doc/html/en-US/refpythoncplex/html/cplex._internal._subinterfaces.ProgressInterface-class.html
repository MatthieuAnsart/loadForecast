get_num_iterations(self)

 

Returns the number of iterations executed so far.

Example usage:

>>> import cplex
>>> c = cplex.Cplex()
>>> out = c.set_results_stream(None)
>>> out = c.set_log_stream(None)
>>> c.read("example.mps")
>>> c.solve()
>>> int(c.solution.progress.get_num_iterations())
2
