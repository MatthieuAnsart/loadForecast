Macro IloFloatVar

Definition file: ilconcert/iloexpression.h
IloFloatVar
An instance of this class represents a constrained floating-point variable in Concert Technology.

An instance of this class represents a constrained floating-point variable in Concert Technology. It is not for use with CP Optimizer.

Most member functions in this class contain assert statements. For an explanation of the macro NDEBUG (a way to turn on or turn off these assert statements), see the concept Assert and NDEBUG.
