Class IlcIntervalVarArray

Definition file: ilcp/cpext.h
Include file: <ilcp/cpext.h>
The array class for the class IlcIntervalVar.

The class IlcIntervalVarArray is the class for an array of instances of IlcIntervalVar. Instances of this class are arrays indexed from 0.

IlcIntervalVarArray

public IlcIntervalVarArray(const IloCP cp, const IloInt size)

This constructor creates an array of length size initialized to empty handles. The argument size must be greater than 0 (zero). The index range of the array is [0 size). Each element of the array should be assigned before using the array.

operator[]

public IlcIntervalVar & operator[](IlcInt index) const

This subscripting operator returns a reference to a constrained interval variable corresponding to the given index in the invoking array of constrained integer variables.

The assignment of an interval to an index position of the invoking array is not reversible.
