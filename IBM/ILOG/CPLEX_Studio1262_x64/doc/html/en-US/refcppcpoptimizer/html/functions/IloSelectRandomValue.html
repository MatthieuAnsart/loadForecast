IloSelectRandomValue

public IloValueSelector IloSelectRandomValue(IloEnv env)
Definition file: ilcp/cp.h
Returns a selector which selects a value randomly.

This function returns a selector of integer variable value assignments that selects a domain value randomly. It selects only one value.
