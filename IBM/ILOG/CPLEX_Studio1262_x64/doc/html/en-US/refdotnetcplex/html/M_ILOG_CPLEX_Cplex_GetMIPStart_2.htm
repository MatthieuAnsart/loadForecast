Parameters

mipstartindex
Type: System..::.Int32
The index of the MIP start among all the MIP starts associated with the current problem.
vars
Type: array< ILOG.Concert..::.INumVar >[]()[]
An array of numeric variables to define the MIP start. This array is an input argument. It will not be modified by this method. It may be null; if it is not null, then none of its elements may be null.
values
Type: array< System..::.Double >[]()[]
The values corresponding to the variables. This array is an output argument. Its values will be modified by this method. Upon return of the method, the array will contain the start values of the variables specified in the array vars. If the array vars is not null, then the array values must not be null either. For variables in vars that are specified in the MIP start, the respective slot in vals will be set to 0 (zero).
isset
Type: array< System..::.Boolean >[]()[]
An array of Boolean values that specify whether or not the corresponding variable is set in the MIP start. This array is an output argument. Its values will be modified by the method. Upon return of the method, the array will contain a value of true or false for the variables specified by vars depending on whether the variable is part of the MIP start or not.

Return Value

The level of effort associated with the MIP start.
