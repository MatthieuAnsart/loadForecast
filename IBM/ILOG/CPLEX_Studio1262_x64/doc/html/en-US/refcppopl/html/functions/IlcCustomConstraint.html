IlcCustomConstraint

public IlcConstraint IlcCustomConstraint(IloCP cp, IloPropagatorI * prop)
Definition file: ilcp/cpext.h
Include file: <ilcp/cpext.h>
Creates a constraint from a propagator.

This function creates a constraint from a propagator. You can then add this constraint to a IloCP instance during search in a user-defined goal. For example, using the propagator defined in the IloPropagatorI documentation: 

       // Code to be executed inside a goal!
       IloPropagatorI * prop = new (env) SignI(x, sgn);
       IlcConstraint signCt = IlcCustomConstraint(cp, prop);
       cp.add(signCt);
