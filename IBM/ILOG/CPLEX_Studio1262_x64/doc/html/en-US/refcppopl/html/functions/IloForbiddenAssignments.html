Definition file: ilconcert/ilotupleset.h
Defines simple constraints that are not predefined.

This function can be used to define simple constraints based on explicitly giving the forbidden assignments for a small group of variables. The function creates and returns a constraint for use in an IBM ILOG Concert Technology model. That constraint is defined for all the constrained variables in the array vars.

The forbidden assignments are specified by a tuple set or integer array. The argument set specifies the combinations of forbidden values of the variables. In the case of a single variable being constrained, the array values determines the forbidden values.

The order of the constrained variables in the array vars is important because the same order is respected in the tuple set set.

To avoid exceptions, you must observe the following condition:

  • The size of vars must be the same as the arity of the set. If the two or three variables are passed directly (not in an array), the arity of the tuple set must be two or three as appropriate.
