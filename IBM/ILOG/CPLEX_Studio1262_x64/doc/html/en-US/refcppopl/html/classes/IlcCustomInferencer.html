Class IlcCustomInferencer

Definition file: ilcp/cpext.h
The handle class for a custom inferencer.

The class IlcCustomInferencer represents custom inferencers in the default search.

For more information on custom inferencers, see the concept Custom inferencer in CP Optimizer.
