IlcLexicographic

public IlcConstraint IlcLexicographic(IlcIntVarArray x, IlcIntVarArray y)
Definition file: ilcp/cpext.h
Include file: <ilcp/cpext.h>
Returns a lexicographic constraint.

The IlcLexicographic function returns a constraint which maintains two arrays to be lexicographically ordered.

More specifically, IlcLexicographic(x, y) maintains that x is less than or equal to y in the lexicographical sense of the term. This means that either both arrays are equal or that there exists i < size(x) such that for all j < i, x[j] = y[j] and x[i] < y[i].

Note that the size of the two arrays must be the same.
