Class IloSynchronize

Definition file: ilconcert/ilosmodel.h
An instance of this class represents a synchronization constraint.

An instance of IloSynchronize represents a synchronization constraint. A synchronization constraint between an interval variable a and a set of interval variables {b1,...,bn} makes all present intervals in {b1,...,bn} start and end together with interval a, if it is present.

You must add this constraint to a model and extract the model to an algorithm in order for it to be taken into account.

Note: This constraint cannot be used in a logical constraint.

For more information on the synchronization constraint, see the concept Constraints on groups of interval variables in CP Optimizer.
