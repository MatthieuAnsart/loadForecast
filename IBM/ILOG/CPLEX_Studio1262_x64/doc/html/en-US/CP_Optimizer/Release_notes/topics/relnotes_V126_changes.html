Changes since CP Optimizer V12.5.1

Describes the changes for the release of the CP Optimizer feature of CPLEX® Optimization Studio V12.6.0.

  • New feature: Strong constraint
    A new constraint for strengthening the model has been introduced.
  • New feature: Same-sequence constraints and same-common-subsequence constraints
    New constraints have been added on sequence variables.
  • New feature: Export model
    A method for exporting the model has been introduced.
  • New feature: Failure-directed search
    A new parameter controls usage of failure directed search.
  • Change: Default TimeMode parameter
    The default value of the TimeMode parameter changed in this version of CP Optimizer.
  • Conversion notes: Deprecated methods and parameters
    The PropagationLog parameter has been deprecated. One of the push methods has been deprecated.
  • Resolved issues
    These are the issues that are fixed in the CP Optimizer feature of CPLEX Optimization Studio V12.6.0.
