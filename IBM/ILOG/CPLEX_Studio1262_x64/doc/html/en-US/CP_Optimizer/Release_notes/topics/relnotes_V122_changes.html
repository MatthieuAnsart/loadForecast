Changes since CP Optimizer V2.3

Describes the changes for the release of CP Optimizer V12.2.

  • Transition based scheduling
    Transition based scheduling has been introduced in this version of CP Optimizer.
  • Extensions User’s Manual
    The CP Optimizer Extensions User’s Manual details how to add custom constraints, propagators and search.
  • Scheduling search API
    Classes and functions for customizing search for scheduling problems have been added for this version of CP Optimizer.
  • Custom inferencer
    Custom inferencers have been introduced in this version of CP Optimizer.
  • New default value for the RestartGrowthFactor parameter
    The default value of the RestartGrowthFactor parameter has changed.
  • Resolved issues
    These are the registered issues that are fixed in CP Optimizer V12.2 since the previous release (CP Optimizer V2.3).
