New feature: Element expressions on floating-point arrays

Announces the introduction of floating-point element expressions.

The restriction that the element expression may only be applied to arrays of integers has been lifted. The element expression can now be applied to arrays of floating-point values. In the C++ API, there is a new operator on the class IlcFloatArray as well as a lifted restriction on the operator IloNumArray::operator[]. In the Java™ API, the restriction has been lifted on the IloCP::element methods. In the C# API, the restriction has been lifted on the CP.element methods.
