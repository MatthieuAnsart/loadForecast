Changes since CP Optimizer V12.2

Describes the changes for the release of the CP Optimizer feature of CPLEX Optimization Studio V12.3.

  • New feature: New method for custom constraints
    Announces the introduction of a new method for informing the optimizer that a custom constraint is violated.
  • New feature: Generalized objectives for scheduling
    Announces the removal of the semi-convex restriction on the piecewise linear function used in certain expressions.
  • New feature: Piecewise linear expressions
    Announces the introduction of a new method for representing piecewise linear functions in some APIs.
  • New feature: Multi-criteria objectives
    Announces the introduction of multi-criteria objectives.
  • New feature: Isomorphism constraint
    Announces the introduction of the isomorphism constraint for interval variables.
  • New feature: New value and default for the search type
    Describes a new value and a new default for the parameter SearchType.
  • New feature: New value and default for the workers parameter
    Describes a new value and a new default for the parameter Workers.
  • New feature: New enumeration field representing effective workers
    Describes a new field, EffectiveWorkers, of the integer information enumeration.
  • Conversion notes: Deprecated methods
    Lists methods deprecated in CP Optimizer V12.3.
  • Conversion notes: Change in search log for propagate
    Describes a minor change in search log functionality when the propagate method is called.
  • Conversion notes: Change in search log display
    Describes a change in the search log display.
  • Resolved issues
    These are the registered issues that are fixed in CP Optimizer V12.3 since the previous release (CP Optimizer V12.2).
