Custom inferencer

Custom inferencers have been introduced in this version of CP Optimizer.

This version of CP Optimizer introduces the concept of a custom inferencer. A custom inferencer is a way to integrate costly inference algorithm in the default search and should be used instead of defining a "heavy" constraint which would slow down the search for a solution.

The custom inferencer is only available for the C++ API. For more information on the new custom inferencer API, please refer to the concept “Custom inferencer” in the CP Optimizer C++ API Reference Manual. An example showing the usage of a custom inferencer is given in the example networkcfg.cpp.
