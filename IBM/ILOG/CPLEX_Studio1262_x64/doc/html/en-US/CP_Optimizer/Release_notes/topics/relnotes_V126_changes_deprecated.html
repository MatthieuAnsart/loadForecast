Conversion notes: Deprecated methods and parameters

The PropagationLog parameter has been deprecated. One of the push methods has been deprecated.

Propagation log parameter

The propagation log parameter has been deprecated in all APIs.

In the C++ API of CP Optimizer, the deprecated parameter is IloCP::PropagationLog.

In the Java™ API of CP Optimizer, the deprecated parameter is IloCP.IntParam.PropagationLog.

Likewise, in the C# API of CP Optimizer, the deprecated parameter is CP.IntParam.PropagationLog.

Push method

The method IlcConstraintI::push(IlcInt priority) in the C++ API which takes an integer as the argument has been deprecated.
