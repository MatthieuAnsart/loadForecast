New feature: Source file locations

With this version of CP Optimizer, a source file location can be associated with some model elements.

For each constraint, variable or expression, it is possible to associate a source file location, the file name and line number. CP Optimizer can use these locations later for reporting errors and conflicts. Locations are also included in exported and dumped models.

Source file locations can be provided to CP Optimizer in the following ways:

  • In the C++ API of CP Optimizer using the macros ILOADD and ILOSETLOCATION.
  • In the CP Optimizer file format using #line directives.
  • In Java™ API of CP Optimizer, locations are assigned automatically by functions IloCP.add, IloCP.intVar, IloCP.intervalVar, IloCP.intervalSequenceVar, and IloCP.stateFunction.

Support for file locations can be turned off using the parameter UseFileLocations.
