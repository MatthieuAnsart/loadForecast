New feature: New method for custom constraints

Announces the introduction of a new method for informing the optimizer that a custom constraint is violated.

A new method for informing the optimizer that a custom constraint has been violated has been added in both the C++ API and the Java™ API. In the C++ API, this new method is IloPropagatorI::violate. In the Java API, this new method is IloCustomConstraint.violate.
