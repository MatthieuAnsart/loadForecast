New feature: Count different expression

A new expression for counting the number of different values in an array is available.

The constrained expression "count different" represents the number of different values taken on by an array of constrained variables or expressions. This value is at least 1 (if all elements of the array take the same value) and at most the number of element in the array (if all element of the array take different values).

In the C++ API of CP Optimizer, the count different expression is represented by IloIntExpr IloCountDifferent(const IloIntExprArray x).

In the Java™ API of CP Optimizer, the count different expression is represented by IloCP.countDifferent(IloIntExpr[] x).

Likewise, in the C# API of CP Optimizer, the count different expression is represented by IIntExpr CP.CountDifferent(IIntExpr[] x).

In the C++ extensions API of CP Optimizer, the count different expression is represented by IlcIntExp IlcCountDifferent(IlcIntVarArray x) and IlcIntExp IlcCountDifferent(IlcIntVarArray x, IlcFilterLevel fl).

The inference level of this constraint can be controlled via a new inference level parameter which can be set in the different interfaces as follows.

In the C++ API of CP Optimizer, the count different expression inference level is set using IloCP::setParameter(IloCP::CountDifferentInferenceLevel, ...).

In the Java API of CP Optimizer, the count different expression inference level is set using IloCP.setParameter(IloCP.IntParam.CountDifferentInferenceLevel, ...);.

Likewise, in the C# API of CP Optimizer, the count different expression inference level is set using CP.SetParameter(CP.IntParam.CountDifferentInferenceLevel, ...).
