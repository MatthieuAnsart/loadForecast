Conversion notes: Deprecated methods

Lists methods deprecated in CP Optimizer V12.3.

These methods have been deprecated in CP Optimizer V12.3 and will be removed from a future version of the product.

In the class IloIntTupleSet in the C++ API, the following methods have been deprecated:

  • setBigTuple

  • setHoloTuple

  • setSmallTuple

The default holotuple data structure is always most efficient, thus there is no longer a need to provide methods to modify this value.

The methods IloCP::RegisterLicense in the C++ API, IloCP.RegisterLicense in the Java™ API and CP.RegisterLicense in the .NET Languages API have been deprecated.
