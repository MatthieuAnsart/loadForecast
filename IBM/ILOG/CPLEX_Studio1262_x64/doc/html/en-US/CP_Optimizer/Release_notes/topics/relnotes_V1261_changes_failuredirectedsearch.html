New feature: Failure-directed search emphasis parameter

A new parameter controls the time spent in failure-directed search.

The new failure-directed search emphasis parameter controls how much time CP Optimizer invests in failure-directed search once it is started. For scheduling problems where failure-directed search plays a major role in finding solutions or proving optimality, increasing the value of this parameter can improve the performance.

In the C++ API of CP Optimizer, failure-directed search emphasis is controlled by the new parameter IloCP::FailureDirectedSearchEmphasis.

In the Java™ API of CP Optimizer, failure-directed search emphasis is controlled by the new parameter IloCP.DoubleParam.FailureDirectedSearchEmphasis.

Likewise, in the C# API of CP Optimizer, failure-directed search emphasis is controlled by the new parameter CP.DoubleParam.FailureDirectedSearchEmphasis.
