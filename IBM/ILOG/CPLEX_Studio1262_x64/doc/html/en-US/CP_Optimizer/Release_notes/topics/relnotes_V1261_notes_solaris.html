Notes for Solaris users

Describes troubleshooting for Solaris users.

If you have trouble launching 64-bit executables (port ultrasparc64_solaris), make sure that you have the 64-bit compiler runtime libraries in your path. You can do this by setting the environment variable LD_LIBRARY_PATH. For example, in csh: 

   setenv LD_LIBRARY_PATH /usr/lib/sparcv9:$LD_LIBRARY_PATH
