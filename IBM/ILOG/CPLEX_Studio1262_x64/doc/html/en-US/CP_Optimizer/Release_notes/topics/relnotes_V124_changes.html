Changes since CP Optimizer V12.3

Describes the changes for the release of the CP Optimizer feature of CPLEX Optimization Studio V12.4.

  • New feature: Deterministic parallel mode
    The multi-worker mode now follows a deterministic behavior.
  • New feature: Element expressions on floating-point arrays
    Announces the introduction of floating-point element expressions.
  • Documentation: Improvement regarding overflow
    The documentation for the constructor of the integer variables has been improved.
  • Resolved issues
    These are the registered issues that are fixed in CP Optimizer V12.4.
