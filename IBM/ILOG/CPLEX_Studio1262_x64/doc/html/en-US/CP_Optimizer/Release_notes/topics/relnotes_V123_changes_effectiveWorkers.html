New feature: New enumeration field representing effective workers

Describes a new field, EffectiveWorkers, of the integer information enumeration.

The EffectiveWorkers field of the integer information enumeration represents number of workers used for search. The value is determined in the following way:

  1. If the search did not start yet, then:

    1. If the Workers parameter is set to anything other than Auto then this value is returned.

    2. Otherwise it returns number of CPU cores available on the machine.

  2. During or after the search it returns the effective number of workers used during the search.

In the C++ API, the number of effective workers is represented by IloCP::IntInfo::EffectiveWorkers. In the Java™ API, this field is IloCP.IntInfo.EffectiveWorkers, and in the .NET languages API it is CP.IntInfo.EffectiveWorkers.

For example, the effective workers can be queried in the C++ API using the method IloCP::getInfo:

cout << "Workers = " << cp.getInfo(IloCP::EffectiveWorkers) << endl;
