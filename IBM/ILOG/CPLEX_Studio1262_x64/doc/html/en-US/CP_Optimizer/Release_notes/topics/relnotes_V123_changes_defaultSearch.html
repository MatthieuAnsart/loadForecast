New feature: New value and default for the search type

Describes a new value and a new default for the parameter SearchType.

In the C++ API, the IloCP::Auto value has been added to the set of possible values of the parameter IloCP::SearchType. In the Java™ API, this value is IloCP.ParameterValues.Auto, and in the .NET Languages API, it is CP.ParameterValues.Auto.

When SearchType is set to Auto in sequential mode, restart search will be used in general. In parallel mode, when SearchType is set to Auto, different searches among restart and multipoint are automatically dispatched over the workers; otherwise all the workers execute the type of search specified by SearchType.

The default value of the parameter is now Auto.
