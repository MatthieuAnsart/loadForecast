Changes since CP Optimizer V12.6.0

Describes the changes for the release of the CP Optimizer feature of CPLEX® Optimization Studio V12.6.1.

  • New feature: CP Optimizer file format
    Methods for dumping, exporting and importing the model have been introduced.
  • New feature: Failure-directed search emphasis parameter
    A new parameter controls the time spent in failure-directed search.
  • New feature: Source file locations
    With this version of CP Optimizer, a source file location can be associated with some model elements.
  • Conversion notes: Deprecated functions
    The member and not member functions have been deprecated.
