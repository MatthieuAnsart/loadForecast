Conversion notes: Deprecated functions

The member and not member functions have been deprecated.

Member functions

The member functions have been deprecated in all APIs. For equivalent functionality, new allowed assignment functions have been added.

In the C++ API of CP Optimizer, the deprecated functions are IloMember and IloNotMember. The new functions are IloAllowedAssignments(IloEnv, IloIntExpr exp, IloIntArray values) and IloForbiddenAssignments(IloEnv, IloIntExpr exp, IloIntArray values).

In the Java™ API of CP Optimizer, the deprecated method is IloCP.member. The new methods are IloCP.allowedAssignments(IloIntExpr exp, int[] values) and IloCP.forbiddenAssignments(IloIntExpr exp, int[]values).

Likewise, in the C# API of CP Optimizer, the deprecated method is CP.Member. The new methods are CP.AllowedAssignments(IIntExpr exp, int[] values) and CP.ForbiddenAssignments(IIntExpr exp, int[] values).
