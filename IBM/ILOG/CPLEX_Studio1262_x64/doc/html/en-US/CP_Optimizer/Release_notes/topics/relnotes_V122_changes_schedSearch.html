Scheduling search API

Classes and functions for customizing search for scheduling problems have been added for this version of CP Optimizer.

While the modeling objects provided by Concert Technology and the algorithms provided by CP Optimizer will generally be sufficient to model and solve your problem, in some situations you may need the full flexibility of writing and maintaining the solution process in various ways, such as writing custom constraints or custom constructive search by means of goals. With this release, a number of new classes and functions are provided in the CP Optimizer engine extensions for implementing custom scheduling constraints and search.

For more information on the new API, please refer to the concept “Search API for scheduling” in the CP Optimizer C++ API Reference Manual and to the section Writing custom scheduling constraints and goals in the CP Optimizer Extensions User’s Manual.
