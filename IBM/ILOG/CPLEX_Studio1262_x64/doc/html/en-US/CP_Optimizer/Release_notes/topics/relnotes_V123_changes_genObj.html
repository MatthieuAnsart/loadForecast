New feature: Generalized objectives for scheduling

Announces the removal of the semi-convex restriction on the piecewise linear function used in certain expressions.

In prior versions of CP Optimizer, the piecewise linear functions used in the expressions IloStartEval, IloEndEval, IloSizeEval and IloLengthEval were required to be semi-convex. This limitation no longer exists, and all types of piecewise linear functions are now allowed. This extension makes it possible to represent very general preference functions for the start and end dates of activities.
