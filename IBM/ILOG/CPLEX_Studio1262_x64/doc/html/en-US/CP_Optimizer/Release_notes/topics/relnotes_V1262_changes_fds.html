Improved feature: Failure-directed search

With this version of CP Optimizer, failure-directed search has been improved.

If your scheduling problem benefits from failure-directed search (failure-directed search will indicate that it is in use on the CP Optimizer log), then version V12.6.2 may provide performance increases when more than a single worker is used to solve the problem.

In addition, the FailureDirectedSearchEmphasis parameter can be set to a value greater than 1 in order to use failure-directed search in multiple workers.
