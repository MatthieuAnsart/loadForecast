New feature: Deterministic parallel mode

The multi-worker mode now follows a deterministic behavior.

When the same random seed, the same number of workers and the same deterministic (time independent) limit such as a branch or fail limit are specified across multiple runs, CP Optimizer will deliver the same solutions.

Please note that although logs produced by such runs may differ at some point depending on how workers are individually scheduled by the system, individual worker behaviors will be the same as well as solutions delivered by the parallel run.
