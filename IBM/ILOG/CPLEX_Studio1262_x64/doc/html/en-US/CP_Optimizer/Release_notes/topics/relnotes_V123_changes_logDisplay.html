Conversion notes: Change in search log display

Describes a change in the search log display.

The columns in the search log output have been reordered; the best known solution of an optimization problem is now displayed first. The variable in the branch decision is displayed last in the decision, and its name is no longer shortened. As a consequence, the log output can be arbitrary wide depending on the variable names in the model.
