New feature: Export model

A method for exporting the model has been introduced.

The export model function exports the currently loaded model into the file named filename. This file name must have the extension .cpo. The file is an ordinary text file which can be used to help support resolve a user problem. The names of variables and constraints are contained inside the file as well as the parameter settings. The names of variables and constraints can be anonymized using the model anonymizer parameter.

In the C++ API of CP Optimizer, the model export method is IloCP:exportModel. The model anonymizer parameter is IloCP::ModelAnonymizer.

In the Java™ API of CP Optimizer, the model export method is IloCP.exportModel. The model anonymizer parameter is IloCP.IntParam.ModelAnonymizer.

Likewise, in the C# API of CP Optimizer, the model export method is CP.ExportModel. The model anonymizer parameter is CP.IntParam.ModelAnonymizer.
