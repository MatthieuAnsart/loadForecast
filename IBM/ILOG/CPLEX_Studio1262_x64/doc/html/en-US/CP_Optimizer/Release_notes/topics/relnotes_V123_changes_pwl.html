New feature: Piecewise linear expressions

Announces the introduction of a new method for representing piecewise linear functions in some APIs.

A new method for expressing piecewise linear constraints has been added in both the Java™ API and the .NET Languages API. In the Java API, these new methods are called IloCP.piecewiseLinear. In the C# API, these new methods are called CP.piecewiseLinear. These functions are still available in the C++ API as IloPiecewiseLinear.
