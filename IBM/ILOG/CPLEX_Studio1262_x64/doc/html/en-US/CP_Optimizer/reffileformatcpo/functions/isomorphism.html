isomorphism

Returns an isomorphism constraint between two sets of interval variables.

Syntax

constraint isomorphism(intervalVarArray array1, intervalVarArray array2, intExprArray map = null, int absentValue = 0)

Parameters

  • array1, array2: the two isomorphic sets of interval variables.
  • map: array of integer expressions mapping intervals of array2 on array1
  • absentValue: value of map[i] when array2[i] is absent.

Description

This function creates an isomorphism constraint between the set of interval variables in the array array1 and the set of interval variables in the array array2. If an integer expression array map is used, it is used to reflect the mapping of the intervals of array1 on the intervals of array2, that is, interval variable array2[i], if present, is mapped on interval variable array1[map[i]]. If array2[i] is absent, index map[i] takes value absentValue.

For more information on the isomorphism constraint, see the concept Isomorphism constraint.

Example

In the example below we have a set of 8 activities a1,...,a8 of different size. These activities should be grouped by pairs of contiguous activities with a minimal separation time between each pair. There is a minimal total duration for each pair of contiguous activities.

a1 = intervalVar(size=8);
a2 = intervalVar(size=3);
a3 = intervalVar(size=6);
a4 = intervalVar(size=7);
a5 = intervalVar(size=4);
a6 = intervalVar(size=2);
a7 = intervalVar(size=9);
a8 = intervalVar(size=3);
minSepTime = 5;
minPairDur = 10;
slot11 = intervalVar(size=2..9);
slot12 = intervalVar(size=2..9);
slot21 = intervalVar(size=2..9);
slot22 = intervalVar(size=2..9);
slot31 = intervalVar(size=2..9);
slot32 = intervalVar(size=2..9);
slot41 = intervalVar(size=2..9);
slot42 = intervalVar(size=2..9);
endAtStart(slot11,slot12);
endAtStart(slot21,slot22);
endAtStart(slot31,slot32);
endAtStart(slot41,slot42);
endBeforeStart(slot12,slot21,minSepTime);
endBeforeStart(slot22,slot31,minSepTime);
endBeforeStart(slot32,slot41,minSepTime);
lengthOf(slot11)+lengthOf(slot12)>=minPairDur;
lengthOf(slot21)+lengthOf(slot22)>=minPairDur;
lengthOf(slot31)+lengthOf(slot32)>=minPairDur;
lengthOf(slot41)+lengthOf(slot42)>=minPairDur;
isomorphism([a1,a2,a3,a4,a5,a6,a7,a8], [slot11,slot12,slot21,slot22,slot31,slot32,slot41,slot42]);

A possible solution to this problem is the following mapping:

  • slot11 : a6 [0,2)
  • slot12 : a1 [2,10)
  • slot21 : a2 [15,18)
  • slot22 : a4 [18,25)
  • slot31 : a8 [30,33)
  • slot32 : a7 [33,42)
  • slot41 : a3 [47,53)
  • slot42 : a5 [53,57)
