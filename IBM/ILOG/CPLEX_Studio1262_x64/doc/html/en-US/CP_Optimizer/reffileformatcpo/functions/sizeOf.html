sizeOf

Returns the size of a specified interval variable.

Syntax

intExpr sizeOf(intervalVar, int absentValue = 0)

Parameters

  • itv: interval variable.
  • absentValue: value to return if the interval variable itv becomes absent.

Description

This function returns an integer expression that is equal to size of the interval variable itv if it is present. If it is absent then the value of the expression is absentValue (zero by default).

Example

a = intervalVar(optional, size=10..20);
b = intervalVar(optional, size=10..20);
// In case both interval variables a and b are present, the size of b should be larger than size of a.
sizeOf(a) < sizeOf(b, intervalmax);
