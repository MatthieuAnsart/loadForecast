before

Constrains an interval variable to be before another interval variable in a sequence.

Syntax

constraint before(sequenceVar seq, intervalVar itv1, intervalVar itv2)

Parameters

  • seq: sequence variable.
  • itv1, itv2: interval variables.

Description

This function returns a constraint that states that whenever both interval variables itv1 and itv2 are present, itv1 must be ordered before itv2 in the sequence variable seq.

Example

a = intervalVar(size=5);
b = intervalVar(optional, size=6);
c = intervalVar(size=7);
// A sequence variable defined on {a,b,c}
seq = sequenceVar([a,b,c]);
// The interval variables of the sequence cannot overlap
noOverlap(seq);
// If interval variable b is present, it must be ordered before c in the sequence
before(seq,b,c);

Requirements

  • Interval variables itv1 and itv2 should both belong to sequence variable seq.
