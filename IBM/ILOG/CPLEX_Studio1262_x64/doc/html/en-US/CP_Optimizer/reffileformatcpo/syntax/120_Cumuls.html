Cumulative expressions

Parent topic: CP Optimizer file format syntax

Type cumulAtom

The cumulAtom type represents a basic cumul function expression. A cumulAtom is created by calling one of the following functions:

  • pulse
  • stepAtStart
  • stepAtEnd
  • stepAt

Cumul atoms can be combined together with operators + and - to form a cumulExpr.

Example

a1 = intervalVar(size=2);
a2 = intervalVar(size=4);
a3 = intervalVar(size=3);
// Functions below are cumul atoms
f1 = pulse(a1,4);     
f2 = stepAtStart(a2,1); 
f3 = stepAtEnd(a3,1);
f4 = stepAt(4,5);
// Functions below are composite cumul function expressions
f5 = f1 - f2 + f3 + f4;
f6 = pulse(a1,2) +  stepAtStart(a2,2) - stepAtEnd(a3,3) + stepAt(4,5);
// Constraints can be posted on cumul functions expressions
f1 <= 5;
f5 <= 5;
alwaysIn(f6,0,5,0,4);

Type cumulExpr

The cumulExpr type represents a general cumul function expression.
