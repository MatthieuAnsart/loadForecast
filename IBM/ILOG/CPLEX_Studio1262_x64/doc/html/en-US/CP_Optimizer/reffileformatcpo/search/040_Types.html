Types in search section

All statements in the search section must be related to the fixing strategy. For example, it is not possible to declare new variables in the search section. Therefore there is a limitation in respect of types that can be used within search section.

From the already described types only the following can be used in the search section:

  • float
  • floatArray
  • int
  • intArray
  • intervalVarArray
  • intExprArray
  • sequenceVarArray

Furthermore there are additional types for evaluators, selectors and search phases that can be used only in the search section and not anywhere else:

intVarEval
An integer variable evaluator is a function that associates an evaluation to each integer variable in order to select some for fixing, for example domainSize. In general, the evaluation can depend on the current state of the solver. For a complete list of integer variable evaluators see Integer variable evaluators.
intVarSelector
An integer variable selector selects one or more variables as fixing candidates. The selector usually uses an evaluator to evaluate the candidates first. A complete list of selectors is given here.
intVarSelectorArray
Multiple integer variable selectors can be used together as described here. In this case, the sequence of selectors is encoded in intVarSelectorArray. Note that intVarSelectorArray follows the ordinary array syntax as described in Arrays and array types.
intVarChooser
Type intVarChooser covers types intVarSelector and intVarSelectorArray. i.e. types intVarSelector and intVarSelectorArray are specializations of intVarChooser.
The Function searchPhase takes an intVarChooser argument, however there is no function that returns intVarChooser. The supplied argument can therefore be a single intVarSelector or multiple selectors contained in an intVarSelectorArray.
intValueEval
An integer value evaluator is a function that assigns an evaluation to each value in the current domain of integer variable. In general the evaluation can also depend on the current state of the solver. The evaluation can be used by a selector to choose the value for the variable. A list of available evaluators is given here.
intValueSelector
An integer value selector selects one or more values from the current domain of an integer variable. Usually an evaluator is used in order to evaluate the different options. A list of available selectors is here.
intValueSelectorArray
When multiple integer value selectors are used then the sequence of selectors is encoded in an intValueSelectorArray. Note that intValueSelectorArray follows the ordinary array syntax as described in Arrays and array types.
intValueChooser
Type intValueChoser covers types intValueSelector and intValueSelectorArray.
The searchPhase takes an intValueChooser argument, however there is no function that returns intValueChooser. The supplied argument can therefore be a single intValueSelector or multiple selectors contained in an intValueSelectorArray.
searchPhase
A search phase is a variable fixing strategy. It can specify merely a group of variables to fix but can also specify how to choose both the variable to fix and to which value it is to fixed.
