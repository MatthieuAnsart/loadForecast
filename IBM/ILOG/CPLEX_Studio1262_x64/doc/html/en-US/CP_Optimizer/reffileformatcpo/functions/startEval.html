startEval

Evaluates segmentedFunction at the start of an interval variable.

Syntax

floatExpr startEval(intervalVar itv, segmentedFunction f, float absentValue = 0)

Parameters

  • itv: interval variable.
  • f: function to evaluate.
  • absentValue: value to return if interval variable itv is absent.

Description

Evaluates function f at the start of interval variable itv. However if itv is absent then it does not have any defined start and absentValue is returned.

Example

Task T is urgent: there is a cost associated with starting it late which is expressed by segmentedFunction:

In the picture above, task T starts at 12 and the cost is therefore 5.

The cost can be computed using startEval as:

f = segmentedFunction((0, 0), (10, 5, 0), (15, 5, 3));
T = intervalVar();
cost = startEval(T, f);
