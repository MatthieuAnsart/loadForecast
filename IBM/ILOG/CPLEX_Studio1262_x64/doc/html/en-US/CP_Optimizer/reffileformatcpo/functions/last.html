last

Constrains an interval variable to be the last in a sequence.

Syntax

constraint last(sequenceVar seq, intervalVar itv)

Parameters

  • seq: sequence variable.
  • itv: interval variable.

Description

This function returns a constraint that states that whenever interval variable itv is present, it must be ordered last in the sequence variable seq.

Example

a = intervalVar(size=5);
b = intervalVar(size=6);
c = intervalVar(optional, size=7);
// A sequence variable defined on {a,b,c}
seq = sequenceVar([a,b,c]);
// The interval variables of the sequence cannot overlap
noOverlap(seq);
// If interval variable c is present, it must be the last one in the sequence
last(seq,c);

Requirements

  • itv should be an interval variable of sequence variable seq.
