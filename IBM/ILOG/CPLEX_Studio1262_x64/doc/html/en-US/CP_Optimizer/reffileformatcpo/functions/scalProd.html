scalProd

Returns the scalar product of two vectors.

Syntax

intExpr scalProd(intArray x, intExprArray y)
intExpr scalProd(intExprArray x, intArray y)
intExpr scalProd(intExprArray x, intExprArray y)

floatExpr scalProd(floatArray x, floatExprArray y)
floatExpr scalProd(floatExprArray x, floatArray y)
floatExpr scalProd(floatExprArray x, floatExprArray y)

Parameters

  • x, y: Input arrays (vectors) to be multiplied.

Description

The function scalProd returns an integer or floating-point expression that represents the scalar product of two vectors x and y. Depending on the type of x and y the result is either integer or floating-point expression.

The versions with constant arrays (intArray or floatArray) are preferred because they can be slightly faster.

Example

a = intVar(0..9);
b = intVar(0..9);
c = intVar(0..9);
scalProd([a, b, c], [100, 10, 1]) == 256;

The scalProd expression above is equal to 100*a + 10*b + 1*c. Therefore the solution of this model is a = 2, b = 5 and c = 6.

Requirements

Arrays x and y must have the same length.

Notes

When x and y are empty then the result of scalProd is the constant zero.
