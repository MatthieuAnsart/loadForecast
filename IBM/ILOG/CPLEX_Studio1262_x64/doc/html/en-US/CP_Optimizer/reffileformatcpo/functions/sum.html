sum

Returns the sum of the input.

Syntax

floatExpr sum(floatExprArray x)
intExpr sum(intExprArray x)

Parameters

  • x: an array of integer or floating-point expressions.

Description

The function sum computes the sum of x. Depending on the type of x the result is an integer or a floating-point expression.

Example

a = intVar(1..3);
b = intVar(1..3);
c = intVar(1..3);
c == sum([a,b]);
