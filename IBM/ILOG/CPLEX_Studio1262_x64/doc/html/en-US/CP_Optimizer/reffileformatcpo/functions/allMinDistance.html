allMinDistance

Constraint on the minimum absolute distance between a pair of integer expressions in an array.

Syntax

boolExpr allMinDistance(intExprArray exprs, int distance)

Parameters

  • exprs: array of integer expressions
  • distance: value used to constrain the distance between two elements of exprs

Description

This constraint makes sure that the absolute distance between any pair of integer expressions in exprs will be greater than or equal to the given integer distance. In short, for any i, j distinct indices of exprs , it enforces abs(exprs[i] - exprs[j]) >= distance.

Example

Imagine we have to assign frequencies to communication channels to radio controlled toys so that interference can be minimized. This can be done by ensuring that toy frequencies must be separated by at least 15 kHz. Such problem can be represented by:

fmin=49830; // kHz
fmax=49890;
// declare five toy channels
toys = [intVar(fmin..fmax),intVar(fmin..fmax),intVar(fmin..fmax),intVar(fmin..fmax),intVar(fmin..fmax)];
// ensure kHz frequency separation
allMinDistance(toys, 15);
