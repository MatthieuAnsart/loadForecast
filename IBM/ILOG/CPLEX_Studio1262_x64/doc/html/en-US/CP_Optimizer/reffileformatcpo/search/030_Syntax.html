Syntax of search section

The search section starts by search { and ends by }. Inside the section it is possible to declare search phases and any other objects necessary for them (i.e. evaluators, selectors etc.). The section can also be empty.

The following picture summarizes the syntax of the search section:

Statement identifier=expression; assigns a name to a given expression so that it can be used later by that name. The expression must follow the expression syntax as described here. Additionally, only expressions of a certain type are allowed in the search section, see Types in search section.

The second statement searchPhaseExpression; adds a search phase into the fixing strategy. Note that the order of search phases is important. The syntax of searchPhaseExpression also follows the standard expression syntax, even if the type of the expression must be searchPhase.
