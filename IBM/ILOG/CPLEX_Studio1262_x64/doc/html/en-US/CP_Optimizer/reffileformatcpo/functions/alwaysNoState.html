alwaysNoState

This constraint ensures that a state function is undefined on an interval.

Syntax

constraint alwaysNoState(stateFunction f, intervalVar itv)
constraint alwaysNoState(stateFunction f, sint start, sint end)

Parameters

  • f: constrained state function.
  • itv: interval variable during which function f is constrained.
  • start, end: fixed interval [start,end) during which function f is constrained.

Description

This function returns a constraint that ensures that state function f is undefined everywhere on the interval (either interval variable itv when it is present or fixed interval [start,end)). This constraint will ensure, in particular, that no interval variable that requires the function to be defined (see alwaysEqual, alwaysConstant) can overlap with interval variable itv or fixed interval [start,end)).

For more information on state functions, see the concept State functions in CP Optimizer.
