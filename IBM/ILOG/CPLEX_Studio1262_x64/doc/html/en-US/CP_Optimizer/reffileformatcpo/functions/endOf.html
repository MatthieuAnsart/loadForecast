endOf

Returns the end of specified interval variable.

Syntax

intExpr endOf(intervalVar itv, int absentValue = 0)

Parameters

  • itv: interval variable.
  • absentValue: value to return if the interval variable itv becomes absent.

Description

This function returns an integer expression that is equal to end of the interval variable itv if it is present. If it is absent then the value of the expression is absentValue (zero by default).

Example

A worker must perform four tasks (a, b, c and d). Task c is optional. Find a solution with minimum schedule length (minimize makespan):

a = intervalVar(length=3);
b = intervalVar(length=3);
c = intervalVar(optional, length=9);
d = intervalVar(length=6);
// Worker can perform only one task at a time:
noOverlap([a, b, c, d]);
// Minimize total length of the schedule:
minimize(max([endOf(a), endOf(b), endOf(c), endOf(d)]));

The optimal solution of this problem is to leave task c unperformed (absent). The last task then ends at time 3+3+6=12.
