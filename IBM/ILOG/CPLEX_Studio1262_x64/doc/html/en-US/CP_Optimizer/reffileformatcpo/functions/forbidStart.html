forbidStart

Forbid an interval variable to start during specified regions.

Syntax

constraint forbidStart(intervalVar itv, stepFunction f)

Parameters

  • itv: interval variable being restricted.
  • f: if the function f has value 0 at point t then the interval variable itv cannot start at t.

Description

This constraint restricts possible start times of interval variable using a step function. The interval variable can start only at points where the function value is not zero. When the interval variable is absent then this constraint is automatically satisfied (since such interval variable does not have any start at all).

In declaration of intervalVar it is only possible to specify a range of possible start times. This function allows more precise specification of when the interval variable can start.

Example

Forbid task itv to start during weekends.

workingDays = stepFunction((0, 100), (5, 0), (7, 100), (12, 0), (14, 100), (19, 0));
itv = intervalVar(length=4, end=0..21);
forbidStart(itv, workingDays);

Notes

  • When the interval variable is optional and this constraints eliminates all the remaining possible start times, then then interval variable is set to absent.
