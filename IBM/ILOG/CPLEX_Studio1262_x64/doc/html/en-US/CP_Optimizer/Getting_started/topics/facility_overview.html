Overview

A solution to the warehouse location problem can be determined using CP Optimizer and the three-stage method of describe, model and solve.

In this lesson, you will learn how to:

  • use the template IloArray;

  • use the class IloIntVarArray to declare arrays of decision variables;

  • use the functions IloCount and IloScalProd;

  • use arithmetic and element expressions;

  • use objectives.

You will learn to model and solve a logistics problem, a facility location problem. To find an optimal solution to this problem using CP Optimizer, you will use the three-stage method: describe, model and solve.
