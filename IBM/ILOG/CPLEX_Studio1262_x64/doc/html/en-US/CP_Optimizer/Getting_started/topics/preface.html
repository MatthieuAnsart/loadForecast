Welcome to Getting Started with CP Optimizer

This is the Getting Started with CP Optimizer manual.

  • Overview
    CP Optimizer is a software library which provides a constraint programming engine.
  • About this manual
    The Getting Started with CP Optimizer manual is composed of lessons which are designed to guide you in learning about the concepts and features of CP Optimizer.
  • Prerequisites
    The lessons in the Getting Started with CP Optimizer manual assume that you have a working knowledge of one of the available APIs and have installed CP Optimizer.
  • Related documentation
    The Getting Started with CP Optimizer manual is part of a collection of manuals.
  • Installing CP Optimizer
    The installation directions for CP Optimizer are in the Electronic Product Delivery package.
  • Location of examples
    The partial and completed versions of the examples used in the lessons in the Getting Started with CP Optimizer manual are available in the distribution of CP Optimizer.
  • Typographic and naming conventions
    The naming conventions used in Getting Started with CP Optimizer manual follow the standard practices of the programming language being used.
