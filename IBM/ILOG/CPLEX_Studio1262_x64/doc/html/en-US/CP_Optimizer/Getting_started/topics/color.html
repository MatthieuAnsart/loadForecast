Modeling and solving a simple problem with integer variables: map coloring

In this lesson, you will learn how to use CP Optimizer and a three-stage method to find a solution to a map coloring problem.

  • Overview
    A solution to the map coloring problem can be determined using CP Optimizer and the three-stage method of describe, model and solve.
  • Describe
    The first stage in finding a solution to the map coloring problem is to describe the problem using natural language.
  • Model
    Once the map coloring problem has been described using natural language, you then use Concert Technology classes to model the constraint programming problem.
  • Solve
    Once the map coloring problem has been described and modeled, you use CP Optimizer classes and functions to solve the constraint programming problem.
  • Review exercises
    Complete these exercises to review what you have learned in the tutorial. Suggested answers are provided.
  • Complete program
    The location of the complete map coloring program and the results are provided.
