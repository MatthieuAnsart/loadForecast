Overview

CP Optimizer offers debugging features including exception handling, logging and conflict refining.

When you are debugging an application that employs CP Optimizer, this library offers several features, such as logging and conflict refining, to make the task easier in whatever development environment you prefer. The search log introduces the search log of CP Optimizer. In the search log, you can see the effectiveness of propagation, the progress of the search and the status of a solution. This section documents other features of CP Optimizer that facilitate debugging.
