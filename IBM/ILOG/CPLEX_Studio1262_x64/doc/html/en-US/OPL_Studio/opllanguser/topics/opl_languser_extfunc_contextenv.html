Environment prerequisites

Lists what software you need to use external functions.

To call Java™ code, you need to have a Java Runtime Environment installed on your machine. Once it is installed, the JRE is automatically detected at run time (see the function IloOplImportJava for details). OPL supports versions 5.0 and above of IBM® JDK or JRE.

On Unix, the LD_LIBRARY_PATH must also contain the path to the shared libraries of the Java Virtual Machine. For example, /jre/lib/i386 and /jre/lib/i386/client for Linux.

See also Working Environment.
