Setting your locale

Set your locale in the Windows Control Panel.

About this task

On Windows, language settings are specified in Control Panel > Regional and Language Options.

Procedure

To set your locale on Windows:

  1. Open the Windows Control Panel.
  2. Double-click Regional and Language Options.

    The Regional Options tab enables you to select a language.

  3. Select the language you want from the drop-down list, then click OK and close the Control Panel.

Results

The language you select defines the default ISO locale.
