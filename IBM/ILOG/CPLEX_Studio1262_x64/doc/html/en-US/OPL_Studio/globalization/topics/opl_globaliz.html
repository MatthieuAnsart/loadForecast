Globalization

The CPLEX Studio IDE is available in several languages. You can specify the language that you want and, if necessary, override the default setting.

  • Setting your locale
    Set your locale in the Windows Control Panel.
  • Overriding regional settings
    Override the default regional settings with a command in the Windows command prompt.
  • Supported languages and features
    The CPLEX Studio IDE is localized in Brazilian Portuguese, Chinese, French, German, Japanese, Korean, Polish, Russian, Spanish, and Turkish.
  • Not supported
    There are some restrictions on localized versions of the CPLEX Studio IDE. For example, French accents are not supported.
