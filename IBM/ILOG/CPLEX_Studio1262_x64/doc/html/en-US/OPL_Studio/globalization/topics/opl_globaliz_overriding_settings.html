Overriding regional settings

Override the default regional settings with a command in the Windows command prompt.

If you want to use the CPLEX Studio IDE in another language without changing your regional settings, you can override the regional settings by using a specific command in the Windows command prompt.

The commands are:

oplide -nl en to use the IDE in English

oplide -nl fr to use the IDE in French

oplide -nl de to use the IDE in German

oplide -nl es to use the IDE in Spanish

oplide -nl ja to use the IDE in Japanese

oplide -nl zh to use the IDE in Chinese

oplide -nl ko to use the IDE in Korean

oplide -nl ru to use the IDE in Russian

oplide -nl pl to use the IDE in Polish

oplide -nl tr to use the IDE in Turkish

oplide -nl pt_BR to use the IDE in Brazilian Portuguese
