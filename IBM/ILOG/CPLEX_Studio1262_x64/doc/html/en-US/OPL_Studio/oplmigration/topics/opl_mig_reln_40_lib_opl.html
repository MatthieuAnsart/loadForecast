OPL classes deprecated or moved to Concert

Presents the classes from the OPL 3.7 component libraries that are replaced by redesigned interfaces in OPL 4.0.

In OPL 4.0, the OPL 3.7 component libraries are replaced by redesigned ILOG OPL Interfaces. This new API integrates smoothly with ILOG Concert Technology.

For information on the purpose, design, and use of the API, see the Interfaces User’s Manual.
