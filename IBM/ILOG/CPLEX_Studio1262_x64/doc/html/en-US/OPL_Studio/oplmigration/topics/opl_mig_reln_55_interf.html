Interfaces

Lists improvements to the interfaces to OPL.

CPLEX callbacks in Java and .NET

CPLEX callbacks are now supported with parallel search in Java and .NET interfaces.

Postprocessing of feasible solutions

A custom CPLEX callback can now call postprocessing instructions on solutions.

Exceptions in .NET

A bug has been fixed in ILOG Concert .NET interfaces. Now all exceptions thrown by OPL inherit from ILOG.Concert.Exception. So it is no longer possible to catch/throw these exceptions as java.lang.Exception.
