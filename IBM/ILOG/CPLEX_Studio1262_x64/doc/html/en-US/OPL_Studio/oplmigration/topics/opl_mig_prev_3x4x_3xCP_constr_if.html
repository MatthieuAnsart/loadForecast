if-then-else

Gives the syntax and an example of if-then-else for OPL 3.x and OPL 6.x.

See also if/then/else of the Release Notes.

In OPL 3.x

if ... then ... endif
if ... then ... else ... endif

In OPL 4.x/5.x/6.x

if (...) { ... }
if (...) { ... } else { ... }
