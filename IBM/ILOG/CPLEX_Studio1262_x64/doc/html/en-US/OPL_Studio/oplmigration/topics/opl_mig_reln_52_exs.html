Examples and models

New code samples and industry models have been added to illustrate how to model constraint programming problems with OPL.

Where to find the files

You can find the corresponding files at the following location:

<Install_dir>\examples\opl\cp for CP code samples

<Install_dir>\examples\opl\models for CP industry and academic models

where <Install_dir> is the OPL installation directory.

See the Language and Interfaces Samples manual for detailed descriptions.

Code samples

Code samples for the following CP problem or modeling features have been added to the examples\opl\cp directory:

Car sequencing: carseq.prj

Inventory matching; steelmill.prj

See OPL examples that use CP Optimizer (Constraint Programming) in the Language and Interfaces Manual for details.

Industry and academic models

All the models distributed in the Model Library directory

<Install_dir>\examples\opl\models

have been rewritten as necessary to follow the recommended modeling practices:

Namely:

  • decision expressions (dexpr keyword) for objectives

  • labeling of constraints (rather than names or nothing at all),

  • array initialization instead of scripting statements whenever possible

See OPL model library in the Language and Interfaces Manual for details.
