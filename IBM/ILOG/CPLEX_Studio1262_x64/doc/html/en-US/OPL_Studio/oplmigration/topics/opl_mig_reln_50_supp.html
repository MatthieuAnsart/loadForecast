Support for ILOG ODM

Introduces ILOG Optimization Decision Manager (ODM).

Models developed in ILOG OPL Development Studio 5.0 can be used directly with ILOG ODM 2.0 to create applications. See From Problem Solving to What-If Scenarios: A Presentation of ILOG Optimization Decision Manager in the Getting Started with the IDE.

A specific dexpr syntax has been introduced to help you write objectives more efficiently when working with ODM. For example: 

dexpr int allocationCost = sum(i in r) allocation[i];

See the ODM documentation for more information.

Note that arrays of decision expressions (dexpr arrays) are not supported in OPL 5.0. They will be added in the next minor release.
