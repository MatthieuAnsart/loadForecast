Branching (mipsearch)

Explains how to use an execute scripting block in OPL 4.0.

Parent topic: The modeling language

In OPL 3

To associate a priority c[j]/maxCoef[j] and a branching direction with each variable x[j], you wrote (see the knapsack.prj example):

...
setting mipsearch {
forall(j in Columns) {
setPriority(x[j],c[j]/maxCoef[j]); 
setBranchingDirection(x[j],branchDirLow); 
}
...

In OPL 4

You use an execute scripting block or properties and you write:

execute setting_mipsearch {
  for(var j in Columns) {
     cplex.setPriority(x[j],c[j]/MaxCoef[j]);
     cplex.setDirection(x[j],"BranchDown");
  }
  /* ALTERNATIVE use properties
  for(var j in Columns) {
     x[j].priority = c[j]/MaxCoef[j];
     x[j].direction = "BranchDown";
  }
  */
