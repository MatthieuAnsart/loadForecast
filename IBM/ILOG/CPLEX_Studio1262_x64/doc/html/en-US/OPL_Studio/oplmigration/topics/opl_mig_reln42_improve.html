Improvements in OPL 4.2

Includes new CPLEX features, improved model building performance and improved project management.

  • New features of CPLEX 10

    • Improvements in performance, conflict refinements, enhanced FeasOpt, solution polishing, logical constraints, and indicators

  • Conflicts supported in OPL IDE (formerly "irreducibly inconsistent sets", iis)

    • Automatic relaxations of infeasible models and conflict display. See Relaxing infeasible models in the IDE Tutorials.

  • Parameters to access new CPLEX features

    • including solution polish, repair, FeasOpt modes, numerical and memory emphasis. See Mathematical programming options in Parameters and Settings in OPL .

  • Improved model building performance and less memory consumption. See Profiling the execution of a model in the IDE Tutorials.

  • Improved project management and model browsing.

  • Correction of known issues: you can see a detailed list on the Customer Support Site at https://support.ilog.fr/

    .
