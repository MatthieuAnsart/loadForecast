The IDE

Describes new and modified features in the user interface.

Removed from the IDE

ILOG OPL Development Studio 4.0 only supports CPLEX.

The following ILOG Solver features are not available:

  • the Solver statistics tabbed pane at the bottom of the Start window

  • the Solver search tree

  • the Solver options

Features that were specific to the OPL 3.x language will not be exposed in the OPL IDE.

Added to the IDE

Among the features of the OPL IDE:

  • The expand feature enables you to "unroll" the constraints in forall blocks. This can be very helpful for model debugging.

  • A state-of-the-art ILOG Script debugger for pre- and postprocessing, and flow control.

  • Major effort on usability and look-and-feel

    • Notification when new solutions are found to an optimization model

    • The ability to pause the optimization, review an interim solution, and resume the optimization

  • The CPLEX progress chart
