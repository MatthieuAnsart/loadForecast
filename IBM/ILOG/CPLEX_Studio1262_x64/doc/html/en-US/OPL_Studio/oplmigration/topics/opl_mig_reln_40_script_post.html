Postprocessing

Describes the impact of ILOG Script on postprocessing.

Scripting is also available for postprocessing solutions. Just as for preprocessing, you embed ILOG Script syntax in a model within an execute block. For example:

execute DISPLAY {
  function printRoute(r) {
    write("  ",r.p,":");
    writeln(r.e.o,"->",r.e.d);
  }

  writeln("Routes:");
  for (var r in Routes) {
    printRoute(r);
  }
}

In addition to preprocessing, decision variables evaluate to their values within the solution.
