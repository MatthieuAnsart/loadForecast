alldifferent

Gives the syntax and an example of alldifferent for OPL 3.x and OPL 6.x.

Parent topic: Constraints, functions, and operators

In OPL 3.x

alldifferent (CstrComposite) Propagation

Example

enum Letters {D,O,N,A,L,G,E,R,B,T}; 
var int l[Letters] in 0..9; 

solve { 
   alldifferent(l) onDomain; 
   l[D] <> 0; 
   l[G] <> 0; 
   l[R] <> 0; 
      100000*l[D] + 1000*l[N] + 200*l[A] + 20*l[L] + 2*l[D]  
   +  100000*l[G] + 9900*l[E]  
   =  99010*l[R] + 1000*l[B] + l[T]; 
};

In OPL 6.x

allDifferent (dvar int[])

Note the uppercase D.

Example

allDifferent(all (i in R : i%2 == 0) x[i] );
