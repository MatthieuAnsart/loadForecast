Conditional statements

Nested conditional expressions are used in OPL 4.0.

Up to 3.7.1, the case keyword was used to model certain conditional statements. Starting with OPL 4.0, case is replaced with nested conditional expressions.

Parent topic: The modeling language

In OPL 3

You wrote:

int c = 1;
int a = case {c = 1 -> 2; c = 2 -> 5; 8}; 
display a; 
solve;

In OPL 4

You write:

int c = 2;
int a = (c == 1) ? 2 :
        (c == 2) ? 5 : 
        8;
execute {
   writeln("a = ",a);
}
