Setting CPLEX parameters

Explains how to use the class IloCplex.

In OPL 3

To set a CPLEX parameter using OPLScript, you would call one of the following methods:

  • setFloatParameter(theParam, theValue)

  • setIntegerParameter(theParam, theValue)

  • setStringParameter(theParam, theValue)

In OPL 4 and later

To set a CPLEX parameter, you need to work with an instance of the class IloCplex and the CPLEX parameters are named properties of the IloCplex object. Therefore, in a scripting block, you must write:

cplex.theParam = theValue
