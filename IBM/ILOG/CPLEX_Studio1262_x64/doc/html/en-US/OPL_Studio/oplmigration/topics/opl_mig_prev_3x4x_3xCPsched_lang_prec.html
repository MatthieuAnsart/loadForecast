Precedence constraints

Explains the use of precedence constraints in OPL 6.x.

Precedence constraints are specified in a similar way in OPL 3.7 and OPL 6.x.

A precedence constraint stating that interval a must end before interval b starts:

In OPL 3.7

  Activity a(10);
  Activity b(10);
  solve {
    a precedes b;
  };

In OPL 6.x

  using CP;
  dvar interval a size 10;
  dvar interval b size 10;
  constraints {
    endBeforeStart(a,b);
  }
