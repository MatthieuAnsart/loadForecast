The language

Lists the language improvements in OPL.

The major improvements to the language are:

  • There are fewer limitations on posting conditional constraints.

  • There is a new syntax for generic arrays, controlling iteration, index, and item expressions. See As generic indexed arrays in the Language Reference Manual and the code samples nurses.prj, oilDB.prj, transp4.prj.

  • OPL opens a spreadsheet in read-only mode to read data from it. The types of data elements supported are the same as for databases, plus scalar integers, floats or strings, and one- or two-dimensional arrays. See What Data Can Be Read from an Excel Spreadsheet in the Language Reference Manual.

  • OPL now supports the CPLEX warm start functionality for LP and MIP problems in flow control scripting.

  • The functions item, maxl, minl are documented. See the Language Quick Reference.
