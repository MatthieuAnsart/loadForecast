Moved to scripting

Contains the OPL 3.7 language facilities that have been moved to the new 4.0 scripting language.

Some of the OPL 3.7 language facilities have been moved to the new scripting language:

  • display instructions (cout, display)

  • import

  • Algorithmic settings, like

    execute INITIALIZE {
  cplex.nodeLim = 1000;
}

See IBM ILOG Script for OPL in the Language Reference Manual for details on the API available for the different types of modeling elements.
