ILOG Script extension classes for OPL

Lists the improvements in ILOG Script specific to OPL.

New ILOG Script API

  • IloConstraint: the lower and upper bounds are readable in postprocessing and flow scripting in constraints inside forall statements.

  • Several new classes:

    • IloOplCplexBasis and IloOplCplexVectors support the warm start functionality for linear programming and mixed integer programming respectively.

    • IloOplInputFile and IloOplOutputFile

    • The new global function IloOplGetEnv to access a named environment variable.
