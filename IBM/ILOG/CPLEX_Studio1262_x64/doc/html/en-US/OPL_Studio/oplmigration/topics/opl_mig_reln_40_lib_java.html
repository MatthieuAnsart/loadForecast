The Java API

Introduces the Java packages.

The Java interface offers basically the same functionality as the C++ one.

The Java API is split into three packages:

  • The ilog.opl package contains the OPL control API, for loading and accessing models.

  • The ilog.concert package contains the Concert modeling API, for modifying models.

  • The ilog.cplex package contains the CPLEX control API, for controlling the solving process.

The Java API is written as wrappers on the equivalent C++ libraries; it offers the same functionality and performance as the C++ API. This means that there is no loss of performance when solving models, whatever language you choose. However, the adaptation from C++ to Java has some consequences on the way you can use either API. See The Java interface in the Integration Libraries User’s Manual for details.
