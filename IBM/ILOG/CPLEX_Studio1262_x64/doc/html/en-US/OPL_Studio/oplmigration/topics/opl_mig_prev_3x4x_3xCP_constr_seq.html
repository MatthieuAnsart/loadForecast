sequence

Gives the syntax and an example of sequence for OPL 3.x and OPL 6.x.

Parent topic: Constraints, functions, and operators

In OPL 3.x

sequence (Expr, Expr, Expr, CstrComposite, Composite, Composite)

In OPL 6.x

It must be written explicitly.

Example

using CP;

int nbMin = 2;
int nbMax = 8;
int width = 10;
int values[1..5] = [1,2,3,4,5];

dvar int vars[1..100] in 1..8;
dvar int cardi[1..5] in 0..100;

constraints {

   // sequence(nbMin,nbMax,width,vars,values,cardi);

   forall (i in 1..5) {
      cardi[i] == count(vars, i);
   }

   forall (i in 1..100-width) {
     nbMin <= sum(k in 1..5) count(all (j in i..i+width) vars[j], values[k]) 
                                                                      <= nbMax;
   }

};
