Command line options

Describes changes to the commands oplrun and oplide.

oplrun

oplrun supports two additional options: -relax and -profile. See Appendix A, oplrun Command Options in the IDE User’s Manual.

oplide

  • The option -nosplash has been replaced with a setting in the IDE. See Start Options in the IDE User’s Manual.

  • The -s option is deprecated.
