Transition times

Explains how to model transition times in OPL 6.x

In OPL 3.7, transition times were declared as a 2-dimensional array of integers. In OPL 6.x, they are described as a tuple set and, more precisely, as a set of triplets of integers. Like in OPL 3.7, transition times are used for modeling disjunctive resources (see Disjunctive resources or State resources).

In OPL 3.7

int n = 10;
range R 1..n;
int tt[i in R, j in R] = abs(i-j);  };

In OPL 6.x

using CP;
int n = 10;
range R = 1..n;
tuple triplet { int id1; int id2; int value; };
{ triplet } tt = { <i,j, ftoi(abs(i-j))> | i,j in R };
