Migration from OPL 3.x (non-CP projects)

Provides a quick-reference list of the changes you need to make to most OPL 3.x models for them to be compatible with OPL 6.x.

  • Converting an OPL 3.x non-CP model
    Provides a procedure to convert a basic OPL 3.x model to an OPL 6.x model.
  • Declaration of data and variables
    Lists the changes to make to data and variable declarations in an OPL 3.x model for compatibility OPL 4 and later
  • Constraints
    Lists the changes to make to constraints in an OPL 3.x model for compatibility with OPL 4 and later.
  • Data
    Lists the changes to make to input data in OPL 3.x for compatibility with OPL 4 and later.
  • Scripting
    Lists the changes to make to script statements in an OPL 3.x model for compatibility with OPL 4 and later.
  • Functions
    Lists the changes to make to functions in an OPL 3.x model for compatibility with OPL 4 and later.
  • Other
    Describes some additional changes in OPL 3.x syntax necessary for compatibility with OPL 4 and later.
  • Optional
    Provides some replacements for syntax that is deprecated.
