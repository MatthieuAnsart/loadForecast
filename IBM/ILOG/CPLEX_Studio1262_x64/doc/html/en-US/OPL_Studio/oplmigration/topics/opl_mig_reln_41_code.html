Code samples

Describes changes in the distributed code samples.

  • No standalone models are delivered as code samples with the OPL distribution. Models with no associated data files have been included in projects.

  • All constraints are named in all code samples, except in product.prj. See Naming Constraints in the Language Reference Manual.

  • New examples delivered:

    • iterators.cpp/java/vb : provided in all three languages to illustrate iteration through OPL elements.

    • nurses.prj : provided to illustrate the relaxation search. See Tutorial: Debugging Infeasible Models in the IDE User’s Manual.

    • profiler.prj : provided to illustrate the profiling facility. See Tutorial: Profiling the Execution of a Model in IDE User’s Manual.
