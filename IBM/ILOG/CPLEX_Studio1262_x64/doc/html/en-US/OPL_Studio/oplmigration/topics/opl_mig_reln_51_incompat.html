Incompatibilities

Two incompatibilities with the previous version, OPL 5.0.

  • Model files compiled with OPL 5.0 or earlier need to be recompiled.

  • Relative paths in scripting statements for IloOplInputFile and IloOplOutputFile instances are now resolved relative to the invoking model file, not relative to the current working directory.
