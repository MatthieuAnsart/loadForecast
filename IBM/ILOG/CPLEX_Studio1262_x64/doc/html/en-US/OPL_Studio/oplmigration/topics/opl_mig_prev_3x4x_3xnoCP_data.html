Data

Lists the changes to make to input data in OPL 3.x for compatibility with OPL 4 and later.

  1. Move the inline initialization syntax <"file" to data files. You need to convert the data first. See Flat files.

  2. Move data blocks (data {}) in model files to data files. See Data block in model files.

  3. Move the database (DB*) or (Sheet*) syntax to data files (see section Databases and Creating a connection to a spreadsheets, and the oilDB.dat example).
