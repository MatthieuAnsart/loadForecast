IDE

Lists new and changed features in the OPL IDE.

Standard toolbar

The standard toolbar now has two new buttons:

  • the Launch oplrun button, which enables you to start the oplrun command line from the IDE. See oplrun in these release notes.

  • the Tune model button, which enables you to start the tuning process. See Tuning tool in these release notes.

Problem Browser

At the top of the Problem Browser, a drop-down list displays the solutions collected in the solution pool, if any, in addition to the final solution. See Solution pool in these release notes.

Math programming options

More CPLEX parameters are available as OPL settings. See Setting MP options.
