predicate

Gives the syntax and an example of predicate for OPL 3.x and OPL 6.x.

Parent topic: Constraints, functions, and operators

In OPL 3.x

predicate p(int i,int j,int k) in { (1,2,3) , (4,5,6), (4,5,7) };
solve {
p(x,y,z);
};

In OPL 6.x

You need to define tuples and sets of legal assignments.

allowedAssignments

Example

using CP;
tuple legal {
int i;
int j;
int k;
};
{legal} legalSet = {<1,2,3> , <4,5,6>, <4,5,7>};
dvar int x;
dvar int y;
dvar int z;
constraints {
allowedAssignments (legalSet,x,y,z);
};
