Scripting

Lists the changes to make to script statements in an OPL 3.x model for compatibility with OPL 4 and later.

  1. Move data and result display instructions to ILOG Script execute blocks.

  2. Reimplement the initalize sections via execute {} blocks using the scripting language.

  3. To set CPLEX parameters, write the statement cplex.theParam = theValue instead of using the setXxxParameter OPLScript methods.

See also Migrating script statements from OPL 3 to OPL 4.
