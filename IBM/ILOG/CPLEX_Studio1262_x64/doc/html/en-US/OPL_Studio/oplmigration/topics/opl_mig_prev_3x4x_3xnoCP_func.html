Functions

Lists the changes to make to functions in an OPL 3.x model for compatibility with OPL 4 and later.

  1. Replace the function name nearest with round. For example, write

    float f1 = 0.6;
float f2 = round(f1);

    instead of

    float f1 = 0.6;
float f2 = nearest(f1);

  2. For procedural function calls, replace the method calls (object-oriented style) by function calls. For example, write S.last() instead of last(S).
