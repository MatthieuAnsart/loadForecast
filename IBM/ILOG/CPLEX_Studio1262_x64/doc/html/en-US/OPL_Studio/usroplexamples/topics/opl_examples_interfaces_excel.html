Excel integration

Describes an example that demonstrates the integration of OPL with Microsoft Office.

This example shows how to control OPL from Microsoft Excel, as well as how to feed input data to OPL and how to write results back to the worksheet.

Note:

The example is available for .NET only.

For Excel 2007:ExcelWarehouse.vb.html

See Integrating OPL with Excel using Visual Studio Tools for Office in the Interfaces User’s Manual.

Note: For a list of supported versions of Microsoft Excel, please see the detailed system requirements http://www-01.ibm.com/support/docview.wss?uid=swg27019100.
