Language and Interfaces Examples

CPLEX Studio is delivered with numerous examples that illustrate the features of the product. You can use the examples to help you write your own OPL models.

  • OPL examples that use CPLEX (Mathematical Programming)
    These examples illustrate specific features of the OPL modeling language and scripting language for models that use IBM ILOG CPLEX.
  • OPL examples that use CP Optimizer (Constraint Programming)
    These examples illustrate specific features of the OPL modeling language and scripting language for models that use IBM® ILOG® CP Optimizer.
  • Solving several OPL examples consecutively
    This example consists of a script that solves several examples consecutively.
  • Solving in batch mode and comparing KPIs
    You can execute the same model with several different data sets and compare the resulting KPIs contained in a generated HTML document. You can also compare several models. To use this feature, you need to write some ILOG Script code.
  • Interfaces examples
    These examples illustrate specific features of the API libraries.
  • OPL model library
    The OPL model library is a collection of OPL models that describe known problems in the academic and industrial fields.
