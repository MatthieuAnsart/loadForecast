Relaxation of infeasible models

Provides an example of a realistic nurse scheduling problem that includes constraints on compatibilities, vacations, required assignments.

The nurses example solves a realistic nurse scheduling problem using mixed integer programming. The model includes constraints on compatibilities, vacations, required assignments. The supplied data file nurses.dat is infeasible. This example is used in IDE Tutorials to describe how OPL relaxation features can help figure out the origin of the infeasibility.

To access this example, go to: examples/opl/nurses.

See also Relaxing infeasible models in IDE Tutorials.
