Where to find the industry examples

Describes where to find the academic and industry examples in the CPLEX Studio distribution.

Academic and industry examples are in

<Install_dir>/opl/examples/opl/models

Most of these problems are described in the book Model Building in Mathematical Programming by H.P. Williams - Wiley - Fourth Edition 2005 - ISBN 0-471-99788-9.

More optimization problems expressed as OPL models are available in

<Install_dir>/opl/examples/opl/models

and described in OPL examples that use CPLEX (Mathematical Programming) and OPL examples that use CP Optimizer (Constraint Programming). Examples illustrating the use of OPL API libraries are described in Interfaces examples.

Table 3, Table 4, and Table 5 order the models alphabetically, by industry, and by mathematical programming technique, respectively. In Table 3, the Project Location column lets you access the directories containing the project, model, and data files (which you can then manually open in the IDE) while the Model Name column provides direct links to the model files displayed in HTML format.
