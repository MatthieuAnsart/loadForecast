Connecting from the command line on UNIX or Linux

To connect to a database on UNIX or Linux platforms, you often need to relink to the database drivers.

To relink to a database driver via the command line interface on UNIX platforms (Solaris, AIX), or Linux, type the following, where <Install_dir> is your installation directory for CPLEX® Optimization Studio:

  1. setenv IL_HOME <Install_dir>/opl

  2. setenv ILDHOME <Install_dir>/opl

  3. In Install_dir>/opl/libport name>/shared_libraries_directory>, execute:

    make -f makefile.drv <shared library>.so

    For example, to connect to Oracle using the version 11.1 driver, enter:

    make -f makefile.drv libdbora11.so

    Depending on your database client installation, you may need to edit the file makefile.drv. For example, the Oracle libraries may be in $ORACLE_HOME/lib instead of $ORACLE_HOME/lib32.

  4. Copy the driver to the bin directory of OPL, for example:

    cp libdbora11.so <Install_dir>/opl/bin/<port_name>/libdbora11.so
