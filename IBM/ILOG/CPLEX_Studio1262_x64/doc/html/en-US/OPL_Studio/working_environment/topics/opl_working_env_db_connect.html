How to connect to a database

It is often necessary to relink to a database driver for OPL before you can connect to a database from UNIX or Linux platforms.

For the command to use when you connect to a database, see Database connection strings in the Language Reference Manual.

  • Connecting from the command line on UNIX or Linux
    To connect to a database on UNIX or Linux platforms, you often need to relink to the database drivers.
  • Connecting from the IDE on Linux
    To connect to a database on Linux platforms, you often need to relink to the database drivers.
