Environment variables for Windows platforms

Environment variables are created automatically when you install CPLEX Optimization Studio on Windows platforms.

Two environment variables are created on Windows platforms by the installer:

  • CPLEX_STUDIO_DIR<version>
  • CPLEX_STUDIO_BINARIES<version>

where <version> corresponds to the version of CPLEX Optimization Studio.

For example, when you install CPLEX Optimization Studio 12.6.1, the following variable names and values are created:

  • Name: CPLEX_STUDIO_DIR1261

    Value: C:\Program Files\IBM\ILOG\CPLEX_Studio1261 (if the product is installed in C:\Program Files).

  • Name: CPLEX_STUDIO_BINARIES1261

    Value: 
    C:\Program Files\IBM\ILOG\CPLEX_Studio1261\opl\bin\x64_win64;
C:\Program Files\IBM\ILOG\CPLEX_Studio1261\opl\oplide\;
C:\Program Files\IBM\ILOG\CPLEX_Studio1261\cplex\bin\x64_win64;
C:\Program Files\IBM\ILOG\CPLEX_Studio1261\cpoptimizer\bin\x64_win64
    (if the product is installed on 64-bit Windows).
