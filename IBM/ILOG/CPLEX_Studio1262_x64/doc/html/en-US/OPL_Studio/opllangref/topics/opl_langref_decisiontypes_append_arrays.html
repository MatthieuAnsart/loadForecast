Appending arrays

You can concatenate several arrays using the append function.

For example, if you want to express that all variables from array x and array y are different, you can use an allDifferent constraint applied to the appended arrays, as shown in this example.

Appending arrays

using CP;

range R=1..10;
dvar int x[R] in 0..20;
dvar int y[R] in 0..20;

minimize sum(i in R)(x[i]+y[i]);
subject to
{
  allDifferent(append(all(i in 1..2) x[i],all(i in 4..6) y[i]));
}
