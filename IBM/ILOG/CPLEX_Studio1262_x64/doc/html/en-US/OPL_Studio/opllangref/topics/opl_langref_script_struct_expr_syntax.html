Syntax of different types of expression

Describes the syntax of several types of expression in OPL.

The following types of expression are described:

  • A reference to a script variable

  • Access to a property

  • Assignment operators

  • Function calls

  • Special keywords

  • Special operators

  • Other operators

Note:

For C/C++ programmers: The syntax of Script expressions is very close to the C and C++ syntax. Expressions include assignments, function calls, property access, and so on.
