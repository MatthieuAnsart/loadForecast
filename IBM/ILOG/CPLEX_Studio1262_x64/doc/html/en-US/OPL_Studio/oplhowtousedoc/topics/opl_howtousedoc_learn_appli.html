Learning the application programming interfaces (APIs)

Provides tips on where to find information to help you learn the OPL application programming interfaces (APIs).

If you want access to your model data, to the solutions, to the engine API, and other advanced features of the C++, Java, and .NET API, read Working with OPL interfaces in the Interfaces User's Manual and the API reference manual for each programming language.
