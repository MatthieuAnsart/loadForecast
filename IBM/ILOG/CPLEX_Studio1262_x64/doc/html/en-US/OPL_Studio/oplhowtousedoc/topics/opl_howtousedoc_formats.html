The documentation formats

Explains the different documentation formats that are provided for CPLEX Studio.

The CPLEX Studio documentation set is delivered:

  • as online help inside the IDE

  • in Eclipse Help format available from the Windows Start Menu and read via conventional browsers

  • In HTML format on Unix platforms.

Note:

To navigate from one online manual to another, either follow a hypertext link in the manual you are reading or click on the title of the manual you want from the table of contents in the left frame.

On Windows

The online documentation is accessible from the Windows Start Menu. Select:

Start > IBM ILOG > CPLEX Optimization Studio <version> > Documentation to open the Eclipse Help. To close the help window, select Shutdown CPLEX Studio Help.

On UNIX

The online documentation is delivered only in cross-platform HTML format. To access it, open the file

<Install_dir>/opl/doc/html/en-US/documentation.html

where <Install_dir> is your installation directory.
