Prerequisites for using this tutorial

It is assumed that SPSS Modeler is installed and you know how to use it.

  • IBM SPSS Modeler must be installed on the same machine as the CPLEX Studio IDE.

  • You must have a license for SPSS Modeler. A trial license is included with the installation and lasts for 90 days.

  • This tutorial assumes you have a working knowledge of SPSS Modeler.

    An overview of SPSS Modeler can be viewed here: http://www-01.ibm.com/software/analytics/spss/products/modeler/.

    Demos of SPSS Modeler can be viewed here: http://www-01.ibm.com/software/analytics/spss/downloads/demos.html.
