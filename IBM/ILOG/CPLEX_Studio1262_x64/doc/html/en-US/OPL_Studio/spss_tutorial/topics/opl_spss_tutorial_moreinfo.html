Getting more information on SPSS stream execution

After stream execution, the log from SPSS can be displayed in the engine log of the IDE.

If you execute an SPSS stream in verbose mode, the log information from SPSS Modeler is displayed in the engine log of the CPLEX Studio IDE. To enable the verbose mode, add the statement SetVerboseSPSS to the .dat file.

For example, the file test_selflearn.dat now contains:

prepare {  
   includeScript("oplspss.js");
   SetVerboseSPSS();
}   
SPSSConnection spss("pm_selflearn.str","");
Table from SPSSRead(spss,"Filter");

Save the modified data file and run the configuration test_selflearn. The log information from SPSS Modeler is displayed in the engine log of the IDE.
