The Welcome page

Describes the CPLEX Studio Welcome screen and how to close it and begin working with the IDE.

When you first launch the CPLEX Studio IDE, a Welcome page is displayed.

The buttons on the Welcome page provide access to user guides, sample manuals, release notes, and user forums.

To access information from the Welcome page

The buttons at the top of the page lead to information on the release and parts of the CPLEX Studio documentation you might frequently refer to.

  • Overview — displays links to:

    From Operations Research to CPLEX Studio and Decision Optimization Center

  • Tutorials — displays a set of links to different sections of the IDE Tutorials.

  • Samples — displays a set of links to different sections of the Language and Interfaces Examples manual.

  • What's New — displays a set of links to:

    • The Release Notes for the three features of CPLEX Optimization Studio.

    • A set of links to various optimization user forums. These links are driven by an RSS feed, so they are constantly updated to reflect the latest information on those forums.

To close the Welcome page and use the CPLEX Studio IDE

  • Click the X in the Welcome tab to close the page.

  • Or, click the Workbench icon at the top right of the Welcome page.

  • When you close the Welcome window using either method, the main window appears. It is described in the next section.

To return to the Welcome page from the IDE

Select Help > Welcome from the main menu.
