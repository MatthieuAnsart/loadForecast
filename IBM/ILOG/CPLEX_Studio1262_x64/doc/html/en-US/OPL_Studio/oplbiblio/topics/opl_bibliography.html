Bibliography

Provides some useful optimization references.

The following books have been used as sources in the documentation of IBM ILOG CPLEX Studio, in particular in the Language User’s Manual and in the Language and Interfaces Samples manual. More bibliographic references are provided in the glossary.

  • Model Building in Mathematical Programming

    Author: H. Paul Williams

    Publisher: John Wiley & Sons, Ltd.

    Date: 2005, Fourth Edition

  • Applications and Algorithms

    Author: W. Winston

    Publisher: PWS Publishing

    Date: Latest edition 2002

  • Combinatorial Optimization: Algorithms and Complexity

    Authors: C.H. Papadimitriou and K. Steiglitz

    Publisher: Prentice-Hall, Englewood Cliffs, NJ.

    Date: Latest edition 1998
