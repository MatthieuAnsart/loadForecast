CPLEX Studio IDE

You can learn about the CPLEX Studio IDE through tutorials and a reference manual.

  • Getting Started with the IDE
    An Introduction to the CPLEX Studio IDE, and the Getting Started Tutorial explain how to launch the CPLEX Studio IDE, create an empty project, enter an OPL model, add data and a settings file, create run configurations and execute the final project.
  • IDE Tutorials
    The tutorials use examples from the product distribution to illustrate typical features and use cases of the CPLEX Studio IDE.
  • IDE Reference
    Many of the IDE elements are described in the tooltips that are displayed when a mouse hovers over them. This document provides more detailed descriptions, and some explanations of how to use the IDE.
