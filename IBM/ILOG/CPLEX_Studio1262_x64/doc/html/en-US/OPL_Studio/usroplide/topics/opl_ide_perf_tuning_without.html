Tuning without fixed settings

Provides a procedure for tuning a model in the CPLEX Studio IDE.

Procedure

  1. Right-click on the warehouse project and select Tune project .

    The tuning wizard opens.

    Figure 1. Tuning wizard

    By default, all the run configurations defined in the project are selected. Deselect the ones you don’t want to tune. If no run configurations are selected, the Finish button is unavailable.

  2. In this exercise, deselect all configurations except Scalable data.
  3. In the Result file area, you need to provide a name for the settings file (.ops) output.
  4. Click Finish. During the tuning process, the Engine Log reports the percentage of progress, the settings worked on at each solve iteration, the execution time, the objective value, the best bound value. "Time limit exceeded" means that the particular tuning iteration took longer than the fastest iteration run so far.
    Note:

    Please note that the progress of the tuning operation is shown in two different locations — in the Engine Log, and in the IDE Status Bar. The percentages shown in these two locations may not be identical.
