Temporary tuning files

Temporary files created by the tuning process can be saved and examined later.

About this task

You can save the temporary .sav files created by the tuning process, to examine their content later.

Procedure

  1. Add a settings file as explained in Adding a settings file in Getting Started with the IDE.
  2. Make sure you add it to the run configurations you want to tune.
  3. In the new settings file, check the option Language > General > Keep temporary tuning files.
  4. Enter a target directory for the temporary files.
    Figure 1. Keeping temporary tuning files

    Next time you tune run configurations of the project, you will find the corresponding .sav file in the directory you have selected. You can reuse this file in CPLEX.
