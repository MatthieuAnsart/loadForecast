Changing the data

Shows how to change the data in the nurses.dat file.

About this task

The list of required assignments is given in the data file nurses.dat.

In this example, it appears that the assignment to relax is the fifth value from the left in the top row.

Procedure

  1. Open nurses.dat in the Editing Area and scroll to the RequiredAssignments section.
  2. Change it to 0 to make the model feasible.
