Coloring of CP keywords and functions in the IDE

Constraint programming in the CPLEX Studio IDE is distinguished by the colored names of the functions.

If you double-click steelmill.mod to open it in the IDE editor, you will notice that the first support of constraint programming in the IDE is the colored names of the functions used in the model to define specialized constraints, for example pack on line 54.
