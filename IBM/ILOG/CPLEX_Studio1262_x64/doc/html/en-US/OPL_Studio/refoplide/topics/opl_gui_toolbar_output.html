Toolbars in the output tabs

Provides information about the options available on the different output tabs of the IDE.

The buttons and toolbar options available on the different output tabs of the IDE vary from tab to tab. Some exist only on one tab, others exist on several different tabs.

Point to these toolbar buttons to see tooltips for them.

  • Clear log/Clear Console – on the Scripting log, Solutions, Engine log and Console tabs, clears the current contents of the view.

  • Copy contents to clipboard – on the Conflicts, Relaxations, and Profiler tabs, copies the current contents of the view to the Windows clipboard.

  • Customize thresholds – on the Profiler tab, allows you to set threshold preferences.

  • Detailed memory format – on the Profiler tab, toggles the display to show more details of memory usage.

  • Display Selected Console – on the Console tab, opens a view of the type selected from the drop-down list.

  • View Menu – on the Problems tab, reveals a drop-down menu specific to the tab.

  • Open Console – on the Console tab, opens a view of the type selected from the drop-down list.

  • Pin Console – on the Console tab, ensures that the current Console tab remains on top (for example, if you had multiple Console tabs open)

  • Scroll Lock – on the Scripting log and Engine log tabs, stops the scrolling of output in the window.
