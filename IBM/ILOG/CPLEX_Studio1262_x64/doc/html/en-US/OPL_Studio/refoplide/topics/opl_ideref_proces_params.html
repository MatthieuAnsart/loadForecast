Passing parameters to a remote process

Explains how to pass parameters to the virtual memory when a run configuration is executed on a remote process.

You can execute a run configuration in a remote process if you need extra memory, for example, to execute large models.

In the IDE, click the arrow to the right of the Run button and select Run Configurations.

In the Run Configurations window, select the Arguments tab and enter the parameters for the remote process in which you want to execute a run configuration.
