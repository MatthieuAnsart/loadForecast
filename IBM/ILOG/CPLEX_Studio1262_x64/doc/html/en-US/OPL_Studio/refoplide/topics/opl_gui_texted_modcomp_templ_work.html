Working with model templates

Model templates can be selected from the code completion popup window whenever it is open.

Procedure

  1. To display the code completion window, press Ctrl + Space in the model editor.

    The popup window displays entries that are appropriate for the context of your current cursor location in the file.

  2. To filter the display, type one or more characters of the keyword or function you are searching for, before or after pressing Ctrl + space. For example, here the user has typed the letters su and then selected the template Subject to block.

    Note that when you select a template in the popup window a preview is displayed that allows you to see the contents of the template before you select it.

  3. To insert the template, double-click it or select it and press Enter. In this example, the user has selected the Subject to block template, and it is inserted into the editor at the cursor position:
  4. If the template contains variables, your cursor position is automatically shifted after insertion to highlight the first variable, so that you can replace the placeholder variable with the real one.

    If the template code contains multiple variables, you can navigate to them by pressing the Tab key.

    You can continue pressing Ctrl + space as you write your model, to see a list of possible keywords, functions, and templates at any point.
