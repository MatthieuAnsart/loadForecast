Help menu

Describes the options available from the Help menu in the IDE.

The commands of the Help menu are listed below. Some menu items may also have keyboard shortcuts associated with them.

Table 1. Help menu: commands and equivalent shortcuts
Commands in the Help Menu Shortcut Description
Welcome   Displays the Welcome window in the IDE.
Help Contents   Opens the documentation in its own viewer.
Search   Searches for an item in the Help Contents.
Dynamic Help   Displays the Help window in the IDE. If a keyword is selected, displays help for that keyword.
Key Assist Ctrl+Shift+L Displays a popup list of keyboard shortcuts.
Check for Updates    
Install New Software    
About IBM ILOG CPLEX Studio   Displays a window that indicates the version of CPLEX® Studio and the products used by this version of IBM® ILOG® CPLEX Studio. Also contains copyright information.
