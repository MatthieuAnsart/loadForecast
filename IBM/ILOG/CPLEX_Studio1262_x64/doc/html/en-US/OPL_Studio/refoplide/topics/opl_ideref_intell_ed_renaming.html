Renaming elements globally

When you rename an element in a model, all occurrences of the element are renamed.

In an OPL model, highlight an element and right click. Select Refactor > Rename from the menu.

A popup window allows you to enter a new name in the field.

If you click the Preview button, you can compare the original code with the code containing the renamed element.

When you click OK, all occurrences of the selected element are renamed.
