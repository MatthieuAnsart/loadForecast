The OPL editor

Describes the editor feature in the IDE, in particular the intelligent code completion that is aware of OPL syntax and OPL model structures.

  • Editor features
    Provides a table of the features available in the IDE editor.
  • Keyboard shortcuts
    Lists the keyboard shortcuts available to you in the CPLEX® Studio IDE.
  • Code completion
    Describes the features of the IDE editor that provide assistance when you enter OPL code.
  • Highlighting all occurrences of an OPL element
    When you select an OPL element in a model, all occurrences of the element are highlighted.
  • Finding and removing references to an OPL element
    When you select an OPL element in a model, the OPL editor identifies all references to the element. You can remove some, or all, of them.
  • Renaming elements globally
    When you rename an element in a model, all occurrences of the element are renamed.
  • Code formatting
    The code formatting function allows you to correctly format a selected block of code. This is particularly useful if you receive a file containing unformatted code.
  • LP viewer
    Export a model using the CPLEX solver to an LP file, and view the LP file in the IDE.
  • Export a model using CP to a CPO file
    You can export a model that uses a CP solver to a CPO file to help you debug the model.
  • Model templates
    Describes how to use the model templates feature in the IDE.
  • Local History and its related features
    How to track and compare different versions of your files as you edit them in the CPLEX Studio IDE.
  • 'Compare With' features
    How to compare files with each other and with Local History.
  • 'Replace With' features
    How to compare files with each other and replace the contents from other versions of the file in Local History.
