Setting programming options

Use the settings editor to modify options for the Optimization Programming Language (OPL), mathematical programming (MP) and constraint programming (CP).

About this task

Unlike the IDE Preferences, which apply to an IDE session and are saved with it, MP, CP, and OPL language output options are set within an .ops file associated with a project and run configuration. When you double-click an .ops file in a project or run configuration, the settings editor opens (see nurses.ops in the screen capture below).

Procedure

To access the option pages:

  1. Make sure the project is selected.
  2. Create a settings file as explained in Adding a settings file in Getting Started with the IDE.

Results

In the Editing Area, you see the settings editor.
