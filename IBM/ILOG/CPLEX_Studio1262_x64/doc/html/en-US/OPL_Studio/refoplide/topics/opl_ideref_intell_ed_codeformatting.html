Code formatting

The code formatting function allows you to correctly format a selected block of code. This is particularly useful if you receive a file containing unformatted code.

If you enter your OPL code quickly, without stopping to format, or if you have received a file with unformatted code, you can highlight a block of code, then right click and select Source > Format selection.

The code is formatted with the correct indentations and line length.

You can change the formatting options by selecting Preferences from the menu and then Formatter in the Preferences dialog box.
