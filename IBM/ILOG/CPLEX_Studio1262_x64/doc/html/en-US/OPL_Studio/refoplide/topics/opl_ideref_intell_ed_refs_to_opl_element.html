Finding and removing references to an OPL element

When you select an OPL element in a model, the OPL editor identifies all references to the element. You can remove some, or all, of them.

In an OPL model, right click on an OPL element and select Find references.

The number of references to OPL elements, and the lines on which they occur, are displayed below the editor.

You can remove some, or all, of the matches by clicking on one of the buttons Remove Selected Matches (Delete) or Remove All Matches in the output console.

Alternatively, you can right click and select the same options.
