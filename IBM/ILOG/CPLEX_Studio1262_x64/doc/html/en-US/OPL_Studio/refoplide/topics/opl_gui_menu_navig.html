Navigate menu

Describes the options available from the Navigate menu in the IDE.

The commands of the Navigate menu are listed below. Some menu items may also have keyboard shortcuts associated with them. These commands are active when the cursor is in the editing area.

Table 1. Navigate menu commands
Commands in the Navigate menu Shortcut Description
Last Edit Location Ctrl+Q  
Back Alt+Left  
Forward Alt+Right  
Go to Line Ctrl+L Displays the Go To dialog box for specifying a line where the cursor should be placed in the Editing Area.
