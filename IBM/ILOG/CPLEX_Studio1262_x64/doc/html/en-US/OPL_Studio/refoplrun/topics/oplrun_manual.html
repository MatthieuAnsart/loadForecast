Executing oplrun manually

You can open a Command Prompt window or a UNIX command line and enter the oplrun command manually.

Procedure

  1. Open a Command Prompt window.

    When you work from Windows XP, click Start > Accessories > Command Prompt. This sequence may be different on previous versions of Windows.

  2. Place the cursor in the directory <Install_dir>\opl\bin\<platform_name_and_version>\
  3. Enter the oplrun command using the appropriate options as described in Syntax. Make sure you specify the complete path to the OPL model, data, or project file.
