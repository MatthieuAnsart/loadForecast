Executing oplrun from the IDE

If the project you are working on is already open in the IDE, you can start oplrun from the IDE.

Procedure

  1. To launch oplrun interactively (that is, to select a run configuration in OPL Projects Navigator and launch it in its own Console tab in the Output Area), follow the instructions in Creating an oplrun launch configuration interactively.
  2. To create a saved and reusable launch configuration for oplrun, follow the instructions in Creating a customized oplrun launch configuration and External Tools Dialog options.
  3. When oplrun is launched from within the CPLEX Studio IDE, the output is displayed in the Console tab of the Output Area. To learn more about the toolbar options available to you there, see Toolbars in the output tabs.
