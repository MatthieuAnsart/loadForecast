Using the command line interface

Describes the different ways in which you can run an OPL project or model using the oplrun executable. You can open the Command Prompt window either from the IDE or manually.

  • Executing oplrun from the IDE
    If the project you are working on is already open in the IDE, you can start oplrun from the IDE.
  • Executing oplrun manually
    You can open a Command Prompt window or a UNIX command line and enter the oplrun command manually.
