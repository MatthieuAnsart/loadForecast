Setting CP options

How to consult or change option values for constraint programming.

Procedure

  1. Create a settings file for your project or run configuration as indicated in Setting programming options.
  2. Double-click the settings (.ops) file in OPL Projects Navigator to open an editor for it.
  3. Navigate to Constraint Programming in the left pane.
  4. For details on how to set constraint programming options, read the sections Propagation control options and Search control/Log control.
