Setting MP options

How to consult or change option values for mathematical programming.

Procedure

  1. Create a settings file for your project or run configuration as indicated in Setting programming options.
  2. Double-click the settings (.ops) file in OPL Projects Navigator to open an editor for it.
    Figure 1. Mathematical programming: General options
  3. Navigate to Mathematical programming>General in the left pane of the editor to view or change the general settings.
    Note:

    You can filter the available parameters in the text box at the top of the parameter list. For example, type “Algorithm” in the text box to see only those categories that contain a setting with the word Algorithm.

  4. Navigate to other options categories in the left pane to view or change settings for those categories.
  5. See the reference manual, Parameters of CPLEX for details about a specific option.
