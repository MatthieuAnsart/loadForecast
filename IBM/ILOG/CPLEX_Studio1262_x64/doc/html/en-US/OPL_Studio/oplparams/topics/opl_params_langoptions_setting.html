Setting language options

How to consult or change OPL language options.

Procedure

  1. Create a settings file for your project or run configuration as indicated in Setting programming options.
  2. Double-click the settings (.ops) file in the OPL Projects Navigator to open an editor for it.

    Navigate to OPL Language>General in the left pane of the editor to view or change the general settings, as shown in the following screen capture.

    Figure 1. OPL Language: General options
  3. Navigate to OPL Language>Run in the left pane of the editor to view or change the Run settings, as shown below.
    Figure 2. OPL Language: Run options
  4. See the table of OPL language options for details on each option.
