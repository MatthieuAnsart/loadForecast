Propagation control options

Describes the options that enable you to control propagation in constraint programming models.

Click Constraint Programming>Propagation control on the left, as shown in the following screen capture.

Figure 1. CP options: Propagation Control

In the table below, Propagation Control options are ordered alphabetically.

Table 1. CP Propagation control options 
Name Description Default Value
AllDiff inference level Specifies the inference level for every constraint allDifferent extracted to the invoked IloCP instance. The default value allows the inference strength of all allDifferent constraints to be controlled via the DefaultInferenceLevel parameter. Default
AllMin distance inference level Specifies the inference level for every constraint allMinDistance extracted to the invoked IloCP instance. The default value allows the inference strength of all allMinDistance constraints to be controlled via the DefaultInferenceLevel parameter. Default
Count inference level Specifies the inference level for every constraint count extracted to the invoked IloCP instance. The default value allows the inference strength of all count constraints to be controlled via the DefaultInferenceLevel parameter. Default
Cumul function inference level Controls the inference level of constraints defined on cumulative function expressions. Basic
Default inference level Specifies the general inference level for constraints whose particular inference level is the default. Possible values for this parameter (in increasing order of inference strength) are Low, Basic, Medium, and Extended. Basic
Element inference level Specifies the inference level for every element constraint (created from IloIntVarArray::operator[]) extracted to the invoked IloCP instance. The default value allows the inference strength of all element constraints to be controlled via the DefaultInferenceLevel parameter. Default
Interval sequence inference level Controls the internal inference level of sequence variables. Basic
No overlap inference level Controls the inference level of the no-overlap constraints. Basic
Precedence inference level Controls the inference level of precedence constraints. Basic
Presolve Controls the aggregation of some groups of basic constraints to less basic ones in order to achieve better domain reduction. On
Propagation log

Turns on the propagation trace and sets the level of reporting. By default, the propagation trace is turned off.

  • Quiet: no trace

  • Terse: reports constraint propagation at the root; also reports failures.

  • Verbose: reports all constraint propagation; that is, propagation of constraints expressed explicitly in the model (marked P in the trace) as well as constraints added internally to maintain extractable objects (marked M in the trace); also reports failures; labels the report with names of branches.
