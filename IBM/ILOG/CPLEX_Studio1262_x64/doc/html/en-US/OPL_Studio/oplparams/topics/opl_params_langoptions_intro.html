Introduction

Provides an overview of the options available for the display of OPL language items.

You can customize the display of OPL language items in the Output window for an open project by changing language options in the project. Contrary to Preferences options, which are settings that apply to an IDE session and are saved with it, OPL Language options are set within an .ops file associated with a project and run configuration.
