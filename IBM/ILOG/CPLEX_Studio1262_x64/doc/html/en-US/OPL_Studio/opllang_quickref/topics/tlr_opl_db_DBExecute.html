DBExecute

OPL keyword for database interaction.

Purpose

OPL keyword to create, remove, or change data tables from .dat files

context Data files (.dat)

Syntax

Publish: "DBExecute" '(' Id String ')'
         | Id "to" "DBUpdate" '(' Id String ')'
         | Id "to" "SheetWrite" '(' Id String ')'

Description

Use DBExecute statements in the data file to:

  • create the table that will hold the results of the model solve in the database so that the database can be updated

  • remove a data table

  • change the structure a data table 

DBExecute(db,"drop table Result");
DBExecute(db,"create table Result(oil varchar(10), gas varchar(10), blend real, a real)");
Result to DBUpdate(db,"INSERT INTO Result(oil,gas,blend,a) VALUES(?,?,?,?)");

This example illustrates how to send SQL DDL (data definition language) instructions to the Relational Database Management Server (RDBMS).
