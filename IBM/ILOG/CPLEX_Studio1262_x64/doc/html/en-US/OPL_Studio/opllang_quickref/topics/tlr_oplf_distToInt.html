distToInt

OPL function to return the distance from a value to the nearest integer

Purpose

OPL function to return the distance from f to the nearest integer.

type 
float

Syntax

distToInt(float f)

Description

Returns the distance from a floating-point value to the nearest integer. You can use this function within IBM ILOG Script statements by specifying the OPL namespace:

( Opl.xxx() )

Example

float a=-2.5;
float b=3.875;

float distToIntofa=distToInt(a);
float distToIntofb=distToInt(b);

execute
{
   writeln("distToInt(",a,") gives ",distToIntofa);
   writeln("distToInt(",b,") gives ",distToIntofb); 
}

Result

distToInt(-2.5) gives 0.5
distToInt(3.875) gives 0.125
