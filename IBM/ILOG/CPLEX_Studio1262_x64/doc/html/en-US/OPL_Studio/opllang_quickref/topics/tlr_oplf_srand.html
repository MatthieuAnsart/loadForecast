srand

OPL function for a generator of pseudorandom numbers

Purpose

OPL function to initialize a generator of pseudorandom numbers.

type 
int

Syntax

int srand(int)

Description

Initializes a generator of pseudo random numbers. You can use this function within IBM ILOG Script statements by specifying the OPL namespace:

( Opl.xxx() )

Example

int whatever=srand(2);

int r1[i in 1..5]=rand(10);
int r2[i in 1..5]=rand();

execute
{
  writeln("srand(2) gives ",whatever);
  writeln("int r1[1..5]=rand(10) gives ",r1);  
 writeln("int r2[1..5]=rand() gives ",r2);

}

Result (true only on 32–bit Windows)

srand(2) gives 2
int r1[1..5]=rand(10) gives  [2 3 2 6 0]
int r2[1..5]=rand() gives  [2056603190 2001300612 411429355 735724454 1226205810]
