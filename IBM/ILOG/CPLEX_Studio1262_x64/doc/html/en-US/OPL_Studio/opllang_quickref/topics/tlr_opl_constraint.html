constraint

OPL keyword to declare constraint arrays.

Purpose

OPL keyword to declare the constraint array that will receive labeled constraints.

context Model files (.mod)

Syntax

LiteralType: "int"
           | "float"
           | "boolean"
           | "int+"
           | "float+"
           | "string"
           | "range"
           | "range" "float"
           | "constraint"

Description

If you use indexed labels, this keyword enables you to declare the constraint array that will receive the labeled constraints before adding the indexing expressions to the label.

constraint ctSupply[Products][Cities];
...
ctSupply[p][o]: 
...

See Constraint labels in the Language Reference Manual.
