IBM ILOG Script keywords and functions for OPL models

A list of the IBM ILOG Script keywords implemented for OPL

You can use the following IBM ILOG Script keywords, functions and operators in scripting statements within OPL models.

Important:

You cannot use IBM ILOG Script keywords and functions as variable names.
