Date

IBM ILOG Script keyword used to manipulate date and time data.

Dates can be considered as internally represented by the number of milliseconds since 00:00:00 UTC, January 1, 1970. The date number can be negative and can extend forward about 285,616 years from 1970.

For information about date constructors, methods, functions and operators, see Dates in the Language Reference Manual.
