SheetConnection

OPL keyword to connect to a spreadsheet.

Purpose

OPL keyword to connect to a spreadsheet (external data initialization).

context Data files (.dat)

Syntax

Connection: "DBConnection" Id '(' String String ')'
          | "SheetConnection" Id '(' String ')'

Description

There are basically two ways of initializing data in OPL:

  • Internal Initialization consists in initializing the data at the same time as it is declared. Inline initializations can contain expressions to initialize data items, such as 

    int a[1..5] = [b+1, b+2, b+3, b+4, b+5];

  • External Initialization consists in specifying initialization subsequently as an OPL statement in a separate data file (.dat). This includes reading from a spreadsheet.

In the context of spreadsheet initialization, you use the keyword SheetConnection to connect to one or several spreadsheets, as shown in the following example.

Example

In the oilSheet example, the instruction

SheetConnection sheet("transport.xls");

establishes a connection sheet to a spreadsheet named “transport.xls”. The connection sheet should be viewed as a handle on the spreadsheet.
