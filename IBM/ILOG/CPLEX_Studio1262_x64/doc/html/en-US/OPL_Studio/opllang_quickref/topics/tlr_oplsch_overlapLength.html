overlapLength

OPL function (CP, scheduling) to return the length of the overlap of two interval variables, or an interval variable and a constant interval.

Purpose

OPL function (CP, scheduling) that returns an expression representing the length of the overlap of two interval variables, or an interval variable and a constant interval.

context Model files (.mod)
type 
dexpr int

Syntax

dexpr int overlapLength(dvar interval intervalVar, dvar interval intervalVar)
dexpr int overlapLength(dvar interval intervalVar, dvar interval intervalVar, int absValue)
dexpr int overlapLength(dvar interval intervalVar, int start, int end)
dexpr int overlapLength(dvar interval intervalVar, int start, int end, int absValue)

Description

This function returns an integer expression that represents:

  • The length of the overlap of interval variable a and interval variable b, whenever the interval variables a and b are present. If the interval variable a or b is absent, the function returns the constant integer value absValue (zero by default).

  • The length of the overlap of interval variable a and the constant interval [start, end), whenever the interval variable a is present. If the interval variable a is absent, the function returns the constant integer value absValue (zero by default).

Example

using CP;
dvar interval a in 1..6 size 5;
dvar interval b in 1..20 size 10;
subject to {
   overlapLength(a, b) <= 3;
   startOf(a) >= 1;
   startOf(b) >= 1;
}
execute {  
   writeln(Opl.overlapLength(a, b));
}
