pow

OPL “power” function

Purpose

OPL function to return x y.

type 
float

Syntax

pow(float x,float y)

Description

Returns x y. x can be a decision variable or a constant. The exponent y must be a constant (data or literal).

You can use this function within IBM ILOG Script statements by specifying the OPL namespace:

( Opl.xxx() )
Note:

The expression a b is accepted when b is the constant zero. The expression evaluates to unity, even if a is zero. That is, we follow the convention that pow(0,0) = 1.

Example

float p1=pow(4,0.5);
float p2=pow(2,5);


execute
{
 
 writeln("pow(4,0.5) gives ",p1);
 writeln("pow(2,5) gives ",p2);
}

Result

pow(4,0.5) gives 2
pow(2,5) gives 32
