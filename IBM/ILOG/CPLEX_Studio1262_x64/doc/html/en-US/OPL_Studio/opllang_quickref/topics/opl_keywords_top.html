    OPL aggregate to express cumulative constraints.

    OPL keyword to check some assumptions.

    OPL keyword.

    OPL keyword to declare constraint arrays.

    OPL keyword to specify CP.

    OPL keyword for connection to a database

    OPL keyword for database interaction.

    OPL keyword to modify data in databases.

    OPL keyword for false constraints
  • float
    OPL keyword for floats.

    OPL keyword to specify constraints.
  • from

  • if
    OPL keyword for conditional constraints

    OPL keyword for ascending sort order

    This OPL keyword invokes the SPSS Connector, which connects CPLEX Studio to a data stream in SPSS Modeler. The SPSS Connector is available on Windows 32-bit and 64-bit platforms.

    Reads data from an SPSS stream into an OPL tuple set, via SPSS Modeler.

    OPL keyword (CP, scheduling) to create a state function.

  • true
