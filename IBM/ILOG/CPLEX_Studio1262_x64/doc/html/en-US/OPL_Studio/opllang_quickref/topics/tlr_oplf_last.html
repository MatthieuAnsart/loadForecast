last

OPL function to return the last element in a set.

Purpose

OPL function to return the last element in a set. For the scheduling function on interval variables, see last (scheduling).

type 
int, float, string, or a tuple type

Syntax

int last({int})
float last({float}) 
string last({string}) 
<tuple type> last({<tuple type>})

Description

Returns the last element in a set. You can use this function within IBM ILOG Script statements by specifying the OPL namespace:

(Opl.xxx())

Example

{string} Days ={"Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"};

string l = last(Days);

execute
{
 writeln("last(",Days,") gives ",l);  
}

initializes l to Sunday. This also indicates that sets in OPL are totally ordered according to their order of appearance in the initialization.

Result

last( {"Monday" "Tuesday" "Wednesday" "Thursday" "Friday" "Saturday" "Sunday"}) gives Sunday
