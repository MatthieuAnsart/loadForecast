key

OPL keyword for tuple declaration

Purpose

OPL keyword for access to data organized in tuples.

context Model files (.mod)

Syntax

TupleComponent: Type VariableDeclarator ';'
                | "key" Type VariableDeclarator ';'

Description

The key keyword used in declarations of tuples enables you to access data organized in tuples using a set of unique identifiers. See Keys in tuple declarations in the Language Reference Manual.

Example

In the following example, the nurse tuple is declared with the key name of type string.

tuple nurse {
  key string name;
  int seniority;
  int qualification;
  int payRate;
}

In this second example, a nonsingleton set of keys is used to declare the shift tuple.

tuple shift {
   key string departmentName;
   key string day;
   key int startTime;
   key int endTime;
   int minRequirement;
   int maxRequirement;   
}
