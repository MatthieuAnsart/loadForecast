SPSSConnection

This OPL keyword invokes the SPSS Connector, which connects CPLEX Studio to a data stream in SPSS Modeler. The SPSS Connector is available on Windows 32-bit and 64-bit platforms.

Context

Data files (.dat)

Syntax

SPSSConnection spss("your_file.str", "");
where your_file.str is a data stream in SPSS Modeler.
The second argument, the empty string "", is for future extensions of SPSS Connector.

Description

If SPSS Modeler is installed on the same machine as the CPLEX Studio IDE, you can establish a connection to SPSS Modeler from the IDE using SPSSConnection.

Activating the keyword

The prepare statement activates the keyword SPSSConnection in CPLEX Studio. You need to write the following code in an OPL.dat file:

prepare {includeScript("oplspss.js");}

Example

prepare {includeScript("oplspss.js");}
SPSSConnection spss("pm_selflearn.str", "");
