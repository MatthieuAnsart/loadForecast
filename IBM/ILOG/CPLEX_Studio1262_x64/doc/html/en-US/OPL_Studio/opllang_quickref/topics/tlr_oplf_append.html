append

OPL function to concatenate int arrays and integer variables

Purpose

OPL function to concatenate integer arrays and interval variables.

context Model files (.mod) - Not allowed in a CPLEX constraint block.
type 
int[]
interval[]

Syntax

append( int, int, ... )
append( int[], int, ... )
append( int[], int[], ... )

append( interval, interval, ... )
append( interval[], interval, ... )
append( interval[], interval[], ... )

Description

This function enables you to concatenate integer arrays and integer variables. You can use this function within IBM ILOG Script statements by specifying the OPL namespace:

( Opl.xxx() )

Example

using CP;

range R=1..10;
dvar int x[R] in 0..20;
dvar int y[R] in 0..20;

minimize sum(i in R)(x[i]+y[i]);
subject to
{
  allDifferent(append(all(i in 1..2) x[i],all(i in 4..6) y[i]));
}
  noOverlap(append(all (h in houses, w in workerNames, t in taskNames) normalPlan[h][w][t], 
                   all (h in lines, w in workerNames, nt in newTasks) newPlan[h][w][nt]));
