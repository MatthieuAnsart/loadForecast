IBM ILOG Script keywords and functions

Lists the words reserved for IBM ILOG Script. Some of them are used as keywords, others are reserved for future use.

    A definition of the kind of statements you can write with IBM ILOG Script
  • Keywords common to OPL and IBM ILOG Script
    Identifies keywords that are common to OPL and IBM ILOG Script.
  • IBM ILOG Script keywords and functions for OPL models
    A list of the IBM ILOG Script keywords implemented for OPL
  • Array
    IBM ILOG Script keyword to create a new array.
  • break
    IBM ILOG Script keyword to break a routine.
  • continue
    IBM ILOG Script keyword to continue the current routine.
  • Date
    IBM ILOG Script keyword used to manipulate date and time data.
  • delete
    IBM ILOG Script special operator used to remove variables or names.
  • eval
    IBM ILOG Script function to execute a string as a program.
  • for
    IBM ILOG Script keyword used to create a loop or iterate over properties in order to execute a statement.
  • function
    IBM ILOG Script keyword that calls a function.
  • infinity
    IBM ILOG Script keyword that is a special number representing infinity.
  • Math
    IBM ILOG Script keyword used to access certain math functions and constants.
  • NaN
    IBM ILOG Script keyword that is a special number used to indicate errors in number manipulations.
  • new
    IBM ILOG Script special operator used as constructor.
  • null
    IBM ILOG Script keyword used to specify an absence of information.
  • Number
    IBM ILOG Script keyword for number representations and functions.
  • Object
    IBM ILOG Script keyword for object representation and function.
  • return
    IBM ILOG Script keyword to return a value from a function.
  • static
    IBM ILOG Script keyword used with function definition.
  • String
    IBM ILOG Script keyword.
  • this
    IBM ILOG Script keyword used to return an object.
  • typeof
    IBM ILOG Script special operator to return a string representing a value.
  • var
    IBM ILOG Script keyword used to declare a script variable.
  • while
    IBM ILOG Script keyword to create an execution loop.
  • write
    IBM ILOG Script function to print a string.
  • writeln
    IBM ILOG Script function to print a string with a new line at end of output.
  • IBM ILOG Script keywords reserved for future use
    Lists the keywords that are currently not implemented in OPL and, therefore, not available for use in scripting blocks.
