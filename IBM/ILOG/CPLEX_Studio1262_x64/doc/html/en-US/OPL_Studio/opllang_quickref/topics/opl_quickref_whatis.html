What is OPL?

A brief definition of OPL as a programming language to express optimization problems.

OPL (Optimization Programming Language) is a modeling language for combinatorial optimization, designed to substantially simplify optimization problems.

More specifically:

  • OPL combines the strengths of mathematical programming and constraint programming modeling languages.

    • It increases the applicability of modeling languages by incorporating techniques from constraint programming.

    • It improves the expressive power of traditional constraint programming tools by borrowing ideas from modeling languages.

  • OPL supports scheduling and resource allocation applications, which are ubiquitous in industry.

  • OPL provides novel modeling, as well as access to special-purpose algorithms such as the edge-finder procedure.

OPL as a modeling language

OPL shares many features with other modeling languages and provides computer equivalents for traditional algebraic notations. It preserves the strengths of modeling languages in linear and integer programming.

OPL also allows logical combinations of constraints and higher-order constraints.

OPL statements

An OPL model definition for an optimization application typically contains:

  • a sequence of declarations that include data initialization

  • preprocessing and postprocessing

  • constraints

  • an objective
