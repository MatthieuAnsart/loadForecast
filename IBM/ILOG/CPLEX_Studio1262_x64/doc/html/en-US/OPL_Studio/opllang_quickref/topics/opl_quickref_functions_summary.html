Summary table of OPL functions

Provides the syntax and short description of each OPL function, with a link to the corresponding full description.

This table lists the predefined functions available in OPL. It provides the syntax and short description of each function, with a link to the corresponding individual full description. They are of the form:

returnType functionName(argumentType [,argumentType]*)

All functions marked CP cannot be used in constraint blocks of CPLEX models.

You can use OPL functions within IBM ILOG Script statements by specifying the Opl namespace, as follows:

(Opl.xxx())

For example, for the abs function:

int i = 10;

execute {
   var resu = Opl.abs(i);
}

In OPL, when the code contains a table, you can also use the OPL keyword all. For an example, see the function allDifferent.
