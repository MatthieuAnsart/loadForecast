rand

OPL function to return a pseudorandom integer

Purpose

OPL function to return a pseudorandom integer.

type 
int

Syntax

rand()
rand(int)

Description

rand() returns a pseudorandom integer.

rand(int m) returns a pseudorandom integer modulo m. You can use this function within IBM ILOG Script statements by specifying the OPL namespace:

( Opl.xxx() )

Example

int r1[i in 1..10]=rand(10);
int r2[i in 1..10]=rand();

execute
{
 writeln("int r1[1..10]=rand(10) gives ",r1);  
 writeln("int r2[1..10]=rand() gives ",r2);

}
