A few things to know about the Language Quick Reference.

Make sure that you read Navigating in the documentation set for a description of the documentation set plus details on prerequisites, conventions, documentation formats, and other general information.

This Language Quick Reference guide describes individual keywords and functions, as well as operators and grammar for OPL and IBM ILOG Script for OPL.

It contains:

  • OPL keywords

  • OPL functions

  • IBM ILOG Script keywords and functions

  • OPL Operators and OPL Syntax

This guide functions as both a standard reference publication and as an online help. To use it as contextual help in the IDE, simply click on any colorized word in a model file (*.mod), press F1, and relevant items appear in the Help window. This manual is also available in its entirety in the IDE, along with the complete OPL documentation set, by selecting Help > Help Contents from the toolbar.
