lengthEval

OPL numerical expression (scheduling) used to evaluate a piecewise linear function on the length of an interval

Purpose

OPL numerical expression (scheduling) used to evaluate a piecewise linear function on the length of an interval.

context Model files (.mod)
type 
dexpr float

Syntax

dexpr float lengthEval (a,f[,dval]);
 
Where:

pwlFunction f;
dvar interval a;
float dval;

Description

This numerical expression is used to evaluate a piecewise linear function at the length of an interval (interval length is end time minus start time). Special care must be taken for optional intervals: in this case a float value dval must be specified, to represent the value of the expression when the interval is absent. If this value is omitted, it is considered to be 0.

For an example of using a numerical expression, see endEval.
