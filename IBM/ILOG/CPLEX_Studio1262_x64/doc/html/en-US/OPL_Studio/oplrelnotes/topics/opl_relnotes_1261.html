Release notes for IDE and OPL V12.6.1

These notes summarize new and improved features in the IDE and the OPL modeling language.

  • Using the CPLEX distributed MIP algorithm with OPL
    You can use the CPLEX distributed MIP algorithm to solve a standalone OPL model from the IDE, from the OPL APIs, or from OPLScript.
  • New settings
    New settings are available in the IDE settings editor, in the APIs, and in OPLScript.
  • Other changes
    This release includes upgrades to DB Link and ICU. There is also a new /MDd port, and a new OPL example, Flight Pairing. Newer versions of SPSS Modeler are now supported.
