Release notes for IDE and OPL V12.6.0

These notes summarize new and improved features in the IDE and the OPL modeling language.

  • New in the IDE
    IDE improvements include a smarter editor that is more aware of OPL syntax and OPL model structures. The IDE also contains a new OPL search function, an LP Viewer, and supports the export of CP files.
  • New in the modeling language
    There are new OPL constraints to be used in constraint programming.
