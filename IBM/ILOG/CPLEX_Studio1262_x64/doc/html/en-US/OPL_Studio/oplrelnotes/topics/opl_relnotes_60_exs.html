Examples

OPL 6.0 provides examples to illustrate the scheduling feature and an example of how to use a CPLEX deterministic model to handle a stochastic problem.

Examples to illustrate the scheduling feature can be found in the examples directory.

  • <Install_dir>\examples\opl\sched_*

  • <Install_dir>\examples\opl\models\sched_*

There is also a new example of how to use a CPLEX deterministic model to handle a stochastic problem. It can be found in the examples directory.

<Install_dir>\examples\opl\models\YieldStochastic
