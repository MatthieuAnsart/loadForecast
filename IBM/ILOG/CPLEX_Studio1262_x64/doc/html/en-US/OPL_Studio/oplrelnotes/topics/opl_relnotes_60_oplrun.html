oplrun

You can start the oplrun line command from the Run menu in the OPL IDE: Run > External Tools > Run As > oplrun

  • You can start the oplrun line command from the Run menu in the standard toolbar of the IDE.

    Run > External Tools > Run As > oplrun

  • There is a new flag -locale. For information, see the document oplrun Command Line Interface

  • The flags -di, -de have been enhanced. They now support — as filename for output to stdout. For example:

    oplrun -di - -p ..\..\..\examples\opl\warehouse

  • .ops files support the new settings category Run .

  • The parameters on the Run page of the settings file are no longer restricted to the OPL IDE. They are now also available for oplrun and ODM.
