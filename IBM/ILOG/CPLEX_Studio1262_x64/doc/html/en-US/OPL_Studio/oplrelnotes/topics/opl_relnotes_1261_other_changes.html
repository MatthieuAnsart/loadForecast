Other changes

This release includes upgrades to DB Link and ICU. There is also a new /MDd port, and a new OPL example, Flight Pairing. Newer versions of SPSS Modeler are now supported.

DB Link

Rogue Wave DB Link has been upgraded to version 5.7.

ICU

ICU (International Components for Unicode) has been upgraded to version 5.1. ICU provides Unicode and Globalization support for OPL.

/MDd port

There is a new /MDd port of OPL for Visual Studio 2010 and Visual Studio 2012. The port directory is named stat_mdd.

Flight Pairing example

The Flight Pairing example replaces the Crew Scheduling example. It can be found in the installation directory, for example:

C:\Program Files\IBM\ILOG\CPLEX_Studio1261\opl\examples\opl\models\flight_pairing

This example is described in Flight pairing, in the OPL document:

IDE and OPL > Language and Interfaces Examples > OPL model library.

SPSS Modeler

Version 14 of SPSS Modeler is no longer supported.

CPLEX Optimization Studio 12.6.1 supports SPSS Modeler versions 15 and 16.
