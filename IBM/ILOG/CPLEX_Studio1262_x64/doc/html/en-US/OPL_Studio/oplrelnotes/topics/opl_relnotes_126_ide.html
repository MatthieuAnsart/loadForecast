New in the IDE

IDE improvements include a smarter editor that is more aware of OPL syntax and OPL model structures. The IDE also contains a new OPL search function, an LP Viewer, and supports the export of CP files.

Smarter code completion

Code completion in the editor provides more "intelligent" filtering than previously.

When you activate code completion, via Ctrl + Space or by typing . (dot), the OPL editor is able to sort and filter elements according to the position of the cursor in the model structure - the modeling part or the scripting part. The popup window therefore offers a restricted and relevant list of options to select from, and insert into the model.

If you type 'c' after the dot, the list will contain only elements beginning with 'c'. In this example, "cost" is the only element available.

Editing an OPL model

When editing an OPL model, you can:

  • Highlight all occurrences of an element
  • Identify the locations of all references to an OPL element
  • Rename all occurrences of an OPL element simultaneously

In the following screen capture, three matches are found for the selected element TasksNotBreakable. In the pane below the model, the line numbers of all 3 occurrences are displayed.

To rename these elements globally, right click on one of them and select Refactor > Rename.

Exporting CP files

You can now export a CP model to a .cpo file to help you debug the model. You can also send the .cpo file to technical support if you need assistance.

Create a settings file for your CP project, and set the export format to CPO.

Add the settings file to the default run configuration, then run the configuration to produce a .cpo file for debugging purposes.

The OPL search

The search function operates on all the projects open in the workspace.

The OPL Search can be called "intelligent" as it is aware of the OPL syntax and model structure. For example, a search on "day" would return only OPL elements declared as "day", or referenced in models. It would not return "today", "weekday", "Monday", as an ordinary search would do.
