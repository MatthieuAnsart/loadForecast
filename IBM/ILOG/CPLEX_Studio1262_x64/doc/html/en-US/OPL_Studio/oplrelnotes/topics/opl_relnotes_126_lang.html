New in the modeling language

There are new OPL constraints to be used in constraint programming.

sameSequence

sameSequence is a binary constraint on a pair of sequence variables. The constraint states that the relative position of related interval variables is the same in both sequences.

sameCommonSubsequence

sameCommonSubsequence is a binary constraint on a pair of sequence variables. The constraint states that the relative position of some related subsets of interval variables is the same in both sequences.

strong

The strong constraint gives a strengthening indication to a set of variables.

More information
