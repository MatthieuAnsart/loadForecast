Release notes for CPLEX Studio V12.3

The IDE

Migration to Eclipse 3.6.2

The IDE is based on Eclipse 3.6.2, to take advantage of some of the latest Eclipse features.

Linux

The IDE is now available on Linux 32-bit and 64-bit platforms.

Interactive scripting

You can create script prototypes for pre- and post-processing of a model. The IDE contains a scrapbook in which you can write script code and run it to see the results immediately. It is no longer necessary to wait for a model to solve after each modification of script.

This feature is described in Interactive Scripting.

JViews 8.7

CPLEX Studio 12.3 supports JViews 8.7.

Modeling improvements

Isomorphism

A new scheduling function, isomorphism, is added to the modeling language, OPL. The function is used to model a one-to-one correspondence between the intervals present in two arrays. For more information see isomorphism.

New error message

Some constraints are documented as non-usable in meta-constraints. However, the use of meta-constraints containing such constraints was not detected by the CP Optimizer engine. Now, the engine sends an error message when meta-constraints involving non-usable constraints are used in a model.

Improved search

The CP Optimizer search is now more stable and less influenced by apparently insignificant changes to the model.

SPSS connector in CPLEX Studio

An OPL-SPSS connector is available in CPLEX Optimization Studio 12.3, on Windows 32-bit platforms. If SPSS Modeler is installed on the same machine as the Studio IDE, you can establish a connection to SPSS Modeler from the IDE. You can also establish a connection via OPL run time.

More information can be found in the SPSS tutorial: SPSS Connector in CPLEX Studio

Deprecated or obsolete in CPLEX Studio 12.3

The modeling language

The OPL operators < and > are no longer available for integer expressions.

Databases

Oracle 9 is no longer supported.

Visual Studio 2005

Microsoft Visual Studio 2005 is no longer supported.

Office 2003

Microsoft Office 2003 plug-in code is deprecated in CPLEX Studio 12.3.
