.NET interfaces

Changes were made to the .NET interfaces in OPL 6.0.

Renaming in .NET interfaces

All methods with the first character in lowercase have been renamed with the same name, and with the first character in uppercase.

Properties replaced by methods

In ILOG.CPLEX.Cplex, the following properties have been replaced by methods.

ParameterSet (getter) —> GetParameterSet()
ParameterSet (setter) —> SetParameterSet()
Objective —> GetObjective()
DeleteMode (getter) —> GetDeleteMode()
DeleteMode (setter) —> SetDeleteMode()
Algorithm —> GetAlgorithm()
IIS —> GetIIS()

Methods replaced by properties

In ILOG.OPL.OplElement:

isExternalData() —> IsExternalData

In ILOG.OPL.OplModel:

isInternalData() —> IsInternalData
getName() —> Name
getSourceName() —> SourceName

In ILOG.OPL.OplmodelDefinition:

isNull() —> IsExternalData

In ILOG.OPL.OplRunConfiguration:

setSettings() —> Settings
getSettings() —> Settings
getCP() —> CP

In ILOG.OPL.OplSettings:

isSkipAssert() —> IsSkipAssert
isSkipWarnNeverUsedElements() —> IsSkipWarnNeverUsedElements
getResolverPath() —> ResolverPath
setResolverPath() —> ResolverPath
getTmpDir() —> TmpDir
setTmpDir() —> TmpDir
isKeepTmpFiles() —> IsKeepTmpFiles
getRelaxationLevel() —> RelaxationLevel
setRelaxationLevel() —> RelaxationLevel
