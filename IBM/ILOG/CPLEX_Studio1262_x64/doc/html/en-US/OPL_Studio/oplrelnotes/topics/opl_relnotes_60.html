New and changed in OPL 6.0

These release notes highlight improvements and new features in OPL 6.0.

  • The OPL IDE
    New and changed features in the OPL IDE are summarized here.
  • Solving scheduling problems with constraint programming
    OPL 6.0 supports constraint programming for both combinatorial and scheduling problems.
  • Language
    New language elements were introduced in OPL 6.0.
  • .NET interfaces
    Changes were made to the .NET interfaces in OPL 6.0.
  • Databases
    OPL 6.0 supports Microsoft Access 2003 and Microsoft Office Access 2007 through the Office 2007 drivers.
  • oplrun
    You can start the oplrun line command from the Run menu in the OPL IDE: Run > External Tools > Run As > oplrun
  • Examples
    OPL 6.0 provides examples to illustrate the scheduling feature and an example of how to use a CPLEX deterministic model to handle a stochastic problem.
  • Port changes
    New ports available in OPL 6.0, and changes to existing ports, are summarized here.
