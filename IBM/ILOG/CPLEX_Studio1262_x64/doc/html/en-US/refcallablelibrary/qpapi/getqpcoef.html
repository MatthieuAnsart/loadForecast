CPXXgetqpcoef and CPXgetqpcoef

The routine CPXXgetqpcoef/CPXgetqpcoef accesses the quadratic coefficient in the matrix Q of a CPLEX problem object for the variable pair indexed by (rownum, colnum).

int CPXXgetqpcoef( CPXCENVptr env, CPXCLPptr lp, CPXDIM rownum, CPXDIM colnum, double * coef_p )

int CPXgetqpcoef( CPXCENVptr env, CPXCLPptr lp, int rownum, int colnum, double * coef_p )

Description

The routine CPXXgetqpcoef/CPXgetqpcoef accesses the quadratic coefficient in the matrix Q of a CPLEX problem object for the variable pair indexed by (rownum, colnum). The result is stored in *coef_p.

Arguments

Name Description
env A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.
rownum The first variable number (row number in Q).
colnum The second variable number (column number in Q).
coef_p A pointer to a double where the coefficient should be stored.

Return

The routine returns zero on success and nonzero if an error occurs.

Example

 status = CPXgetqpcoef (env, lp, 10, 20, &coef);
