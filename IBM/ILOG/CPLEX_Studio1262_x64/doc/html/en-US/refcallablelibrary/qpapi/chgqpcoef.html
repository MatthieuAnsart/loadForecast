CPXXchgqpcoef and CPXchgqpcoef

The routine CPXXchgqpcoef/CPXchgqpcoef changes the coefficient in the quadratic objective of a quadratic problem (QP) corresponding to the variable pair (i,j) to the value newvalue.

int CPXXchgqpcoef( CPXCENVptr env, CPXLPptr lp, CPXDIM i, CPXDIM j, double newvalue )

int CPXchgqpcoef( CPXCENVptr env, CPXLPptr lp, int i, int j, double newvalue )

Description

The routine CPXXchgqpcoef/CPXchgqpcoef changes the coefficient in the quadratic objective of a quadratic problem (QP) corresponding to the variable pair (i,j) to the value newvalue. If i is not equal to j, both Q(i,j) and Q(j,i) are changed to newvalue.

Arguments

Name Description
env A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.
i An integer that indicates the first variable number (row number in Q).
j An integer that indicates the second variable number (column number in Q).
newvalue The new coefficient value.

Return

The routine returns zero on success and nonzero if an error occurs.

Example

 status = CPXchgqpcoef (env, lp, 10, 12, 82.5);
