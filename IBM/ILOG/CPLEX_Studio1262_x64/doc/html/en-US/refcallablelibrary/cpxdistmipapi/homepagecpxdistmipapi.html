Distributed Parallel MIP Optimizer in the CPLEX Callable Library (C API) Reference Manual

These routines support applications that solve mixed integer programs in parallel across a distributed environment, such as multiple nodes in a cloud, multiple computers in a grid, multiple processors on a given platform, or other distributed computing environments.

  • CPXXdistmipopt and CPXdistmipopt
    Distributed parallel optimization of mixed integer programming models.
  • CPXXdistmipopt_async and CPXdistmipopt_async
    Asynchronous version of CPXXdistmipopt/CPXdistmipopt.
  • CPXXdistmipopt_join and CPXdistmipopt_join
    Asynchronous version of CPXXdistmipopt/CPXdistmipopt.
  • CPXXcopyvmconfig and CPXcopyvmconfig
    Load distributed parallel MIP configuration from a string.
  • CPXXreadcopyvmconfig and CPXreadcopyvmconfig
    Load distributed parallel MIP configuration from a file.
  • CPXXdelvmconfig and CPXdelvmconfig
    Delete a parallel machine configuration.
  • CPXXfinalizeMPIworkers and CPXfinalizeMPIworkers
    Finalize MPI workers that were started by distributed MIP.
  • CPX_FEATURE_DISTRIBUTED_MIP
    Specifies whether distributed parallel MIP is supported on this platform
