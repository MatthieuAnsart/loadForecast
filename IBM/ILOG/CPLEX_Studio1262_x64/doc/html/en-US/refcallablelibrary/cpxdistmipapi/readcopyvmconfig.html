CPXXreadcopyvmconfig and CPXreadcopyvmconfig

Load distributed parallel MIP configuration from a file.

int CPXXreadcopyvmconfig( CPXENVptr env, char const*filename )

int CPXreadcopyvmconfig( CPXENVptr env, char const*filename )

Description

Before you can run distributed parallel MIP (CPXXdistmipopt/CPXdistmipopt), you must load a parallel machine configuration. The parallel machine configuration tells the distributed parallel MIP algorithm which machines it can employ in order to optimize a problem.

This routine loads a parallel MIP machine configuration from a file The file must be in the VMC format. (See the topic VMC file format: virtual machine configuration for distributed parallel optimization for more information.)

CPLEX destroys any previously loaded parallel machine configuration before this routine attempts to load the new configuration.

Arguments

Name Description
env A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
filename The name of a file that contains a parallel machine configuration in VMC format.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example

 status = CPXreadcopyvmconfig (env, "distmip.vmconfig");
