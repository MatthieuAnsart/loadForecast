CPXXcopyvmconfig and CPXcopyvmconfig

Load distributed parallel MIP configuration from a string.

int CPXXcopyvmconfig( CPXENVptr env, char const*xmlstr )

int CPXcopyvmconfig( CPXENVptr env, char const*xmlstr )

Description

Before you can run distributed parallel MIP (CPXXdistmipopt/CPXdistmipopt), you must load a parallel machine configuration. The parallel machine configuration tells the distributed parallel MIP algorithm which machines it can employ in order to optimize a problem.

This routine loads a parallel MIP machine configuration specified by a string. The string must specify a parallel machine configuration according to the VMC file format. The string must be expressed in an encoding that terminates strings by a single NUL character. For example, UTF-8 is such an encoding. This routine does not consider the encoding specified in CPX_PARAM_APIENCODING.

.

Any previously loaded parallel machine configuration will be destroyed before this routine attempts to load the new configuration.

Arguments

Name Description
env A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
xmlstr The string that specifies the parallel machine configuration. Must be expressed in an encoding that uses a single NUL byte to terminate strings. CPX_PARAM_APIENCODING is ignored for this string.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example

 status = CPXcopyvmconfig (env, "<?xml ... ");
