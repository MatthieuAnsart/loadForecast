CPXERR_BADPRODUCT

32023 Licensing problem: License not valid for this product

#define CPXERR_BADPRODUCT 32023
The license is not configured to support the particular part of the product you are trying to use. For example, this error occurs when when anyone tries to run the Interactive Optimizer with a license configured only for runtime application deployment (for example, ILM key type RTNODE). In such a case, either locate a different ILM key that has been provided to you, or try compiling and running one of the simple examples found in the distribution, such as lpex1.c and its language variants, instead of using the Interactive Optimizer. Contact your support representative for assistance if you are unable to determine the correct ILM key for your purposes.
