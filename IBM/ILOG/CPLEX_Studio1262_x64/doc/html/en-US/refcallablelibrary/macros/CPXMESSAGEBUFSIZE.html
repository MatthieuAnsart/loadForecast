CPXMESSAGEBUFSIZE

Maximum size of message buffers managed by the message interface.

#define CPXMESSAGEBUFSIZE 1024

The constant CPXMESSAGEBUFSIZE specifies the maximum length of a string that the routine CPXXmsg/CPXmsg can process. It also specifies the maximum length of a string that CPLEX will pass to a function registered by CPXXaddfuncdest/CPXaddfuncdest.

The string length defined by this constant includes the terminating NUL character. That is, the maximum length of the string is actually size minus one.
