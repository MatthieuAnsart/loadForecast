CPXERR_UNSUPPORTED_CONSTRAINT_TYPE

1212 Unsupported constraint type was used.

#define CPXERR_UNSUPPORTED_CONSTRAINT_TYPE 1212

CPLEX was unable to use the specified constraint type, or the constraint type identifier is invalid in an argument passed to the routine CPXXrefineconflictext or CPXXfeasoptext.
