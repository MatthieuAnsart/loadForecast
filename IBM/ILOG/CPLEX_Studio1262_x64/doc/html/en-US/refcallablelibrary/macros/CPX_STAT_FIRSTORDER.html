CPX_STAT_FIRSTORDER

24 (Barrier only) enum: FirstOrder

#define CPX_STAT_FIRSTORDER 24

Solution satisfies first-order optimality conditions for a solution returned by the barrier optimizer for an indefinite QP when the solution target type parameter specifies a search for a solution that satisfies first-order optimality conditions, but is not necessarily globally optimal (value 2).
