Solution Pool Symbols in the CPLEX Callable Library (C API)

These symbols support access to and manipulation of the pool of solutions produced by CPLEX.

See also Accessing the Solution Pool in the CPLEX Callable Library (C API) for documentation of routines to manage the solution pool.

  • CPX_SOLNPOOL_FIFO
    Solution pool strategy.
  • CPX_SOLNPOOL_OBJ
    Solution pool strategy.
  • CPX_SOLNPOOL_DIV
    Solution pool strategy.
  • CPX_SOLNPOOL_FILTER_DIVERSITY
    Solution pool filter: diversity.
  • CPX_SOLNPOOL_FILTER_RANGE
    Solution pool filter: linear expression values.
