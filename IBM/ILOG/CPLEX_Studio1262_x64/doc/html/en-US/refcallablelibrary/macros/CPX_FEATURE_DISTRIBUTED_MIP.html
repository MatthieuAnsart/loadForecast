CPX_FEATURE_DISTRIBUTED_MIP

Specifies whether distributed parallel MIP is supported on this platform

#define CPX_FEATURE_DISTRIBUTED_MIP

This macro expands to 1 (one) of distributed parallel mixed integer programming (MIP) is supported on this platform or to 0 (zero) otherwise. For more information about this feature, see the topic Solving a MIP with distributed parallel optimization in the CPLEX User's Manual.
