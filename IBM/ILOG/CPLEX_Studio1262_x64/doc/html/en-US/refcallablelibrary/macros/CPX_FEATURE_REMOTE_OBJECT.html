CPX_FEATURE_REMOTE_OBJECT

Specifies whether the CPLEX remote object is supported on this platform

#define CPX_FEATURE_REMOTE_OBJECT

This macro expands to 1 (one) if the CPLEX remote object is supported on this platform or to 0 (zero) otherwise. For more information about this feature see the topic Remote object for distributed parallel optimization in the CPLEX User's Manual.
