CPXERR_FILE_IO

1426 Low-level file I/O error

#define CPXERR_FILE_IO 1426
A low-level error occurred during input or output of a file. To identify the error, check whether the file that CPLEX tried to read from or write to is a valid file and still accessible. Also check that there are no errors at the level of the file system, such as the file system being full during an attempt to write.
