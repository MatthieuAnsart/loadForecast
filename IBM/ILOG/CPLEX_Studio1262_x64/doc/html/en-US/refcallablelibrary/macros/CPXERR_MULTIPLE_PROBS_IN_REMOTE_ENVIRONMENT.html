CPXERR_MULTIPLE_PROBS_IN_REMOTE_ENVIRONMENT

1816 Attempt to create multiple problems in a remote environment.

#define CPXERR_MULTIPLE_PROBS_IN_REMOTE_ENVIRONMENT 1816

Creation of multiple problems in one remote environment object is not supported by this instance of the library.
