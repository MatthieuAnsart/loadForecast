CPXINFOHANDLER

Prototype for functions to handle information messages on a client.

typedef void(CPXPUBLIC CPXINFOHANDLER)( CPXENVptr env, CPXINFOTYPE type, int tag, CPXLONG elems, void const*data, void *handle )
Handlers of this type are registered by a call to CPXsetinfohandler. Registered handlers of this type are invoked whenever an information message (triggered by CPXsendinfo) arrives from the remote end. Information handlers are not allowed to modify any of the data pointed to by the argument data.
Name Description
env The environment to which the information was sent.
type The type of information (determines the type of data to which the argument data points).
tag The tag that was specified when CPXsendinfo was called.
data The data for the info message; the actual type depends on the argument type.
handle User handle.
