CPXCENVGROUPptr

A group of environments.

typedef struct cpxenvgroup const *CPXCENVGROUPptr
Use groups of remote environments to perform multicast invocations of routines that do not have output arguments. Multicasts can greatly improve performance if the same operation has to be performed on a large number of remote objects.
