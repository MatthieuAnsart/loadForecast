Integer Types in the CPLEX Callable Library (C API) Reference Manual

These types of integers are available in the CPLEX Callable Library (C API).

  • CPXBYTE
    An 8-bit signed integer
  • CPXINT
    A 32 bit, signed integer.
  • CPXLONG
    A 64-bit signed integer
  • CPXSHORT
    A 16-bit signed integer
  • CPXSIZE
    Signed size type.
  • CPXULONG
    A 64-bit unsigned integer
