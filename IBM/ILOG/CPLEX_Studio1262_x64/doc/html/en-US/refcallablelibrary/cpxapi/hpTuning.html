Tuning applications

These routines support the tuning feature of CPLEX.

  • CPXXtuneparam and CPXtuneparam
    The routine CPXXtuneparam/CPXtuneparam tunes the parameters of the environment for improved optimizer performance on the specified problem object.
  • CPXXtuneparamprobset and CPXtuneparamprobset
    The routine CPXXtuneparamprobset/CPXtuneparamprobset tunes the parameters of the CPLEX environment for improved optimizer performance for a set of problems.
  • CPXXgettuningcallbackfunc and CPXgettuningcallbackfunc
    The routine CPXXgettuningcallbackfunc/CPXgettuningcallbackfunc accesses the user-written callback routine to be called before each trial run during the tuning process.
  • CPXXsettuningcallbackfunc and CPXsettuningcallbackfunc
    The routine CPXXsettuningcallbackfunc/CPXsettuningcallbackfunc modifies the user-written callback routine to be called before each trial run during the tuning process.
