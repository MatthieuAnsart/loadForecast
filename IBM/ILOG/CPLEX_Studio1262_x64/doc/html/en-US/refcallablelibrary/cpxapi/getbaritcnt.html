CPXXgetbaritcnt and CPXgetbaritcnt

The routine CPXXgetbaritcnt/CPXgetbaritcnt accesses the total number of Barrier iterations to solve an LP problem.

CPXCNT CPXXgetbaritcnt( CPXCENVptr env, CPXCLPptr lp )

int CPXgetbaritcnt( CPXCENVptr env, CPXCLPptr lp )

Description

The routine CPXXgetbaritcnt/CPXgetbaritcnt accesses the total number of Barrier iterations to solve an LP problem.

Arguments

Name Description
env A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.

Return

The routine returns the total iteration count if a solution exists. It returns 0 (zero) if no solution exists or any other type of error occurs.

Example

itcnt = CPXgetbaritcnt (env, lp);
