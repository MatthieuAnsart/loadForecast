CPXXserializerdestroy and CPXserializerdestroy

Destroys a serializer that was created by CPXXserializercreate/CPXserializercreate.

void CPXXserializerdestroy( CPXSERIALIZERptr ser )

void CPXserializerdestroy( CPXSERIALIZERptr ser )

Description

Destroys a serializer that was created by CPXXserializercreate/CPXserializercreate. All resources allocated for the serializer are released.

Arguments

Name Description
ser The serializer to destroy. Must be a valid serializer as returned by CPXXserializercreate/CPXserializercreate.

Return
