CPXXfputs and CPXfputs

The routine CPXXfputs/CPXfputs can be used to write output to a file opened with CPXXfopen/CPXfopen.

int CPXXfputs( char const * s_str, CPXFILEptr stream )

int CPXfputs( char const * s_str, CPXFILEptr stream )

Description

The routine CPXXfputs/CPXfputs can be used to write output to a file opened with CPXXfopen/CPXfopen. The purpose of this routine is to allow user-defined output in a file to be interspersed with the output created by using the routines CPXXaddfpdest/CPXaddfpdest or CPXXsetlogfile/CPXsetlogfile. The syntax of CPXXfputs/CPXfputs is the same as the standard C library function fputs.

Arguments

Name Description
s_str A pointer to a string to be output to the file.
stream A pointer to a file opened by the routine CPXXfopen/CPXfopen.

Return

This routine returns a nonnegative value if successful. Otherwise, it returns the system constant EOF (end of file).

Example

CPXfputs ("Solved first problem.\n", fp);
