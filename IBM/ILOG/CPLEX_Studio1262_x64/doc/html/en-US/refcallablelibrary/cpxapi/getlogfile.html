CPXXgetlogfile and CPXgetlogfile

The routine CPXXgetlogfile/CPXgetlogfile accesses the log file to which messages from all four CPLEX-defined channels are written.

int CPXXgetlogfile( CPXCENVptr env, CPXFILEptr * logfile_p )

int CPXgetlogfile( CPXCENVptr env, CPXFILEptr * logfile_p )

Description

The routine CPXXgetlogfile/CPXgetlogfile accesses the log file to which messages from all four CPLEX-defined channels are written.

Arguments

Name Description
env A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
logfile_p The address of a CPXFILEptr variable. This routine sets logfile_p to be the file pointer for the current log file.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example

status = CPXgetlogfile (env, &logfile);
