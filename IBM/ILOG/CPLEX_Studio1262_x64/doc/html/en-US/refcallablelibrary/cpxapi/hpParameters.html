Using parameters of CPLEX

These routines manipulate parameters of CPLEX. For information about each parameter, see the CPLEX Parameters reference manual.

  • CPXXwriteparam and CPXwriteparam
    The routine CPXXwriteparam/CPXwriteparam writes the name and current setting of CPLEX parameters that are not at their default setting in the environment specified by env.
  • CPXXgetchgparam and CPXgetchgparam
    The routine CPXXgetchgparam/CPXgetchgparam returns an arrary of parameter numbers (unique identifiers) for parameters which are not set at their default values.
  • CPXXgetdblparam and CPXgetdblparam
    The routine CPXXgetdblparam/CPXgetdblparam obtains the current value of a CPLEX parameter of type double.
  • CPXXgetintparam and CPXgetintparam
    The routine CPXXgetintparam/CPXgetintparam obtains the current value of a CPLEX parameter of type CPXINT.
  • CPXXgetlongparam and CPXgetlongparam
    The routine CPXXgetlongparam/CPXgetlongparam obtains the current value of a CPLEX parameter of type CPXLONG.
  • CPXXgetparamname and CPXgetparamname
    The routine CPXXgetparamname/CPXgetparamname returns the name of a CPLEX parameter, given the symbolic constant (or reference number) for it.
  • CPXXgetparamnum and CPXgetparamnum
    The routine CPXXgetparamnum/CPXgetparamnum returns the reference number of a CPLEX parameter, given a character string containing the name for it.
  • CPXXgetparamtype and CPXgetparamtype
    The routine CPXXgetparamtype/CPXgetparamtype returns the type of a CPLEX parameter, given the symbolic constant or reference number for it.
  • CPXXgetstrparam and CPXgetstrparam
    The routine CPXXgetstrparam/CPXgetstrparam obtains the current value of a CPLEX string parameter.
  • CPXXinfodblparam and CPXinfodblparam
    The routine CPXXinfodblparam/CPXinfodblparam obtains the default, minimum, and maximum values of a CPLEX parameter of type double.
  • CPXXinfointparam and CPXinfointparam
    The routine CPXXinfointparam/CPXinfointparam obtains the default, minimum, and maximum values of a CPLEX parameter of type CPXINT.
  • CPXXinfolongparam and CPXinfolongparam
    The routine CPXXinfolongparam/CPXinfolongparam obtains the default, minimum, and maximum values of a CPLEX parameter of type CPXLONG.
  • CPXXinfostrparam and CPXinfostrparam
  • CPXXreadcopyparam and CPXreadcopyparam
    The routine CPXXreadcopyparam/CPXreadcopyparam reads parameter names and settings from the file specified by filename_str and copies them into CPLEX.
  • CPXXsetdblparam and CPXsetdblparam
    The routine CPXXsetdblparam/CPXsetdblparam sets the value of a CPLEX parameter of type double.
  • CPXXsetdefaults and CPXsetdefaults
    The routine CPXXsetdefaults/CPXsetdefaults resets all CPLEX parameters and settings to default values (with the exception of the log file).
  • CPXXsetintparam and CPXsetintparam
    The routine CPXXsetintparam/CPXsetintparam sets the value of a CPLEX parameter of type CPXINT.
  • CPXXsetlongparam and CPXsetlongparam
    The routine CPXXsetlongparam/CPXsetlongparam sets the value of a CPLEX parameter of type CPXLONG.
  • CPXXsetstrparam and CPXsetstrparam
    The routine CPXXsetstrparam/CPXsetstrparam sets the value of a CPLEX string parameter.
