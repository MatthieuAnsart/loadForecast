CPXXcloneprob and CPXcloneprob

The routine CPXXcloneprob/CPXcloneprob can be used to create a new CPLEX problem object and copy all the problem data from an existing problem object to it.

CPXLPptr CPXXcloneprob( CPXCENVptr env, CPXCLPptr lp, int * status_p )

CPXLPptr CPXcloneprob( CPXCENVptr env, CPXCLPptr lp, int * status_p )

Description

The routine CPXXcloneprob/CPXcloneprob can be used to create a new CPLEX problem object and copy all the problem data from an existing problem object to it. Solution and starting information is not copied.

Arguments

Name Description
env A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp A pointer to a CPLEX problem object of which a copy is to be created.
status_p A pointer to an integer used to return any error code produced by this routine.

Return

If successful, CPXXcloneprob/CPXcloneprob returns a pointer that can be passed to other CPLEX routines to identify the problem object that has been created, and the argument *status_p is zero. If not successful, a NULL pointer is returned, and an error status is returned in the argument *status_p.

Example

copy = CPXcloneprob (env, lp, &status);
