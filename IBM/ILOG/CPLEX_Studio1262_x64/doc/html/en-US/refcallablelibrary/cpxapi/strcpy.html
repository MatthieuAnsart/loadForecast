CPXXstrcpy and CPXstrcpy

This routine is deprecated.

CPXCHARptr CPXXstrcpy( char * dest_str, char const * src_str )

CPXCHARptr CPXstrcpy( char * dest_str, char const * src_str )

Description

This routine is deprecated. User your C library's strcpy function instead.

The routine CPXXstrcpy/CPXstrcpy copies strings. It is exactly the same as the standard C library routine strcpy. This routine is provided so that strings passed to the message function routines (see CPXXaddfuncdest/CPXaddfuncdest) can be copied by languages that do not allow dereferencing of pointers (for example, older versions of Visual Basic).

Arguments

Name Description
dest_str A pointer to the string to hold the copy of the string pointed to by src_str.
src_str A pointer to a string to be copied to dest_str.

Return

The routine returns a pointer to the string being copied to.

Example

CPXstrcpy (p, q);
