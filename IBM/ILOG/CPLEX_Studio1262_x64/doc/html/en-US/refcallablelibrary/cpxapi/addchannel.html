CPXXaddchannel and CPXaddchannel

The routine CPXXaddchannel/CPXaddchannel instantiates a new channel object.

CPXCHANNELptr CPXXaddchannel( CPXENVptr env )

CPXCHANNELptr CPXaddchannel( CPXENVptr env )

Description

This routine has been deprecated.

The routine CPXXaddchannel/CPXaddchannel instantiates a new channel object.

Arguments

Name Description
env A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.

Return

If successful, CPXXaddchannel/CPXaddchannel returns a pointer to the new channel object; otherwise, it returns NULL.

Example

mychannel = CPXaddchannel (env);

See also lpex5.c in the CPLEX User's Manual.
