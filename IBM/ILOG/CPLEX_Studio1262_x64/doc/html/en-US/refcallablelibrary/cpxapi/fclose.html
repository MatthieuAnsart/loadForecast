CPXXfclose and CPXfclose

The routine CPXXfclose/CPXfclose closes files that are used in conjunction with the routines CPXXaddfpdest/CPXaddfpdest, CPXXdelfpdest/CPXdelfpdest, and CPXXsetlogfile/CPXsetlogfile.

int CPXXfclose( CPXFILEptr stream )

int CPXfclose( CPXFILEptr stream )

Description

The routine CPXXfclose/CPXfclose closes files that are used in conjunction with the routines CPXXaddfpdest/CPXaddfpdest, CPXXdelfpdest/CPXdelfpdest, and CPXXsetlogfile/CPXsetlogfile. It is used in the same way as the standard C library function fclose. Files that are opened with the routine CPXXfopen/CPXfopen must be closed with the routine CPXXfclose/CPXfclose.

When to use this routine

Call this routine only after the message destinations that use the file pointer have been closed or deleted. Those destinations (such as log files) might be specified by routines such as CPXXaddfpdest/CPXaddfpdest, CPXXdelfpdest/CPXdelfpdest, and CPXXsetlogfile/CPXsetlogfile.

Arguments

Name Description
stream A pointer to a file opened by the routine CPXXfopen/CPXfopen.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs. The syntax is identical to the standard C library routine fclose.

Example

CPXfclose (fp);
