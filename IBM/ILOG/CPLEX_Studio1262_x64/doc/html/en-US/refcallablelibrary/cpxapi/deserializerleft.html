CPXXdeserializerleft and CPXdeserializerleft

Accesses the number of bytes left in a de-serializer.

CPXLONG CPXXdeserializerleft( CPXCDESERIALIZERptr deser )

CPXLONG CPXdeserializerleft( CPXCDESERIALIZERptr deser )

Description

Accesses the number of bytes left in a de-serializer. This routine returns the number of bytes in the memory buffer of the de-serializer that have not yet been de-serialized. The number returned decreases with each de-serialization operation performed on the de-serializer. The number returned is 0 (zero) if all data in the memory buffer of the de-serializer have been processed.

Arguments

Name Description
deser The de-serializer to be queried. The de-serializer must be a valid de-serializer as returned by CPXXdeserializercreate/CPXdeserializercreate.

Return

The number of bytes in the memory buffer of deser that have not yet been processed.
