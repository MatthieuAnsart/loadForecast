CPXXdeserializerdestroy and CPXdeserializerdestroy

Destroys a de-serializer that was created by CPXXdeserializercreate/CPXdeserializercreate.

void CPXXdeserializerdestroy( CPXDESERIALIZERptr deser )

void CPXdeserializerdestroy( CPXDESERIALIZERptr deser )

Description

Destroys a de-serializer that was created by CPXXdeserializercreate/CPXdeserializercreate. This routine releases all resources allocated for deser.

Arguments

Name Description
deser The de-serializer to destroy. The de-serializer must be a valid de-serializer as returned by CPXXdeserializercreate/CPXdeserializercreate.

Return
