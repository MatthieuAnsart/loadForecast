CPXXgetitcnt and CPXgetitcnt

The routine CPXXgetitcnt/CPXgetitcnt accesses the total number of simplex iterations to solve an LP problem, or the number of crossover iterations in the case that the barrier optimizer is used.

CPXCNT CPXXgetitcnt( CPXCENVptr env, CPXCLPptr lp )

int CPXgetitcnt( CPXCENVptr env, CPXCLPptr lp )

Description

The routine CPXXgetitcnt/CPXgetitcnt accesses the total number of simplex iterations to solve an LP problem, or the number of crossover iterations in the case that the barrier optimizer is used.

Arguments

Name Description
env A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.

Return

If a solution exists, CPXgetitcnt returns the total iteration count. If no solution exists, CPXgetitcnt returns the value 0 (zero).

Example

itcnt = CPXgetitcnt (env, lp);

See lpex6.c in the CPLEX User's Manual.
