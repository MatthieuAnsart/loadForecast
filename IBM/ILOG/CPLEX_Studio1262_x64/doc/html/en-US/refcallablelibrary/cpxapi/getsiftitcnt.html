CPXXgetsiftitcnt and CPXgetsiftitcnt

The routine CPXXgetsiftitcnt/CPXgetsiftitcnt accesses the total number of sifting iterations to solve an LP problem.

CPXCNT CPXXgetsiftitcnt( CPXCENVptr env, CPXCLPptr lp )

int CPXgetsiftitcnt( CPXCENVptr env, CPXCLPptr lp )

Description

The routine CPXXgetsiftitcnt/CPXgetsiftitcnt accesses the total number of sifting iterations to solve an LP problem.

Arguments

Name Description
env A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp A pointer to a CPLEX LP problem object as returned by CPXXcreateprob/CPXcreateprob.

Return

The routine returns the total iteration count if a solution exists. It returns 0 (zero) if no solution exists or any other type of error occurs.

Example

itcnt = CPXgetsiftitcnt (env, lp);
