Network Models in the CPLEX Callable Library (C API) Reference Manual

These routines support models that represent network flow.

  • CPXXNETaddarcs and CPXNETaddarcs
    The routine CPXXNETaddarcs/CPXNETaddarcs adds new arcs to the network stored in a network problem object.
  • CPXXNETaddnodes and CPXNETaddnodes
    The routine CPXXNETaddnodes/CPXNETaddnodes adds new nodes to the network stored in a network problem object.
  • CPXXNETbasewrite and CPXNETbasewrite
    The routine CPXXNETbasewrite/CPXNETbasewrite writes the current basis stored in a network problem object to a file in BAS format.
  • CPXXNETchgarcname and CPXNETchgarcname
    This routine CPXXNETchgarcname/CPXNETchgarcname changes the names of a set of arcs in the network stored in a network problem object.
  • CPXXNETchgarcnodes and CPXNETchgarcnodes
    The routine CPXXNETchgarcnodes/CPXNETchgarcnodes changes the nodes associated with a set of arcs in the network stored in a network problem object.
  • CPXXNETchgbds and CPXNETchgbds
    The routine CPXXNETchgbds/CPXNETchgbds is used to change the upper, lower, or both bounds on the flow for a set of arcs in the network stored in a network problem object.
  • CPXXNETchgname and CPXNETchgname
    The routine CPXXNETchgname/CPXNETchgname changes the name of a node or an arc in the network stored in a network problem object.
  • CPXXNETchgnodename and CPXNETchgnodename
    The routine CPXXNETchgnodename/CPXNETchgnodename changes the names of a set of nodes in the network stored in a network problem object.
  • CPXXNETchgobj and CPXNETchgobj
    The routine CPXXNETchgobj/CPXNETchgobj is used to change the objective values for a set of arcs in the network stored in a network problem object.
  • CPXXNETchgobjsen and CPXNETchgobjsen
    The routine CPXXNETchgobjsen/CPXNETchgobjsen is used to change the sense of the network problem to a minimization or maximization problem.
  • CPXXNETchgsupply and CPXNETchgsupply
    The routine CPXXNETchgsupply/CPXNETchgsupply is used to change supply values for a set of nodes in the network stored in a network problem object.
  • CPXXNETcopybase and CPXNETcopybase
    The routine CPXXNETcopybase/CPXNETcopybase can be used to set the network basis for a network problem object.
  • CPXXNETcopynet and CPXNETcopynet
    The routine CPXXNETcopynet/CPXNETcopynet copies a network to a network object, overriding any other network saved in the object.
  • CPXXNETcreateprob and CPXNETcreateprob
    The routine CPXXNETcreateprob/CPXNETcreateprob constructs a new network problem object.
  • CPXXNETdelarcs and CPXNETdelarcs
    The routine CPXXNETdelarcs/CPXNETdelarcs is used to remove a range of arcs from the network stored in a network problem object.
  • CPXXNETdelnodes and CPXNETdelnodes
    The routine CPXXNETdelnodes/CPXNETdelnodes is used to remove a range of nodes from the network stored in a network problem object.
  • CPXXNETdelset and CPXNETdelset
    The routine CPXXNETdelset/CPXNETdelset is used to delete a set of nodes and arcs from the network stored in a network problem object.
  • CPXXNETextract and CPXNETextract
    The routine CPXXNETextract/CPXNETextract finds an embedded network in the LP stored in a CPLEX problem object and copies it as a network to the network problem object, net.
  • CPXXNETfreeprob and CPXNETfreeprob
    The routine CPXXNETfreeprob/CPXNETfreeprob deletes the network problem object pointed to by net_p.
  • CPXXNETgetarcindex and CPXNETgetarcindex
    The routine CPXXNETgetarcindex/CPXNETgetarcindex returns the index of the specified arc (in the network stored in a network problem object) in the integer pointed to by index_p.
  • CPXXNETgetarcname and CPXNETgetarcname
    The routine CPXXNETgetarcname/CPXNETgetarcname is used to access the names of a range of arcs in a network stored in a network problem object.
  • CPXXNETgetarcnodes and CPXNETgetarcnodes
    The routine CPXXNETgetarcnodes/CPXNETgetarcnodes is used to access the from-nodes and to-nodes for a range of arcs in the network stored in a network problem object.
  • CPXXNETgetbase and CPXNETgetbase
    The routine CPXXNETgetbase/CPXNETgetbase is used to access the network basis for a network problem object.
  • CPXXNETgetdj and CPXNETgetdj
    The routine CPXXNETgetdj/CPXNETgetdj is used to access reduced costs for a range of arcs of the network stored in a network problem object.
  • CPXXNETgetitcnt and CPXNETgetitcnt
    The routine CPXXNETgetitcnt/CPXNETgetitcnt accesses the total number of network simplex iterations for the most recent call to CPXXNETprimopt/CPXNETprimopt, for a network problem object.
  • CPXXNETgetlb and CPXNETgetlb
    The routine CPXXNETgetlb/CPXNETgetlb is used to access the lower capacity bounds for a range of arcs of the network stored in a network problem object.
  • CPXXNETgetnodearcs and CPXNETgetnodearcs
    The routine CPXXNETgetnodearcs/CPXNETgetnodearcs is used to access the arc indices incident to a range of nodes in the network stored in a network problem object.
  • CPXXNETgetnodeindex and CPXNETgetnodeindex
    The routine CPXXNETgetnodeindex/CPXNETgetnodeindex returns the index of the specified node (in the network stored in a network problem object) in the integer pointed to by index_p.
  • CPXXNETgetnodename and CPXNETgetnodename
    The routine CPXXNETgetnodename/CPXNETgetnodename is used to obtain the names of a range of nodes in a network stored in a network problem object.
  • CPXXNETgetnumarcs and CPXNETgetnumarcs
    The routine CPXXNETgetnumarcs/CPXNETgetnumarcs is used to access the number of arcs in a network stored in a network problem object.
  • CPXXNETgetnumnodes and CPXNETgetnumnodes
    The routine CPXXNETgetnumnodes/CPXNETgetnumnodes is used to access the number of nodes in a network stored in a network problem object.
  • CPXXNETgetobj and CPXNETgetobj
    The routine CPXXNETgetobj/CPXNETgetobj is used to access the objective function values for a range of arcs in the network stored in a network problem object.
  • CPXXNETgetobjsen and CPXNETgetobjsen
    The routine CPXXNETgetobjsen/CPXNETgetobjsen returns the sense of the objective function (i.
  • CPXXNETgetobjval and CPXNETgetobjval
    The routine CPXXNETgetobjval/CPXNETgetobjval returns the objective value of the solution stored in a network problem object.
  • CPXXNETgetphase1cnt and CPXNETgetphase1cnt
    The routine CPXXNETgetphase1cnt/CPXNETgetphase1cnt returns the number of phase 1 network simplex iterations for the most recent call to CPXXNETprimopt/CPXNETprimopt.
  • CPXXNETgetpi and CPXNETgetpi
    The routine CPXXNETgetpi/CPXNETgetpi is used to access dual values for a range of nodes in the network stored in a network problem object.
  • CPXXNETgetprobname and CPXNETgetprobname
    The routine CPXXNETgetprobname/CPXNETgetprobname is used to access the name of the problem stored in a network problem object.
  • CPXXNETgetslack and CPXNETgetslack
    The routine CPXXNETgetslack/CPXNETgetslack is used to access slack values or, equivalently, violations of supplies/demands for a range of nodes in the network stored in a network problem object.
  • CPXXNETgetstat and CPXNETgetstat
    The routine CPXXNETgetstat/CPXNETgetstat returns the solution status for a network problem object.
  • CPXXNETgetsupply and CPXNETgetsupply
    The routine CPXXNETgetsupply/CPXNETgetsupply is used to obtain supply values for a range of nodes in the network stored in a CPLEX network problem object.
  • CPXXNETgetub and CPXNETgetub
    The routine CPXXNETgetub/CPXNETgetub is used to access the upper capacity bounds for a range of arcs in the network stored in a network problem object.
  • CPXXNETgetx and CPXNETgetx
    The routine CPXXNETgetx/CPXNETgetx is used to access solution values or, equivalently, flow values for a range of arcs stored in a network problem object.
  • CPXXNETprimopt and CPXNETprimopt
    The routine CPXXNETprimopt/CPXNETprimopt can be called after a network problem has been copied to a network problem object, to find a solution to that problem using the primal network simplex method.
  • CPXXNETreadcopybase and CPXNETreadcopybase
    The routine CPXXNETreadcopybase/CPXNETreadcopybase reads a basis file in BAS format and copies the basis to a network problem object.
  • CPXXNETreadcopyprob and CPXNETreadcopyprob
    The routine CPXXNETreadcopyprob/CPXNETreadcopyprob reads a network, in the CPLEX .net or DIMACS .min format, from a file and copies it to a network problem object.
  • CPXXNETsolninfo and CPXNETsolninfo
    The routine CPXXNETsolninfo/CPXNETsolninfo is used to access solution information computed by the most recent call to CPXXNETprimopt/CPXNETprimopt.
  • CPXXNETsolution and CPXNETsolution
    The routine CPXXNETsolution/CPXNETsolution accesses solution values for a network problem object computed by the most recent call to CPXXNETprimopt/CPXNETprimopt for that object.
  • CPXXNETwriteprob and CPXNETwriteprob
    The routine CPXXNETwriteprob/CPXNETwriteprob writes the network stored in a network problem object to a file.
