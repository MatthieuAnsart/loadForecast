Remote Object (worker) in the CPLEX Callable Library (C API)

These worker routines enable your application to create and optimize a CPLEX object on a remote platform.

For routines specific to the master in an application of the remote object, see Remote Object (master) in the CPLEX Callable Library (C API).

  • CPXXgetuserfunction and CPXgetuserfunction
    This routine queries the current user function implementation.
  • CPXXsendinfobyte and CPXsendinfobyte
    This routine sends an info message with CPXBYTE payload.
  • CPXXsendinfodouble and CPXsendinfodouble
    This routine sends an info message with double payload.
  • CPXXsendinfoint and CPXsendinfoint
    This routine sends an info message with CPXINT payload.
  • CPXXsendinfolong and CPXsendinfolong
    This routine sends an info message with CPXLONG payload.
  • CPXXsendinfoshort and CPXsendinfoshort
    This routine sends an info message with CPXSHORT payload.
  • CPXXsetuserfunction and CPXsetuserfunction
    This routine register a user function implementation.
