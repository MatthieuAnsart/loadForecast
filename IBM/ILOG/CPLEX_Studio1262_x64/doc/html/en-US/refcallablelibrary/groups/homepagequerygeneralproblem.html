Querying Models (general) in the CPLEX Callable Library (C API)

These routines enable your application to access information about a model in CPLEX after the model has been created or even after the model has been modified.

  • CPXXgetcoef and CPXgetcoef
    The routine CPXXgetcoef/CPXgetcoef accesses a single constraint matrix coefficient of a CPLEX problem object.
  • CPXXgetcolindex and CPXgetcolindex
    The routine CPXXgetcolindex/CPXgetcolindex searches for the index number of the specified column in a CPLEX problem object.
  • CPXXgetcolname and CPXgetcolname
    The routine CPXgetcolname accesses a range of column names or, equivalently, the variable names of a CPLEX problem object.
  • CPXXgetcols and CPXgetcols
    The routine CPXXgetcols/CPXgetcols accesses a range of columns of the constraint matrix of a CPLEX problem object.
  • CPXXgetlb and CPXgetlb
    The routine CPXXgetlb/CPXgetlb accesses a range of lower bounds on the variables of a CPLEX problem object.
  • CPXXgetnumcols and CPXgetnumcols
    The routine CPXXgetnumcols/CPXgetnumcols accesses the number of columns in the constraint matrix, or equivalently, the number of variables in the CPLEX problem object.
  • CPXXgetnumnz and CPXgetnumnz
    The routine CPXXgetnumnz/CPXgetnumnz accesses the number of nonzero elements in the constraint matrix of a CPLEX problem object, not including the objective function, quadratic constraints, or the bounds constraints on the variables.
  • CPXXgetnumrows and CPXgetnumrows
    The routine CPXXgetnumrows/CPXgetnumrows accesses the number of rows in the constraint matrix, not including the objective function, quadratic constraints, or the bounds constraints on the variables.
  • CPXXgetobj and CPXgetobj
    The routine CPXXgetobj/CPXgetobj accesses a range of objective function coefficients of a CPLEX problem object.
  • CPXXgetobjname and CPXgetobjname
    The routine CPXXgetobjname/CPXgetobjname accesses the name of the objective row of a CPLEX problem object.
  • CPXXgetobjsen and CPXgetobjsen
    The routine CPXXgetobjsen/CPXgetobjsen accesses whether the objective function sense of a CPLEX problem object is maximization or minimization.
  • CPXXgetprobname and CPXgetprobname
    The routine CPXXgetprobname/CPXgetprobname accesses the name of the problem set via the call to CPXXcreateprob/CPXcreateprob.
  • CPXXgetprobtype and CPXgetprobtype
    The routine CPXXgetprobtype/CPXgetprobtype accesses the problem type that is currently stored in a CPLEX problem object.
  • CPXXgetrhs and CPXgetrhs
    The routine CPXXgetrhs/CPXgetrhs accesses the righthand side coefficients for a range of constraints in a CPLEX problem object.
  • CPXXgetrngval and CPXgetrngval
    The routine CPXXgetrngval/CPXgetrngval accesses the RHS range coefficients for a set of constraints in a CPLEX problem object.
  • CPXXgetrowindex and CPXgetrowindex
    The routine CPXXgetrowindex/CPXgetrowindex searches for the index number of the specified row in a CPLEX problem object.
  • CPXXgetrowname and CPXgetrowname
    The routine CPXXgetrowname/CPXgetrowname accesses a range of row names or, equivalently, the constraint names of a CPLEX problem object.
  • CPXXgetrows and CPXgetrows
    The routine CPXXgetrows/CPXgetrows accesses a range of rows of the constraint matrix, not including the objective function nor the bound constraints on the variables of a CPLEX problem object.
  • CPXXgetsense and CPXgetsense
    The routine CPXXgetsense/CPXgetsense accesses the sense for a range of constraints in a CPLEX problem object.
  • CPXXgetub and CPXgetub
    The routine CPXXgetub/CPXgetub accesses a range of upper bounds on the variables of a CPLEX problem object.
