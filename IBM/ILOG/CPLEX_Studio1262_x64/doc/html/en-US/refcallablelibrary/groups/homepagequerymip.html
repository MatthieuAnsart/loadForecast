Querying Models (MIP) in the CPLEX Callable Library (C API)

These routines enable your application to access information about a mixed integer program (MIP) in CPLEX after the model has been created or even after the model has been modified.

  • CPXXgetctype and CPXgetctype
    The routine CPXXgetctype/CPXgetctype accesses the types for a range of variables in a problem object.
  • CPXXgetindconstr and CPXgetindconstr
    The routine CPXXgetindconstr/CPXgetindconstr accesses a specified indicator constraint on the variables of a CPLEX problem object.
  • CPXXgetindconstrindex and CPXgetindconstrindex
    The routine CPXXgetindconstrindex/CPXgetindconstrindex searches for the index number of the specified indicator constraint in a CPLEX problem object.
  • CPXXgetindconstrname and CPXgetindconstrname
    The routine CPXXgetindconstrname/CPXgetindconstrname accesses the name of a specified indicator constraint of a CPLEX problem object.
  • CPXXgetnumbin and CPXgetnumbin
    The routine CPXXgetnumbin/CPXgetnumbin accesses the number of binary variables in a CPLEX problem object.
  • CPXXgetnumindconstrs and CPXgetnumindconstrs
    The routine CPXXgetnumindconstrs/CPXgetnumindconstrs accesses the number of indicator constraints in a CPLEX problem object.
  • CPXXgetnumint and CPXgetnumint
    The routine CPXXgetnumint/CPXgetnumint accesses the number of general integer variables in a CPLEX problem object.
  • CPXXgetnumsemicont and CPXgetnumsemicont
    The routine CPXXgetnumsemicont/CPXgetnumsemicont accesses the number of semi-continuous variables in a CPLEX problem object.
  • CPXXgetnumsemiint and CPXgetnumsemiint
    The routine CPXXgetnumsemiint/CPXgetnumsemiint accesses the number of semi-integer variables in a CPLEX problem object.
  • CPXXgetnumsos and CPXgetnumsos
    The routine CPXXgetnumsos/CPXgetnumsos accesses the number of special ordered sets (SOS) in a CPLEX problem object.
  • CPXXgetorder and CPXgetorder
    The routine CPXXgetorder/CPXgetorder accesses all the MIP priority order information stored in a CPLEX problem object.
  • CPXXgetsos and CPXgetsos
    The routine CPXXgetsos/CPXgetsos accesses the definitions of a range of special ordered sets (SOS) stored in a CPLEX problem object.
  • CPXXgetsosindex and CPXgetsosindex
    The routine CPXXgetsosindex/CPXgetsosindex searches for the index number of the specified special ordered set in a CPLEX problem object.
  • CPXXgetsosname and CPXgetsosname
    The routine CPXXgetsosname/CPXgetsosname accesses a range of special ordered set (SOS) names of a CPLEX problem object.
