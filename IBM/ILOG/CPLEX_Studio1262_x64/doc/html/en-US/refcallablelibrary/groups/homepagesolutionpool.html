Accessing the Solution Pool in the CPLEX Callable Library (C API)

These routines support access to and manipulation of the pool of solutions produced by CPLEX.

See also Solution Pool Symbols in the CPLEX Callable Library (C API) for symbols enumerated for solution pool routines.

  • CPXXaddsolnpooldivfilter and CPXaddsolnpooldivfilter
    The routine CPXXaddsolnpooldivfilter/CPXaddsolnpooldivfilter adds a new diversity filter to the solution pool.
  • CPXXaddsolnpoolrngfilter and CPXaddsolnpoolrngfilter
    Adds a new range filter to the solution pool.
  • CPXXchgprobtypesolnpool and CPXchgprobtypesolnpool
    The routine CPXXchgprobtypesolnpool/CPXchgprobtypesolnpool changes the current problem, if it is a mixed integer problem, to a related fixed problem using a solution from the solution pool.
  • CPXXdelsetsolnpoolfilters and CPXdelsetsolnpoolfilters
    The routine CPXXdelsetsolnpoolfilters/CPXdelsetsolnpoolfilters deletes filters from the problem object specified by the argument lp.
  • CPXXdelsetsolnpoolsolns and CPXdelsetsolnpoolsolns
    The routine CPXXdelsetsolnpoolsolns/CPXdelsetsolnpoolsolns deletes solutions from the solution pool of the problem object specified by the argument lp.
  • CPXXdelsolnpoolfilters and CPXdelsolnpoolfilters
    The routine CPXXdelsolnpoolfilters/CPXdelsolnpoolfilters deletes filters from the the problem object specified by the argument lp.
  • CPXXdelsolnpoolsolns and CPXdelsolnpoolsolns
    The routine CPXXdelsolnpoolsolns/CPXdelsolnpoolsolns deletes a range of solutions from the solution pool.
  • CPXXfltwrite and CPXfltwrite
    The routine CPXXfltwrite/CPXfltwrite writes filters from the selected problem object to a file in FLT format.
  • CPXXgetsolnpooldblquality and CPXgetsolnpooldblquality
    The routine CPXXgetsolnpooldblquality/CPXgetsolnpooldblquality accesses double-valued information about the quality of a solution in the solution pool.
  • CPXXgetsolnpooldivfilter and CPXgetsolnpooldivfilter
    Accesses a diversity filter of the solution pool.
  • CPXXgetsolnpoolfilterindex and CPXgetsolnpoolfilterindex
    The routine CPXXgetsolnpoolfilterindex/CPXgetsolnpoolfilterindex searches for the index number of the specified filter of a CPLEX problem object.
  • CPXXgetsolnpoolfiltername and CPXgetsolnpoolfiltername
    Accesses the name of a filter of the solution pool.
  • CPXXgetsolnpoolfiltertype and CPXgetsolnpoolfiltertype
    Accesses the type of a filter of the solution pool.
  • CPXXgetsolnpoolintquality and CPXgetsolnpoolintquality
    The routine CPXXgetsolnpoolintquality/CPXgetsolnpoolintquality accesses integer-valued information about the quality of a solution in the solution pool.
  • CPXXgetsolnpoolmeanobjval and CPXgetsolnpoolmeanobjval
    The routine CPXXgetsolnpoolmeanobjval/CPXgetsolnpoolmeanobjval accesses the the mean objective value for solutions in the pool.
  • CPXXgetsolnpoolnumfilters and CPXgetsolnpoolnumfilters
    The routine CPXXgetsolnpoolnumfilters/CPXgetsolnpoolnumfilters accesses the number of filters in the solution pool.
  • CPXXgetsolnpoolnumreplaced and CPXgetsolnpoolnumreplaced
    The routine CPXXgetsolnpoolnumreplaced/CPXgetsolnpoolnumreplaced accesses the number of solutions replaced in the solution pool.
  • CPXXgetsolnpoolnumsolns and CPXgetsolnpoolnumsolns
    The routine CPXXgetsolnpoolnumsolns/CPXgetsolnpoolnumsolns accesses the number of solutions in the solution pool in the problem object.
  • CPXXgetsolnpoolobjval and CPXgetsolnpoolobjval
    The routine CPXXgetsolnpoolobjval/CPXgetsolnpoolobjval accesses the objective value for a solution in the solution pool.
  • CPXXgetsolnpoolqconstrslack and CPXgetsolnpoolqconstrslack
    The routine CPXXgetsolnpoolqconstrslack/CPXgetsolnpoolqconstrslack accesses the slack values for a range of the quadratic constraints for a member of the solution pool of a quadratically constrained program (QCP).
  • CPXXgetsolnpoolrngfilter and CPXgetsolnpoolrngfilter
    Accesses a range filter of the solution pool.
  • CPXXgetsolnpoolslack and CPXgetsolnpoolslack
    The routine CPXXgetsolnpoolslack/CPXgetsolnpoolslack accesses the slack values for a range of linear constraints for a member of the solution pool.
  • CPXXgetsolnpoolsolnindex and CPXgetsolnpoolsolnindex
    The routine CPXXgetsolnpoolsolnindex/CPXgetsolnpoolsolnindex searches for the index number of the specified solution in the solution pool of a CPLEX problem object.
  • CPXXgetsolnpoolsolnname and CPXgetsolnpoolsolnname
    The routine CPXXgetsolnpoolsolnname/CPXgetsolnpoolsolnname accesses the name of a solution, specified by the argument soln, of the solution pool associated with the problem object specified by the argument lp.
  • CPXXgetsolnpoolx and CPXgetsolnpoolx
    The routine CPXXgetsolnpoolx/CPXgetsolnpoolx accesses the solution values for a range of problem variables for a member of the solution pool.
  • CPXXpopulate and CPXpopulate
    The routine CPXXpopulate/CPXpopulate generates multiple solutions to a mixed integer programming (MIP) problem.
  • CPXXreadcopysolnpoolfilters and CPXreadcopysolnpoolfilters
    The routine CPXXreadcopysolnpoolfilters/CPXreadcopysolnpoolfilters reads solution pool filters from an FLT format file and copies the filters into a CPLEX problem object.
  • CPXXsolwritesolnpool and CPXsolwritesolnpool
    The routine CPXXsolwritesolnpool/CPXsolwritesolnpool writes a solution file, using either the incumbent solution or a solution from the solution pool, for the selected CPLEX problem object.
  • CPXXsolwritesolnpoolall and CPXsolwritesolnpoolall
    The routine CPXXsolwritesolnpoolall/CPXsolwritesolnpoolall writes all the solutions in the solution pool to a file for the selected CPLEX problem object.
