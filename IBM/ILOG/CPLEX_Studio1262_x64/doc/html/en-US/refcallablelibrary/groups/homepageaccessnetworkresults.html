Accessing Network Results in the CPLEX Callable Library (C API)

These routines access and manipulate results produced by CPLEX when it optimizes a network flow model.

  • CPXXNETgetbase and CPXNETgetbase
    The routine CPXXNETgetbase/CPXNETgetbase is used to access the network basis for a network problem object.
  • CPXXNETgetdj and CPXNETgetdj
    The routine CPXXNETgetdj/CPXNETgetdj is used to access reduced costs for a range of arcs of the network stored in a network problem object.
  • CPXXNETgetitcnt and CPXNETgetitcnt
    The routine CPXXNETgetitcnt/CPXNETgetitcnt accesses the total number of network simplex iterations for the most recent call to CPXXNETprimopt/CPXNETprimopt, for a network problem object.
  • CPXXNETgetobjval and CPXNETgetobjval
    The routine CPXXNETgetobjval/CPXNETgetobjval returns the objective value of the solution stored in a network problem object.
  • CPXXNETgetphase1cnt and CPXNETgetphase1cnt
    The routine CPXXNETgetphase1cnt/CPXNETgetphase1cnt returns the number of phase 1 network simplex iterations for the most recent call to CPXXNETprimopt/CPXNETprimopt.
  • CPXXNETgetpi and CPXNETgetpi
    The routine CPXXNETgetpi/CPXNETgetpi is used to access dual values for a range of nodes in the network stored in a network problem object.
  • CPXXNETgetslack and CPXNETgetslack
    The routine CPXXNETgetslack/CPXNETgetslack is used to access slack values or, equivalently, violations of supplies/demands for a range of nodes in the network stored in a network problem object.
  • CPXXNETgetstat and CPXNETgetstat
    The routine CPXXNETgetstat/CPXNETgetstat returns the solution status for a network problem object.
  • CPXXNETgetx and CPXNETgetx
    The routine CPXXNETgetx/CPXNETgetx is used to access solution values or, equivalently, flow values for a range of arcs stored in a network problem object.
  • CPXXNETsolninfo and CPXNETsolninfo
    The routine CPXXNETsolninfo/CPXNETsolninfo is used to access solution information computed by the most recent call to CPXXNETprimopt/CPXNETprimopt.
  • CPXXNETsolution and CPXNETsolution
    The routine CPXXNETsolution/CPXNETsolution accesses solution values for a network problem object computed by the most recent call to CPXXNETprimopt/CPXNETprimopt for that object.
