Querying Models (Network) in the CPLEX Callable Library (C API)

These routines enable your application to access information about a network flow models in CPLEX after the model has been created or even after the model has been modified.

  • CPXXNETgetarcindex and CPXNETgetarcindex
    The routine CPXXNETgetarcindex/CPXNETgetarcindex returns the index of the specified arc (in the network stored in a network problem object) in the integer pointed to by index_p.
  • CPXXNETgetarcname and CPXNETgetarcname
    The routine CPXXNETgetarcname/CPXNETgetarcname is used to access the names of a range of arcs in a network stored in a network problem object.
  • CPXXNETgetarcnodes and CPXNETgetarcnodes
    The routine CPXXNETgetarcnodes/CPXNETgetarcnodes is used to access the from-nodes and to-nodes for a range of arcs in the network stored in a network problem object.
  • CPXXNETgetlb and CPXNETgetlb
    The routine CPXXNETgetlb/CPXNETgetlb is used to access the lower capacity bounds for a range of arcs of the network stored in a network problem object.
  • CPXXNETgetnodearcs and CPXNETgetnodearcs
    The routine CPXXNETgetnodearcs/CPXNETgetnodearcs is used to access the arc indices incident to a range of nodes in the network stored in a network problem object.
  • CPXXNETgetnodeindex and CPXNETgetnodeindex
    The routine CPXXNETgetnodeindex/CPXNETgetnodeindex returns the index of the specified node (in the network stored in a network problem object) in the integer pointed to by index_p.
  • CPXXNETgetnodename and CPXNETgetnodename
    The routine CPXXNETgetnodename/CPXNETgetnodename is used to obtain the names of a range of nodes in a network stored in a network problem object.
  • CPXXNETgetnumarcs and CPXNETgetnumarcs
    The routine CPXXNETgetnumarcs/CPXNETgetnumarcs is used to access the number of arcs in a network stored in a network problem object.
  • CPXXNETgetnumnodes and CPXNETgetnumnodes
    The routine CPXXNETgetnumnodes/CPXNETgetnumnodes is used to access the number of nodes in a network stored in a network problem object.
  • CPXXNETgetobj and CPXNETgetobj
    The routine CPXXNETgetobj/CPXNETgetobj is used to access the objective function values for a range of arcs in the network stored in a network problem object.
  • CPXXNETgetobjsen and CPXNETgetobjsen
    The routine CPXXNETgetobjsen/CPXNETgetobjsen returns the sense of the objective function (i.
  • CPXXNETgetprobname and CPXNETgetprobname
    The routine CPXXNETgetprobname/CPXNETgetprobname is used to access the name of the problem stored in a network problem object.
  • CPXXNETgetsupply and CPXNETgetsupply
    The routine CPXXNETgetsupply/CPXNETgetsupply is used to obtain supply values for a range of nodes in the network stored in a CPLEX network problem object.
  • CPXXNETgetub and CPXNETgetub
    The routine CPXXNETgetub/CPXNETgetub is used to access the upper capacity bounds for a range of arcs in the network stored in a network problem object.
