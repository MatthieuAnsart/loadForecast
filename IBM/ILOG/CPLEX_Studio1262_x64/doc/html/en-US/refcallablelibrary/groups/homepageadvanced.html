Advanced Routines in the CPLEX Callable Library (C API)

These routines are advanced.

Advanced routines typically demand a thorough understanding of the algorithms used by CPLEX. Thus they incur a higher risk of incorrect behavior in your application, behavior that can be difficult to debug. Therefore, the team encourages you to consider carefully whether you can accomplish the same task by means of other Callable Library routines instead.

  • CPXXaddlazyconstraints and CPXaddlazyconstraints
    The routine CPXXaddlazyconstraints/CPXaddlazyconstraints adds constraints to the list of constraints that should be added to the LP subproblem of a MIP optimization if they are violated.
  • CPXXaddusercuts and CPXaddusercuts
    The routine CPXXaddusercuts/CPXaddusercuts adds constraints to the list of constraints that CPLEX can add to the LP subproblem of a MIP optimization if they are violated by the LP relaxation solution.
  • CPXXbasicpresolve and CPXbasicpresolve
    The routine CPXXbasicpresolve/CPXbasicpresolve performs bound strengthening and detects redundant rows.
  • CPXXbinvacol and CPXbinvacol
    The routine CPXXbinvacol/CPXbinvacol computes the representation of the j-th column in terms of the basis.
  • CPXXbinvarow and CPXbinvarow
    The routine CPXXbinvarow/CPXbinvarow computes the i-th row of BinvA where Binv represents the inverse of the matrix B and juxtaposition specifies matrix multiplication.
  • CPXXbinvcol and CPXbinvcol
    The routine CPXXbinvcol/CPXbinvcol computes the j-th column of the basis inverse.
  • CPXXbinvrow and CPXbinvrow
    The routine CPXXbinvrow/CPXbinvrow computes the i-th row of the basis inverse.
  • CPXXbranchcallbackbranchasCPLEX and CPXbranchcallbackbranchasCPLEX
    The routine CPXXbranchcallbackbranchasCPLEX/CPXbranchcallbackbranchasCPLEX accomplishes two tasks.
  • CPXXbranchcallbackbranchbds and CPXbranchcallbackbranchbds
    The routine CPXXbranchcallbackbranchbds/CPXbranchcallbackbranchbds specifies the branches to be taken from the current node.
  • CPXXbranchcallbackbranchconstraints and CPXbranchcallbackbranchconstraints
    The routine CPXXbranchcallbackbranchconstraints/CPXbranchcallbackbranchconstraints specifies the branches to be taken from the current node when the branch is specified by adding one or more constraints to the node problem.
  • CPXXbranchcallbackbranchgeneral and CPXbranchcallbackbranchgeneral
    The routine CPXXbranchcallbackbranchgeneral/CPXbranchcallbackbranchgeneral specifies the branches to be taken from the current node when the branch includes variable bound changes and additional constraints.
  • CPXXbtran and CPXbtran
    The routine CPXXbtran/CPXbtran solves xTB = yT and puts the answer in y.
  • CPXXcallbacksetnodeuserhandle and CPXcallbacksetnodeuserhandle
    The routine CPXXcallbacksetnodeuserhandle/CPXcallbacksetnodeuserhandle sets the user handle for the node specified by nodeindex and optionally returns the old user handle stored at that node.
  • CPXXcallbacksetuserhandle and CPXcallbacksetuserhandle
    The routine CPXXcallbacksetuserhandle/CPXcallbacksetuserhandle sets the user handle at the current node and optionally returns the old user handle stored at the current node.
  • CPXXcopybasednorms and CPXcopybasednorms
    The routine CPXXcopybasednorms/CPXcopybasednorms works in conjunction with the routine CPXXgetbasednorms/CPXgetbasednorms.
  • CPXXcopydnorms and CPXcopydnorms
    The routine CPXXcopydnorms/CPXcopydnorms copies the dual steepest-edge norms to the specified LP problem object.
  • CPXXcopypnorms and CPXcopypnorms
    The routine CPXXcopypnorms/CPXcopypnorms copies the primal steepest-edge norms to the specified LP problem object.
  • CPXXcopyprotected and CPXcopyprotected
    The routine CPXXcopyprotected/CPXcopyprotected specifies a set of variables that should not be substituted out of the problem.
  • CPXXcrushform and CPXcrushform
    The routine CPXcrushform crushes a linear formula of the original problem to a linear formula of the presolved problem.
  • CPXXcrushpi and CPXcrushpi
    The routine CPXXcrushpi/CPXcrushpi crushes a dual solution for the original problem to a dual solution for the presolved problem.
  • CPXXcrushx and CPXcrushx
    The routine CPXXcrushx/CPXcrushx crushes a solution for the original problem to a solution for the presolved problem.
  • CPXXcutcallbackadd and CPXcutcallbackadd
    The routine CPXXcutcallbackadd/CPXcutcallbackadd adds lazy constraints and globally valid cuts to the current node LP subproblem during MIP branch and cut.
  • CPXXcutcallbackaddlocal and CPXcutcallbackaddlocal
    The routine CPXXcutcallbackaddlocal/CPXcutcallbackaddlocal adds local cuts during MIP branch and cut.
  • CPXXdjfrompi and CPXdjfrompi
    The routine CPXXdjfrompi/CPXdjfrompi computes an array of reduced costs from an array of dual values.
  • CPXXdualfarkas and CPXdualfarkas
    The routine CPXXdualfarkas/CPXdualfarkas assumes that there is a resident solution as produced by a call to CPXXdualopt/CPXdualopt and that the status of this solution as returned by CPXXgetstat/CPXgetstat is CPX_STAT_INFEASIBLE.
  • CPXXfreelazyconstraints and CPXfreelazyconstraints
    The routine CPXXfreelazyconstraints/CPXfreelazyconstraints clears the list of lazy constraints that have been previously specified through calls to CPXXaddlazyconstraints/CPXaddlazyconstraints.
  • CPXXfreepresolve and CPXfreepresolve
    The routine CPXXfreepresolve/CPXfreepresolve frees the presolved problem from the LP problem object.
  • CPXXfreeusercuts and CPXfreeusercuts
    The routine CPXXfreeusercuts/CPXfreeusercuts clears the list of user cuts that have been previously specified through calls to CPXXaddusercuts/CPXaddusercuts.
  • CPXXftran and CPXftran
    The routine CPXXftran/CPXftran solves By = x and puts the answer in the vector x, where B is the basis matrix.
  • CPXXgetbasednorms and CPXgetbasednorms
    The routine CPXXgetbasednorms/CPXgetbasednorms works in conjunction with the routine CPXXcopybasednorms/CPXcopybasednorms.
  • CPXXgetbhead and CPXgetbhead
    The routine CPXXgetbhead/CPXgetbhead returns the basis header; it gives the negative value minus one of all row indices of slacks.
  • CPXXgetbranchcallbackfunc and CPXgetbranchcallbackfunc
    The routine CPXXgetbranchcallbackfunc/CPXgetbranchcallbackfunc accesses the user-written callback routine to be called during MIP optimization after a branch has been selected but before the branch is carried out.
  • CPXXgetcallbackctype and CPXgetcallbackctype
    The routine CPXXgetcallbackctype/CPXgetcallbackctype retrieves the ctypes for the MIP problem from within a user-written callback during MIP optimization.
  • CPXXgetcallbackgloballb and CPXgetcallbackgloballb
    The routine CPXXgetcallbackgloballb/CPXgetcallbackgloballb retrieves the best known global lower bound values during MIP optimization from within a user-written callback.
  • CPXXgetcallbackglobalub and CPXgetcallbackglobalub
    The routine CPXXgetcallbackglobalub/CPXgetcallbackglobalub retrieves the best known global upper bound values during MIP optimization from within a user-written callback.
  • CPXXgetcallbackincumbent and CPXgetcallbackincumbent
    The routine CPXXgetcallbackincumbent/CPXgetcallbackincumbent retrieves the incumbent values during MIP optimization from within a user-written callback.
  • CPXXgetcallbackindicatorinfo and CPXgetcallbackindicatorinfo
    The routine CPXXgetcallbackindicatorinfo/CPXgetcallbackindicatorinfo accesses information about the indicator constraints of the presolved problem during MIP callbacks.
  • CPXXgetcallbacklp and CPXgetcallbacklp
    The routine CPXXgetcallbacklp/CPXgetcallbacklp retrieves the pointer to the MIP problem that is in use when the user-written callback function is called.
  • CPXXgetcallbacknodeinfo and CPXgetcallbacknodeinfo
    The routine CPXXgetcallbacknodeinfo/CPXgetcallbacknodeinfo is called from within user-written callbacks during a MIP optimization and accesses information about nodes.
  • CPXXgetcallbacknodeintfeas and CPXgetcallbacknodeintfeas
    The routine CPXXgetcallbacknodeintfeas/CPXgetcallbacknodeintfeas retrieves information for each variable about whether or not the variable is integer feasible in the subproblem at the current node.
  • CPXXgetcallbacknodelb and CPXgetcallbacknodelb
    The routine CPXXgetcallbacknodelb/CPXgetcallbacknodelb retrieves the lower bound values for the subproblem at the current node during MIP optimization from within a user-written callback.
  • CPXXgetcallbacknodelp and CPXgetcallbacknodelp
    The routine CPXXgetcallbacknodelp/CPXgetcallbacknodelp returns a pointer to the subproblem at the current node from within a user-written callback.
  • CPXXgetcallbacknodeobjval and CPXgetcallbacknodeobjval
    The routine CPXXgetcallbacknodeobjval/CPXgetcallbacknodeobjval retrieves the objective value for the subproblem at the current node during MIP optimization from within a user-written callback.
  • CPXXgetcallbacknodestat and CPXgetcallbacknodestat
    The routine CPXXgetcallbacknodestat/CPXgetcallbacknodestat retrieves the optimization status of the subproblem at the current node from within a user-written callback during MIP optimization.
  • CPXXgetcallbacknodeub and CPXgetcallbacknodeub
    The routine CPXXgetcallbacknodeub/CPXgetcallbacknodeub retrieves the upper bound values for the subproblem at the current node during MIP optimization from within a user-written callback.
  • CPXXgetcallbacknodex and CPXgetcallbacknodex
    The routine CPXXgetcallbacknodex/CPXgetcallbacknodex retrieves the primal variable (x) values for the subproblem at the current node during MIP optimization from within a user-written callback.
  • CPXXgetcallbackorder and CPXgetcallbackorder
    The routine CPXXgetcallbackorder/CPXgetcallbackorder retrieves MIP priority order information during MIP optimization from within a user-written callback.
  • CPXXgetcallbackpseudocosts and CPXgetcallbackpseudocosts
    The routine CPXXgetcallbackpseudocosts/CPXgetcallbackpseudocosts retrieves the pseudo-cost values during MIP optimization from within a user-written callback.
  • CPXXgetcallbackseqinfo and CPXgetcallbackseqinfo
    The routine CPXXgetcallbackseqinfo/CPXgetcallbackseqinfo accesses information about nodes during the MIP optimization from within user-written callbacks.
  • CPXXgetcallbacksosinfo and CPXgetcallbacksosinfo
    The routine CPXXgetcallbacksosinfo/CPXgetcallbacksosinfo accesses information about special ordered sets (SOSs) during MIP optimization from within user-written callbacks.
  • CPXXgetdeletenodecallbackfunc and CPXgetdeletenodecallbackfunc
    The routine CPXXgetdeletenodecallbackfunc/CPXgetdeletenodecallbackfunc accesses the user-written callback to be called during MIP optimization when a node is to be deleted.
  • CPXXgetdnorms and CPXgetdnorms
    The routine CPXXgetdnorms/CPXgetdnorms accesses the norms from the dual steepest edge.
  • CPXXgetheuristiccallbackfunc and CPXgetheuristiccallbackfunc
    The routine CPXXgetheuristiccallbackfunc/CPXgetheuristiccallbackfunc accesses the user-written callback to be called by CPLEX during MIP optimization after the subproblem has been solved to optimality.
  • CPXXgetijdiv and CPXgetijdiv
    The routine CPXXgetijdiv/CPXgetijdiv returns the index of the diverging row (that is, constraint) or column (that is, variable) when one of the CPLEX simplex optimizers terminates due to a diverging vector.
  • CPXXgetijrow and CPXgetijrow
    The routine CPXXgetijrow/CPXgetijrow returns the index of a specific basic variable as its position in the basis header.
  • CPXXgetincumbentcallbackfunc and CPXgetincumbentcallbackfunc
    The routine CPXXgetincumbentcallbackfunc/CPXgetincumbentcallbackfunc accesses the user-written callback to be called by CPLEX during MIP optimization after an integer solution has been found but before this solution replaces the incumbent.
  • CPXXgetlazyconstraintcallbackfunc and CPXgetlazyconstraintcallbackfunc
    The routine CPXXgetlazyconstraintcallbackfunc/CPXgetlazyconstraintcallbackfunc accesses the user-written callback for adding lazy constraints.
  • CPXXgetnodecallbackfunc and CPXgetnodecallbackfunc
    The routine CPXXgetnodecallbackfunc/CPXgetnodecallbackfunc accesses the user-written callback to be called during MIP optimization after CPLEX has selected a node to explore, but before this exploration is carried out.
  • CPXXgetobjoffset and CPXgetobjoffset
    The routine CPXXgetobjoffset/CPXgetobjoffset returns the objective offset between the original problem and the presolved problem.
  • CPXXgetpnorms and CPXgetpnorms
    The routine CPXXgetpnorms/CPXgetpnorms returns the norms from the primal steepest-edge.
  • CPXXgetprestat and CPXgetprestat
    The routine CPXXgetprestat/CPXgetprestat accesses presolve status information for the columns and rows of the presolved problem in the original problem and of the original problem in the presolved problem.
  • CPXXgetprotected and CPXgetprotected
    The routine CPXXgetprotected/CPXgetprotected accesses the set of variables that cannot be aggregated out.
  • CPXXgetray and CPXgetray
    The routine CPXXgetray/CPXgetray finds an unbounded direction (also known as a ray) for a linear program where the CPLEX simplex optimizer concludes that the LP is unbounded (solution status CPX_STAT_UNBOUNDED).
  • CPXXgetredlp and CPXgetredlp
    The routine CPXXgetredlp/CPXgetredlp returns a pointer for the presolved problem.
  • CPXXgetsolvecallbackfunc and CPXgetsolvecallbackfunc
    The routine CPXXgetsolvecallbackfunc/CPXgetsolvecallbackfunc accesses the user-written callback to be called during MIP optimization to optimize the subproblem.
  • CPXXgetusercutcallbackfunc and CPXgetusercutcallbackfunc
    The routine CPXXgetusercutcallbackfunc/CPXgetusercutcallbackfunc accesses the user-written callback for adding cuts.
  • CPXXkilldnorms and CPXkilldnorms
    The routine CPXXkilldnorms/CPXkilldnorms deletes any dual steepest-edge norms that have been retained relative to an active basis.
  • CPXXkillpnorms and CPXkillpnorms
    The routine CPXXkillpnorms/CPXkillpnorms deletes any primal steepest-edge norms that have been retained relative to an active basis.
  • CPXXmdleave and CPXmdleave
    The routine CPXXmdleave/CPXmdleave assumes that there is a resident optimal simplex basis, and a resident LU-factorization associated with this basis.
  • CPXXpivot and CPXpivot
    The routine CPXXpivot/CPXpivot performs a basis change where variable jenter replaces variable jleave in the basis.
  • CPXXpivotin and CPXpivotin
    The routine CPXXpivotin/CPXpivotin forcibly pivots slacks that appear on a list of inequality rows into the basis.
  • CPXXpivotout and CPXpivotout
    The routine CPXXpivotout/CPXpivotout pivots a list of fixed variables out of the resident basis.
  • CPXXpreaddrows and CPXpreaddrows
    The routine CPXXpreaddrows/CPXpreaddrows adds rows to an LP problem object and its associated presolved LP problem object.
  • CPXXprechgobj and CPXprechgobj
    The routine CPXXprechgobj/CPXprechgobj changes the objective function coefficients of an LP problem object and its associated presolved LP problem object.
  • CPXXpresolve and CPXpresolve
    The routine CPXXpresolve/CPXpresolve performs LP or MIP presolve depending whether a problem object is an LP or a MIP.
  • CPXXqconstrslackfromx and CPXqconstrslackfromx
    The routine CPXXqconstrslackfromx/CPXqconstrslackfromx computes an array of slack values for quadratic constraints from primal solution values.
  • CPXXqpdjfrompi and CPXqpdjfrompi
    The routine CPXXqpdjfrompi/CPXqpdjfrompi computes an array of reduced costs from an array of dual values and an array of primal values for a QP.
  • CPXXqpuncrushpi and CPXqpuncrushpi
    The routine CPXXqpuncrushpi/CPXqpuncrushpi uncrushes a dual solution for the presolved problem to a dual solution for the original problem if the original problem is a QP.
  • CPXXsetbranchcallbackfunc and CPXsetbranchcallbackfunc
    The routine CPXXsetbranchcallbackfunc/CPXsetbranchcallbackfunc sets and modifies the user-written callback routine to be called after a branch has been selected but before the branch is carried out during MIP optimization.
  • CPXXsetbranchnosolncallbackfunc and CPXsetbranchnosolncallbackfunc
    The routine CPXXsetbranchnosolncallbackfunc/CPXsetbranchnosolncallbackfunc sets the callback function that will be called instead of the branch callback when there is a failure due to such situations as an iteration limit being reached, unboundedness being detected, numeric difficulties being encountered, while the node LP is being solved.
  • CPXXsetdeletenodecallbackfunc and CPXsetdeletenodecallbackfunc
    The routine CPXXsetdeletenodecallbackfunc/CPXsetdeletenodecallbackfunc sets and modifies the user-written callback to be called during MIP optimization when a node is to be deleted.
  • CPXXsetheuristiccallbackfunc and CPXsetheuristiccallbackfunc
    The routine CPXXsetheuristiccallbackfunc/CPXsetheuristiccallbackfunc sets or modifies the user-written callback to be called by CPLEX during MIP optimization after the subproblem has been solved to optimality.
  • CPXXsetincumbentcallbackfunc and CPXsetincumbentcallbackfunc
    The routine CPXXsetincumbentcallbackfunc/CPXsetincumbentcallbackfunc sets and modifies the user-written callback routine to be called when an integer solution has been found but before this solution replaces the incumbent.
  • CPXXsetlazyconstraintcallbackfunc and CPXsetlazyconstraintcallbackfunc
    The routine CPXXsetlazyconstraintcallbackfunc/CPXsetlazyconstraintcallbackfunc sets and modifies the user-written callback for adding lazy constraints.
  • CPXXsetnodecallbackfunc and CPXsetnodecallbackfunc
    The routine CPXXsetnodecallbackfunc/CPXsetnodecallbackfunc sets and modifies the user-written callback to be called during MIP optimization after CPLEX has selected a node to explore, but before this exploration is carried out.
  • CPXXsetsolvecallbackfunc and CPXsetsolvecallbackfunc
    The routine CPXXsetsolvecallbackfunc/CPXsetsolvecallbackfunc sets and modifies the user-written callback to be called during MIP optimization to optimize subproblems (for example, node and heuristic subproblems).
  • CPXXsetusercutcallbackfunc and CPXsetusercutcallbackfunc
    The routine CPXXsetusercutcallbackfunc/CPXsetusercutcallbackfunc sets and modifies the user-written callback for adding cuts.
  • CPXXslackfromx and CPXslackfromx
    The routine CPXXslackfromx/CPXslackfromx computes an array of slack values from primal solution values.
  • CPXXstrongbranch and CPXstrongbranch
    The routine CPXXstrongbranch/CPXstrongbranch computes information for selecting a branching variable in an integer-programming branch-and-cut search.
  • CPXXtightenbds and CPXtightenbds
    The routine CPXXtightenbds/CPXtightenbds changes the upper or lower bounds on a set of variables in a problem.
  • CPXXuncrushform and CPXuncrushform
    The routine CPXXuncrushform/CPXuncrushform uncrushes a linear formula of the presolved problem to a linear formula of the original problem.
  • CPXXuncrushpi and CPXuncrushpi
    The routine CPXXuncrushpi/CPXuncrushpi uncrushes a dual solution for the presolved problem to a dual solution for the original problem.
  • CPXXuncrushx and CPXuncrushx
    The routine CPXXuncrushx/CPXuncrushx uncrushes a solution for the presolved problem to the solution for the original problem.
  • CPXXunscaleprob and CPXunscaleprob
    The routine CPXXunscaleprob/CPXunscaleprob removes any scaling that CPLEX has applied to the resident problem and its associated data.
