Remote Object (general) in the CPLEX Callable Library (C API)

These routines enable your application to create and optimize a CPLEX object on a remote platform.

For routines specific to the master in an application of the remote object, see Remote Object (master) in the CPLEX Callable Library (C API).

For routines specific to the worker in an application of the remote object, see Remote Object (worker) in the CPLEX Callable Library (C API).

  • CPXXloadLPhandle and CPXloadLPhandle
    This routine loads a handle to an LP model from a deserializer.
  • CPXXloadNEThandle and CPXloadNEThandle
    This routine loads a handle to a network problem from a deserializer.
  • CPXXstoreLPhandle and CPXstoreLPhandle
    This routine stores a handle to an LP model in a serializer.
  • CPXXstoreNEThandle and CPXstoreNEThandle
    This routine stores a handle to a network model in a serializer.
