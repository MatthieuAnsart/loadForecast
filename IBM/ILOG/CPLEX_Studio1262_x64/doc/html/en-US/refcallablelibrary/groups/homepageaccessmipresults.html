Accessing MIP Results in the CPLEX Callable Library (C API)

These routines access and manipulate MIP results produced by CPLEX.

  • CPXXgetbestobjval and CPXgetbestobjval
    The routine CPXXgetbestobjval/CPXgetbestobjval accesses the currently best known bound of all the remaining open nodes in a branch-and-cut tree.
  • CPXXgetcutoff and CPXgetcutoff
    The routine CPXXgetcutoff/CPXgetcutoff accesses the MIP cutoff value being used during mixed integer optimization.
  • CPXXgetindconstrinfeas and CPXgetindconstrinfeas
    The routine CPXXgetindconstrinfeas/CPXgetindconstrinfeas computes the infeasibility of a given solution for a range of indicator constraints.
  • CPXXgetindconstrslack and CPXgetindconstrslack
    The routine CPXXgetindconstrslack/CPXgetindconstrslack accesses the slack values for a range of indicator constraints.
  • CPXXgetmipitcnt and CPXgetmipitcnt
    The routine CPXXgetmipitcnt/CPXgetmipitcnt accesses the cumulative number of simplex iterations used to solve a mixed integer problem.
  • CPXXgetmiprelgap and CPXgetmiprelgap
    The routine CPXXgetmiprelgap/CPXgetmiprelgap accesses the relative objective gap for a MIP optimization.
  • CPXXgetmipstartindex and CPXgetmipstartindex
    The routine CPXXgetmipstartindex/CPXgetmipstartindex searches for the index number of the specified MIP start in a CPLEX problem object.
  • CPXXgetmipstartname and CPXgetmipstartname
    The routine CPXXgetmipstartname/CPXgetmipstartname accesses a range of names of MIP starts in a CPLEX problem object.
  • CPXXgetmipstarts and CPXgetmipstarts
    The routine CPXXgetmipstarts/CPXgetmipstarts accesses a range of MIP starts of a CPLEX problem object.
  • CPXXgetnodecnt and CPXgetnodecnt
    The routine CPXXgetnodecnt/CPXgetnodecnt accesses the number of nodes used to solve a mixed integer problem.
  • CPXXgetnodeint and CPXgetnodeint
    The routine CPXXgetnodeint/CPXgetnodeint accesses the node number of the best known integer solution.
  • CPXXgetnodeleftcnt and CPXgetnodeleftcnt
    The routine CPXXgetnodeleftcnt/CPXgetnodeleftcnt accesses the number of unexplored nodes left in the branch-and-cut tree.
  • CPXXgetnumcuts and CPXgetnumcuts
    The routine CPXXgetnumcuts/CPXgetnumcuts accesses the number of cuts of the specified type in use at the end of the previous optimization.
  • CPXXgetnummipstarts and CPXgetnummipstarts
    The routine CPXXgetnummipstarts/CPXgetnummipstarts accesses the number of MIP starts in the CPLEX problem object.
  • CPXXgetsosinfeas and CPXgetsosinfeas
    The routine CPXXgetsosinfeas/CPXgetsosinfeas computes the infeasibility of a given solution for a range of special ordered sets (SOSs).
  • CPXXgetsubmethod and CPXgetsubmethod
    The routine CPXXgetsubmethod/CPXgetsubmethod accesses the solution method of the last subproblem optimization, in the case of an error termination during mixed integer optimization.
  • CPXXgetsubstat and CPXgetsubstat
    The routine CPXXgetsubstat/CPXgetsubstat accesses the solution status of the last subproblem optimization, in the case of an error termination during mixed integer optimization.
  • CPXXindconstrslackfromx and CPXindconstrslackfromx
    The routine CPXXindconstrslackfromx/CPXindconstrslackfromx accesses the slack values for the indicator constraints corresponding to a given vector x.
