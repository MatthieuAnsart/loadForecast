CPXXopenCPLEXremote and CPXopenCPLEXremote

This routine opens a handle to a remote object.

CPXENVptr CPXXopenCPLEXremote( char const * transport, int argc, char const *const * argv, int * status_p )

CPXENVptr CPXopenCPLEXremote( char const * transport, int argc, char ** argv, int * status_p )

Description

This routine opens a handle to a remote object.

An environment created by this routine must be closed by CPXXcloseCPLEX/CPXXcloseCPLEX/CPXcloseCPLEX.

Arguments

Name Description
transport Name of the transport to be used. May be NULL. If this argument is NULL, then the process transport is used.
argc Number of strings in argv.
argv Transport-specific constructor arguments.
status_p If this argument is not NULL, it will be set to 0 (zero) on success and to an error code in case of error.

Return

The newly created environment or NULL in case of error.
