CPXXstoreLPhandle and CPXstoreLPhandle

This routine stores a handle to an LP model in a serializer.

int CPXXstoreLPhandle( CPXCENVptr env, CPXSERIALIZERptr s, CPXCLPptr lp )

int CPXstoreLPhandle( CPXCENVptr env, CPXSERIALIZERptr s, CPXCLPptr lp )

Description

This routine stores a handle to an LP model in a serializer. Specifically, this routine stores the handle lp in the specfied serializer. The handle must be a member of env.

Arguments

Name Description
env The environment to which lp belongs.
s The serializer to which the handle is written.
lp The LP handle to store.

Return

Returns 0 (zero) on success and an error code otherwise.
