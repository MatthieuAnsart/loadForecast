CPXXasynckill and CPXasynckill

This routine kills an asynchronous function.

int CPXXasynckill( CPXASYNCptr asynchandle )

int CPXasynckill( CPXASYNCptr asynchandle )

Description

This routine kills an asynchronous function. The function kills the asynchronous function associated with asynchandle. Not all functions react to such a signal to kill. Basically, only the problem-solving functions react to a signal to kill.

Note: You still have to join the killed function.

Arguments

Name Description
asynchandle The handle specifying the remote function to kill.

Return

Returns 0 (zero) on success or an error code on failure.
