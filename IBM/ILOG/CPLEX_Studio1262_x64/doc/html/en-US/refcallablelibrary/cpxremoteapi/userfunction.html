CPXXuserfunction and CPXuserfunction

This routine invokes a user-defined function on a remote object.

int CPXXuserfunction( CPXENVptr env, int id, CPXLONG insize, void const * indata, CPXLONG outspace, CPXLONG * outsize_p, void * outdata )

int CPXuserfunction( CPXENVptr env, int id, CPXLONG insize, void const * indata, CPXLONG outspace, CPXLONG * outsize_p, void * outdata )

Description

This routine invokes a user-defined function on a remote object.

Arguments

Name Description
env Handle to remote object. This environment must have beencreated by CPXXopenCPLEXremote/CPXopenCPLEXremote.
id A user-defined function identifier.
insize Length of indata.
indata Input data to the user-defined function; may be NULL if insize is 0 (zero).
outspace Size of outdata.
outsize_p Upon successful return, the number of valid bytes in outdata is stored here.
outdata Buffer to hold the output value of the user function. The buffer is assumed to be at least outspace bytes wide.

Return

Returns 0 (zero) on success or an error code otherwise.
