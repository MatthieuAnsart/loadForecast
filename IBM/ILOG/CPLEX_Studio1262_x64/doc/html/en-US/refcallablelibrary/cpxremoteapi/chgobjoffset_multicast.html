CPXXchgobjoffset_multicast and CPXchgobjoffset_multicast

Multicast version of CPXXchgobjoffset/CPXchgobjoffset.

int CPXXchgobjoffset_multicast( CPXENVGROUPptr group, double offset )

int CPXchgobjoffset_multicast( CPXENVGROUPptr group, double offset )

Description

Multicast version of CPXXchgobjoffset/CPXchgobjoffset. See the documentation of CPXXchgobjoffset/CPXchgobjoffset and the user manual for more details, especially Multicast: invoking the same methods on a group of objects.

Arguments

Name Description
group The group of routines, created by CPXXcreateenvgroup/CPXcreateenvgroup, to which the multicast is directed.
offset A double value that specifies the new constant objective offset of the problem.

Return

The routine returns 0 (zero) if successful or an error code otherwise.
