CPXXasynctest and CPXasynctest

This routine tests whether the asynchronous function declared by asynchandle is still running.

int CPXXasynctest( CPXASYNCptr asynchandle, int * running_p )

int CPXasynctest( CPXASYNCptr asynchandle, int * running_p )

Description

This routine tests whether the asynchronous function declared by asynchandle is still running. The argument asynchandle must be a value that was previously returned by one of the _async() functions.

Arguments

Name Description
asynchandle A handle that specifies the function to check.
running_p In this place, this routine stores a true value if the asynchronous function is still running and a false value if not.

Return

Returns 0 (zero) on success or an error code on failure.
