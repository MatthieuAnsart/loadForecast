CPXXsendinfolong and CPXsendinfolong

This routine sends an info message with CPXLONG payload.

int CPXXsendinfolong( CPXCENVptr env, int tag, CPXLONG length, CPXLONG const * data )

int CPXsendinfolong( CPXCENVptr env, int tag, CPXLONG length, CPXLONG const * data )

Description

This routine sends an info message with CPXLONG payload.

The routine sends an info message to the master that is connected to env. If an info handler is registered at the master side then this handler will be invoked on the master with arguments tag, length and data.

Note: The function does no endian conversions. In order to serialize data in a machine independent way use CPXXserializercreate/CPXserializercreate and the serialization functions.

Arguments

Name Description
env The worker environment.
tag A user defined tag (will be passed to the info handler on the master.
length Number of elements in data.
data The message payload.

Return

Returns 0 (zero) on success or an error code otherwise.
