CPXXloadLPhandle and CPXloadLPhandle

This routine loads a handle to an LP model from a deserializer.

int CPXXloadLPhandle( CPXCENVptr env, CPXDESERIALIZERptr d, CPXLPptr * lp_p )

int CPXloadLPhandle( CPXCENVptr env, CPXDESERIALIZERptr d, CPXLPptr * lp_p )

Description

This routine loads a handle to an LP model from a deserializer. Specifically, this routine reads an LP handle from d and stores it in the position specified by lp_p.

Arguments

Name Description
env The environment to which the handle to be read belongs.
d The deserializer from which data is read.
lp_p Where to store the handle read from d.

Return

Returns 0 (zero) on success or an error code otherwise.
