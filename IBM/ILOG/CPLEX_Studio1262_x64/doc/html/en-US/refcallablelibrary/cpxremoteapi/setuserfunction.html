CPXXsetuserfunction and CPXsetuserfunction

This routine register a user function implementation.

int CPXXsetuserfunction( CPXMESSAGEHANDLERptr handler, CPXUSERFUNCTION * userfunction, void * handle )

int CPXsetuserfunction( CPXMESSAGEHANDLERptr handler, CPXUSERFUNCTION * userfunction, void * handle )

Description

This routine register a user function implementation.

The routine registers a user function implementation on this worker that will be invoked when the master invokes CPXXuserfunction/CPXuserfunction.

Arguments

Name Description
handler The message handler in which to install the user function.
userfunction The user function handler.
handle User data that is passed to userfunction.

Return

Returns 0 (zero) on success or an error code otherwise.
