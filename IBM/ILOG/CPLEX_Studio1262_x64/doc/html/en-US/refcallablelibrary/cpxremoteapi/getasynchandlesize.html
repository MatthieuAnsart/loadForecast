CPXXgetasynchandlesize and CPXgetasynchandlesize

This routine accesses the size of an asynchronous handle.

int CPXXgetasynchandlesize( CPXCASYNCptr asynchandle )

int CPXgetasynchandlesize( CPXCASYNCptr asynchandle )

Description

This routine accesses the size of an asynchronous handle.

Instances of the struct asynchandle should be treated as opaque data. However, there are cases in which such an instance must be made persistent (for example, when disconnecting temporarily from the remote end). In this case, you can use a combination of CPXXgetasynchandlesize/CPXgetasynchandlesize and CPXXasynchandlesave/CPXasynchandlesave to store the handle; then you can load it later on by means of CPXXasynchandleload/CPXasynchandleload.

Arguments

Name Description
asynchandle The handle for which the size is queried.

Return

The size of asynchandle in bytes.
