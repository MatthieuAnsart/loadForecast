CPXXfreeenvgroup and CPXfreeenvgroup

This routine deletes a remote environment group that was created by CPXXcreateenvgroup/CPXcreateenvgroup.

void CPXXfreeenvgroup( CPXENVGROUPptr * group_p )

void CPXfreeenvgroup( CPXENVGROUPptr * group_p )

Description

This routine deletes a remote environment group that was created by CPXXcreateenvgroup/CPXcreateenvgroup.

Arguments

Name Description
group_p A pointer to the group to be destroyed. Upon return, the value pointed to is NULL. The behavior and effects of this routine are undefind if group_p is not NULL and group_p does not point to a valid group.

Return
