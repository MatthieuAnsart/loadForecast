CPXXasynchandledel and CPXasynchandledel

This routine deletes an asynchronous handle that was created by an _async function or by CPXXasynchandleload/CPXasynchandleload.

void CPXXasynchandledel( CPXASYNCptr * asynchandle_p )

void CPXasynchandledel( CPXASYNCptr * asynchandle_p )

Description

This routine deletes an asynchronous handle that was created by an _async function or by CPXXasynchandleload/CPXasynchandleload.

Arguments

Name Description
asynchandle_p Pointer to the handle to be delete. This routine sets asynchandle_p to NULL.

Return
