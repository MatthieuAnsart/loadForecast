CPXXgetobjoffset_join and CPXgetobjoffset_join

This routine is the asynchronous version of CPXXgetobjoffset/CPXgetobjoffset.

int CPXXgetobjoffset_join( CPXASYNCptr * handle_p, double * objoffset_p )

int CPXgetobjoffset_join( CPXASYNCptr * handle_p, double * objoffset_p )

Description

This routine is the asynchronous version of CPXXgetobjoffset/CPXgetobjoffset. This routine waits for completion of asynchronous execution of CPXXgetobjoffset/CPXgetobjoffset. See the documentation of CPXXgetobjoffset/CPXgetobjoffset and the user manual for more details.
