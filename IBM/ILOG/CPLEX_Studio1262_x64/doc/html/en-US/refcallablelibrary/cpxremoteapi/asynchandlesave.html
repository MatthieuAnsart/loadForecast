CPXXasynchandlesave and CPXasynchandlesave

This routine saves an asynchronous handle into a buffer.

void CPXXasynchandlesave( CPXCASYNCptr asynchandle, void * buffer )

void CPXasynchandlesave( CPXCASYNCptr asynchandle, void * buffer )

Description

This routine saves an asynchronous handle into a buffer.

The function makes asynchandle persistent by writing a binary representation of it into buffer. The length of buffer must be at least as large as the number that CPXXgetasynchandlesize/CPXgetasynchandlesize returns for asynchandle.

This routine is the inverse of CPXXasynchandleload/CPXasynchandleload.

Arguments

Name Description
asynchandle The handle to make persistent.
buffer The buffer to which the binary representation of asynchandle is written. This buffer must be at least CPXXgetasynchandlesize/CPXgetasynchandlesize bytes long.

Return
