CPXXgetlps and CPXgetlps

Get the problem objects in a remote environment.

int CPXXgetlps( CPXCENVptr env, int capacity, CPXLPptr * lps, int * count_p, int * total_p )

int CPXgetlps( CPXCENVptr env, int capacity, CPXLPptr * lps, int * count_p, int * total_p )

Description

Get the problem objects in a remote environment.

This routine queries all the problem object handles in the remote environment env.

Arguments

Name Description
env The remote environment to query.
capacity Length of lps.
lps Array that receives the handles in env.
count_p Upon success this value is set to the number of elements stored in lps.
total_p Upon success this value is set to the total number of problem objects in env.

Return

Returns 0 (zero) on success or an error code on failure.
