CPXXgetuserfunction and CPXgetuserfunction

This routine queries the current user function implementation.

int CPXXgetuserfunction( CPXCMESSAGEHANDLERptr handler, CPXUSERFUNCTION ** userfunction_p, void ** handle_p )

int CPXgetuserfunction( CPXCMESSAGEHANDLERptr handler, CPXUSERFUNCTION ** userfunction_p, void ** handle_p )

Description

This routine queries the current user function implementation.

The routine returns the user function set with the last call to CPXXsetuserfunction/CPXsetuserfunction as well as the user data set with this function.

Arguments

Name Description
handler The message handler to query.
userfunction_p Where to store the address of the current user function.
handle_p Where to store the data of the current user function.

Return

Returns 0 (zero) on success or an error code otherwise.
