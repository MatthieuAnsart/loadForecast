CPXXgetnumqconstrs and CPXgetnumqconstrs

The routine CPXXgetnumqconstrs/CPXgetnumqconstrs is used to access the number of quadratic constraints in a CPLEX problem object.

CPXDIM CPXXgetnumqconstrs( CPXCENVptr env, CPXCLPptr lp )

int CPXgetnumqconstrs( CPXCENVptr env, CPXCLPptr lp )

Description

The routine CPXXgetnumqconstrs/CPXgetnumqconstrs is used to access the number of quadratic constraints in a CPLEX problem object.

Arguments

Name Description
env A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.

Return

If the problem object or environment does not exist, CPXXgetnumqconstrs/CPXgetnumqconstrs returns the value 0 (zero); otherwise, it returns the number of quadratic constraints.

Example

 cur_numqconstrs = CPXgetnumqconstrs (env, lp);
