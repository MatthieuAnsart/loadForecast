QCP and SOCP in the CPLEX Callable Library (C API) Reference Manual

These routines support models that represent a quadratically constrained program (QCP) or a second-order cone program (SOCP). See the CPLEX User's Manual, especially the topics Solving problems with quadratic constraints (QCP) and Representing SOCP as Lagrangian for more information about SOCP and QCP optimization.

  • CPXXaddqconstr and CPXaddqconstr
    The routine CPXXaddqconstr/CPXaddqconstr adds a quadratic constraint to a specified CPLEX problem object.
  • CPXXdelqconstrs and CPXdelqconstrs
    The routine CPXXdelqconstrs/CPXdelqconstrs deletes a range of quadratic constraints.
  • CPXXgetnumqconstrs and CPXgetnumqconstrs
    The routine CPXXgetnumqconstrs/CPXgetnumqconstrs is used to access the number of quadratic constraints in a CPLEX problem object.
  • CPXXgetqconstrindex and CPXgetqconstrindex
    The routine CPXXgetqconstrindex/CPXgetqconstrindex searches for the index number of the specified quadratic constraint in a CPLEX problem object.
  • CPXXgetqconstr and CPXgetqconstr
    The routine CPXXgetqconstr/CPXgetqconstr is used to access a specified quadratic constraint on the variables of a CPLEX problem object.
  • CPXXgetqconstrname and CPXgetqconstrname
    The routine CPXXgetqconstrname/CPXgetqconstrname is used to access the name of a specified quadratic constraint of a CPLEX problem object.
  • CPXXgetqconstrslack and CPXgetqconstrslack
    The routine CPXXgetqconstrslack/CPXgetqconstrslack is used to access the slack values for a range of the quadratic constraints of a quadratically constrained program.
  • CPXXqconstrslackfromx and CPXqconstrslackfromx
    The routine CPXXqconstrslackfromx/CPXqconstrslackfromx computes an array of slack values for quadratic constraints from primal solution values.
  • CPXXgetqconstrinfeas and CPXgetqconstrinfeas
    The routine CPXXgetqconstrinfeas/CPXgetqconstrinfeas computes the infeasibility of a given solution for a range of quadratic constraints.
  • CPXXgetxqxax and CPXgetxqxax
    The routine CPXXgetxqxax/CPXgetxqxax accesses quadratic constraint activity levels for a range of quadratic constraints in a quadratically constrained program (QCP).
  • CPXXgetqconstrdslack and CPXgetqconstrdslack
    The routine CPXXgetqconstrdslack/CPXgetqconstrdslack accesses the dual slack values for a quadratic constraint.
