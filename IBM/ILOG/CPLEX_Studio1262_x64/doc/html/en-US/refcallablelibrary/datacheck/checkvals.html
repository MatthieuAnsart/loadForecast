CPXXcheckvals and CPXcheckvals

Checks data by comparing an array of indices and an array of values for errors.

Signature

CPXCHECKLIBAPI
int CPXPUBLIC CPXXcheckvals (CPXCENVptr env, 
                             CPXCLPptr lp, 
                             CPXNNZ cnt,
                       const CPXDIM *rowind, 
                       const CPXDIM *colind,
                       const double *values);
CPXCHECKLIBAPI
int CPXPUBLIC CPXcheckvals  (CPXCENVptr env, 
                             CPXCLPptr lp, 
                             CPXNNZ cnt,
                       const CPXDIM *rowind, 
                       const CPXDIM *colind,
                       const double *values);

Description

This routine checks an array of indices and a corresponding array of values for errors in input. This routine is useful for validating the arguments of routines that modify a model, such modification routines as CPXXchgcoeflist and CPXchgcoeflist, CPXXchgbds and CPXchgbds, CPXXchgobj and CPXchgobj, or CPXXchgrhs and CPXchgrhs.

The source code of this data checking routine is in the file check.c, provided as part of the product.

To call this routine, you must compile and link check.c with your program as you compile and link with the CPLEX Callable Library.

Arguments

Name Description
env A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.
cnt The length of the arrays of indices and values to check.
rowind An array containing row indices. Can be NULL.
colind An array containing column indices. Can be NULL.
values An array of values. Can be NULL.

Return

The routine returns nonzero if it detects an error in the data; it returns zero if it does not detect any data errors.

Example

Consider the following call to CPXXchgobj and CPXchgobj:

status = CPXchgobj (env, lp, cnt, indices, values);

You can check the arguments of that invocation by calling CPXXcheckvals like this:

status = CPXcheckvals (env, lp, cnt, NULL, indices, values);
