CPXXgetnummipstarts and CPXgetnummipstarts

The routine CPXXgetnummipstarts/CPXgetnummipstarts accesses the number of MIP starts in the CPLEX problem object.

int CPXXgetnummipstarts( CPXCENVptr env, CPXCLPptr lp )

int CPXgetnummipstarts( CPXCENVptr env, CPXCLPptr lp )

Description

The routine CPXXgetnummipstarts/CPXgetnummipstarts accesses the number of MIP starts in the CPLEX problem object.

Arguments

Name Description
env A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.

Return

If the CPLEX problem object or environment does not exist, CPXXgetnummipstarts/CPXgetnummipstarts returns the value 0 (zero); otherwise, it returns the number of solutions.

Example

 numsolns = CPXgetnummipstarts (env, lp);
