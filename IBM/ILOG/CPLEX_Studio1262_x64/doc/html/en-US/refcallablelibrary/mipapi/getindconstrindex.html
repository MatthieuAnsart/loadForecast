CPXXgetindconstrindex and CPXgetindconstrindex

The routine CPXXgetindconstrindex/CPXgetindconstrindex searches for the index number of the specified indicator constraint in a CPLEX problem object.

int CPXXgetindconstrindex( CPXCENVptr env, CPXCLPptr lp, char const * lname_str, CPXDIM * index_p )

int CPXgetindconstrindex( CPXCENVptr env, CPXCLPptr lp, char const * lname_str, int * index_p )

Description

The routine CPXXgetindconstrindex/CPXgetindconstrindex searches for the index number of the specified indicator constraint in a CPLEX problem object.

Arguments

Name Description
env A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.
lname_str Name of an indicator constraint to search for.
index_p A pointer to an integer to hold the index number of the indicator constraint with the name lname_str. If the routine is successful, *index_p contains the index number; otherwise, *index_p is undefined.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example

 status = CPXgetindconstrindex (env, lp, "resource89", &indconstrindex);
