CPXXgetnodecnt and CPXgetnodecnt

The routine CPXXgetnodecnt/CPXgetnodecnt accesses the number of nodes used to solve a mixed integer problem.

CPXCNT CPXXgetnodecnt( CPXCENVptr env, CPXCLPptr lp )

int CPXgetnodecnt( CPXCENVptr env, CPXCLPptr lp )

Description

The routine CPXXgetnodecnt/CPXgetnodecnt accesses the number of nodes used to solve a mixed integer problem.

Arguments

Name Description
env A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.

Return

If a solution exists, CPXgetnodecnt returns the node count. If no solution, problem, or environment exists, CPXgetnodecnt returns the value 0 (zero).

Example

 nodecount = CPXgetnodecnt (env, lp);
