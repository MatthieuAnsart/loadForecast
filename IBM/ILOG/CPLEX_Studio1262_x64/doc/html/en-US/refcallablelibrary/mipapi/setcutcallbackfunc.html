CPXsetcutcallbackfunc

int CPXsetcutcallbackfunc( CPXENVptr env, int(CPXPUBLIC *cutcallback)(CALLBACK_CUT_ARGS), void * cbhandle )

Description

Warning:

The routine CPXsetcutcallbackfunc is deprecated and will be removed in a future release. Instead, use either CPXXsetlazyconstraintcallbackfunc/CPXsetlazyconstraintcallbackfunc or CPXXsetusercutcallbackfunc/CPXsetusercutcallbackfunc. These recommended replacement routines correctly distinguish user cut callbacks from lazy constraint callbacks. The deprecated routine and its replacements cannot be called in the same application.

Arguments

Name Description
env
cutcallback
cbhandle
