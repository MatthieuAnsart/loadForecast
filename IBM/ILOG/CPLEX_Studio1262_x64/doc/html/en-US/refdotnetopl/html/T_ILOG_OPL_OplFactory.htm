Remarks

The OPL factory allows you to create instances of OPL objects like models, datasources, etc. It initializes the OPL environment. The method OplFactory.end() allows you to clean the memory used by all objects allocated by this factory. Be sure to use it when OPL is no longer used.
You can also manage memory more finely by calling end() on individual objects earlier, as soon as you don't need them.
