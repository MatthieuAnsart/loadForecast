Remarks

The piecewise linear function is evaluated at expr and is defined by the remaining arguments. The array points contains breakpoints such that points[i-1] is less than or equal to points[i]. The array slopes contains the value of the slope for each piece defined by the breakpoints. The slope of the function for values less than points[0] is slopes[0]. Similarly, slopes[i] specifies the slope between points[i-1] and points[i]. Thus the array slope must have one more element than the array points.

By setting points[i-1] == points[i], you can represent a discontinuous piecewise linear function. In this case, the function will make a step of size slopes[i-1] at position points[i].

The values a and fa define the x and y coordinates of one point of the piecewise linear function. This point is referred to as the anchor point.
