Return Value

IsCalculated
Note
This member function is deprecated in OPL 5.1. You should use #OplElement::isInternalData() instead.
This member function enables you to know whether the element is internal data, formerly called "calculated data".

Return Value

IloTrue if the element is internal data.

IsData
Note
This member function is deprecated in OPL 5.1. You should use #OplElement::isExternalData() instead.
This member function enables you to know whether the element is data from a data source.

Return Value

IloTrue if the element is data from a data source.

IsDecisionExpression
This member function enables you to know whether the element is a decision expression.

Return Value

IloTrue if the element is a decision expression.

IsDecisionVariable
This member function enables you to know whether the element is a decision variable.

Return Value

IloTrue if the element is a decision variable.

IsExternalData
This member function enables you to know whether the element is data that is external to the model. The return value indicates whether the data is from an external data source.

Return Value

IloTrue if the element is data from a data source.

IsInternalData
This member function enables you to know whether the element is data that is initialized in the model. The return value indicates whether the data is internal data.

Return Value

IloTrue if the element is internal data.
