Remarks

All terms in the invoking linear expression having any variable in var[start], ..., var[start+num-1] are removed from the invoking ILinearNumExpr object. If variable var occurs more than once in the invoking ILinearNumExpr, all occurrences are removed. The ILinearNumExpr.Enumerator can be used to remove individual terms regardless of duplicates.
