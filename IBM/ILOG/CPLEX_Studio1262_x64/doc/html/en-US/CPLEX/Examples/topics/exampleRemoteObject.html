Examples of the CPLEX Remote Object and Distributed Parallel MIP

These examples in C, C++, and Java illustrate how to use the CPLEX remote object. There are also examples in C, C++, C#.NET, and Python showing how to develop an application of distributed parallel mixed integer programming (MIP).

The following examples are delivered with CPLEX in IBM ILOG CPLEX Optimization Studio. They illustrate how to use the CPLEX remote object to build applications that can be distributed among multiple machines to run in parallel in either the C, C++, or Java programming language.

parbenders.c uses the CPLEX remote object to implement a Benders decomposition of a model and to solve the decomposition in parallel on different machines.
parmipopt.c runs parallel distributed concurrent mipopt to solve the same problem on different machines with a different parameter setting on each machine.
iloparbenders.cpp uses the CPLEX remote object to implement a Benders decomposition of a model and to solve the decomposition in parallel on different machines.
iloparmipopt.cpp runs parallel distributed concurrent mipopt to solve the same problem on different machines with a different parameter setting on each machine.
RemoteParBenders.java uses the CPLEX remote object to implement a Benders decomposition of a model and to solve the decomposition in parallel on different machines.

The following examples are delivered with CPLEX in IBM ILOG CPLEX Optimization Studio. They illustrate how to develop a MIP optimization distributed among multiple machines to run in parallel in either the C, C++, Java, C#.NET, or Python programming language. There are also examples for MATLAB users.

xdistmipex1.c runs MIP optimization distributed in parallel.
xdistmipex2.c runs MIP optimization distributed in parallel and applies an informational callback.
ilodistmipex1.cpp runs MIP optimization distributed in parallel.
ilodistmipex2.cpp runs MIP optimization distributed in parallel and applies an informational callback.
DistMIPEx1.java runs MIP optimization distributed in parallel.
DistMIPEx1.java runs MIP optimization distributed in parallel and applies an informational callback.
distmipex1.py runs MIP optimization distributed in parallel.
distmipex2.py runs MIP optimization distributed in parallel and applies an informational callback.
distmipex1.m runs MIP optimization distributed in parallel. This example is for MATLAB users.
distmipex2.m runs MIP optimization distributed in parallel and applies an informational callback. This example is for MATLAB users.
DistMIPEx1.cs runs MIP optimization distributed in parallel. This example is available only on Windows platforms.
DistMIPEx2.cs runs MIP optimization distributed in parallel and applies an informational callback. This example is available only on Windows platforms.
