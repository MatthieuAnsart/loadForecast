Example: cplexbilpex.m

This example uses the function cplexbilp to solve a binary integer programming model.

The BILP problem solved in this example is:

   Maximize  x1 + 2 x2 + 3 x3 + x4 
   Subject to 
      - x1 +   x2 + x3 + 10 x4 <= 20 
        x1 - 3 x2 + x3         <= 30 
               x2      - 3.5x4  = 0 
   Binary Integer 
        x1 x2 x3 x4
