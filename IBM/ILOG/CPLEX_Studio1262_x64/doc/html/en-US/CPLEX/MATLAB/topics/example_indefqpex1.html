Example: indefqpex1.m

This example illustrates first-order solutions in a quadratic programming problem.

The quadratic problem solved in this example is:

    Maximize  - 0.5 (-3 * x^2 - 3 * y^2 - 1 * x * y)
    Subject to
        -x + y >= 0
         x + y >= 0
    Bounds
         -1 <= x <= 1
          0 <= y <= 1
