Interactive Optimizer of CPLEX

This manual lists commands and options of the Interactive Optimizer, a component of CPLEX. The list of commands includes links to examples in Getting Started and in the User's Manual. For a step-by-step introduction to the Interactive Optimizer, see the Interactive Optimizer tutorial in Getting Started with CPLEX. After the list of commands in this manual, you can also find topics explaining how to manage parameters, how to save a parameter specification file from an interactive session, and how to use a file of commands from the command line of your operating system.
