File formats supported by CPLEX

This manual documents the file formats supported by CPLEX. It begins with a brief description of the file formats in alphabetic order. This manual continues with longer explanations of the formats.

  • Brief descriptions of file formats
    CPLEX recognizes and supports these file formats.
  • Reading and entering file formats in the Interactive Optimizer
    These file formats are appropriate for the Interactive Optimizer.
  • Saving problems in the Interactive Optimizer
    CPLEX offers several file formats to save a problem in the Interactive Optimizer.
  • LP file format: algebraic representation
    CPLEX conforms to these rules for the LP file format.
  • MPS file format: industry standard
    CPLEX supports the industry standard MPS file format and offers extensions to it.
  • Special records in MPS files: CPLEX extensions
    CPLEX extends the MPS standard in several ways.
  • CSV file format: comma separated values
    CPLEX supports CSV file format for comma separated values.
  • BAS file format: advanced basis
    The BAS file format supports an advanced basis.
  • FLT file format: filter files for the solution pool
    The FLT file format supports filters of the solution pool.
  • MST file format: MIP starts
    The MST file format supports MIP starts.
  • NET file format: network flow models
    CPLEX supports the NET file format.
  • ORD file format: priorities and branching orders
    The ORD file format supports priorities and branching orders.
  • PRM file format: parameter settings
    CPLEX supports the PRM file format for nondefault parameter settings.
  • SAV file format: numerically accurate binary files
    CPLEX supports the SAV binary file format for numerical accuracy in reading and writing problems.
  • SOL file format: solution files
    The SOL file format supports solution files.
  • VMC file format: virtual machine configuration for distributed parallel optimization
    File format to specify a virtual machine configuration for distributed parallel optimization
  • XML file format: serialized models and solutions
    CPLEX offers an XML file format to support serialized models and solutions.
