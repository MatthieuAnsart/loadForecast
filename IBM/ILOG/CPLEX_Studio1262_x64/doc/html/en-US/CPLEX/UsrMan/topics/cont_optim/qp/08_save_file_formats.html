Saving QP problems

Recommends appropriate file formats to save a quadratic program.

After you enter a QP problem, whether interactively or by reading a formatted file, you can then save the problem in a formatted file. The formats available to you are LP, MPS, and SAV. (These formats are documented in the reference manual, File formats supported by CPLEX.) When you save a QP problem in one of these formats, the quadratic information will also be recorded in the formatted file.
