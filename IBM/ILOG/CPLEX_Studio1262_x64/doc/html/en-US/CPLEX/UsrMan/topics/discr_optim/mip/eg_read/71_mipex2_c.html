mipex2.c

Shows how to read data from a file and solve a MIP model in the C API.

The example derives from lpex2.c, an LP example explained in the manual Getting Started. That LP example differs from this MIP example in these ways:

  • This example solves only MIPs, so it calls only CPXmipopt, and its command line does not require the user to specify an optimizer.

  • This example does not generate or print a basis.

Like other applications based on the Callable Library, this one calls CPXopenCPLEX to initialize the CPLEX environment; it sets the screen-indicator parameter to direct output to the screen and calls CPXcreateprob to create a problem object. Before it ends, it calls CPXfreeprob to free the space allocated to the problem object and CPXcloseCPLEX to free the environment.
