Transport types for the remote object: local

Documents the local transport protocol supporting the remote object.

If the transport argument to CPXXopenCPLEXremote is NULL or the empty string, then the arguments argc, argv are ignored, and the call to CPXXopenCPLEXremote behaves like an ordinary call to CPXXopenCPLEX. That is, the handle returned is a handle to a local instance of the CPLEX Callable Library.
