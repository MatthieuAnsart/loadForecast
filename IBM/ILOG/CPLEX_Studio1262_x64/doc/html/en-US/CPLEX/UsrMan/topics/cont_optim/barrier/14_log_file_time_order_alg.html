Ordering-algorithm time in the log file

Describes the report of the time used by ordering algorithm of the barrier optimizer.

After the number of nonzeros in the lower triangle of AAT, CPLEX records the time required by the ordering algorithm. (The CPLEX barrier optimizer offers you a choice of several ordering algorithms, explained in Choosing an ordering algorithm.) This section in the log file indicates which ordering algorithm the default Automatic setting chose.
