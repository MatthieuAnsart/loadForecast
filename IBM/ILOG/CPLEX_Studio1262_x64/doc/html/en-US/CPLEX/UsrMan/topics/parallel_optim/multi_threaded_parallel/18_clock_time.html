Clock settings and time measurement

Describes reporting of time in parallel MIP optimization.

On most platforms, CPLEX offers two ways of measuring time: CPU time and wall-clock time. Wall-clock time is the default on all platforms.

You can choose the type of clock by setting the clock type for computation time parameter to a value different from the default 2.

  • In Concert Technology, use the method:

    • IloCplex::setParam(ClockType , i ) in the C++ API;

    • IloCplex.setParam(ClockType, i ) in the Java API;

    • Cplex.SetParam(ClockType, i ) in the .NET API.

  • In the Callable Library, use the routine CPXsetintparam(env, CPX_PARAM_CLOCKTYPE ,  i ) .

  • In the Interactive Optimizer, use the command set clocktype i.

Replace the i with the value 0 (zero, the automatic setting) to use CPU time for sequential optimizers and wall-clock time for parallel optimizers; 1 (one) to specify CPU time; or 2 to specify wall-clock time.
