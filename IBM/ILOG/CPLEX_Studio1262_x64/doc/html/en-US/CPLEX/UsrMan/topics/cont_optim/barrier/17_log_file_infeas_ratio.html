Infeasibility ratio in the log file

Identifies the infeasibility ratio reported in the log file of the barrier optimizer.

If you are using one of the barrier infeasibility algorithms available in the CPLEX barrier optimizer (that is, if you have set BarAlg to either 1 or 2, as discussed later in this chapter), then CPLEX records a column of output titled Inf Ratio, the infeasibility ratio. This ratio, always positive, is a measure of progress for that particular algorithm. In a problem with an optimal solution, you will see this ratio increase to a large number. In contrast, in a problem that is infeasible or unbounded, this ratio will decrease to a very small number. (The infeasibility ratio is not available in the standard barrier algorithm, that is, when the value of BarAlg is set to 3.)
