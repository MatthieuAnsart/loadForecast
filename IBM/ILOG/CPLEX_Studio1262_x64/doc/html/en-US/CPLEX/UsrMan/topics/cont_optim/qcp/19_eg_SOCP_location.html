Examples: SOCP

Lists examples of SOCP.

The following examples, distributed with the product, show how to create and populate that SOCP model and then how to access dual values and reduced costs for the SOCP model in each API:

  • In the Callable Library (C API), see socpex1.c and xsocpex1.c.

  • In the C++ API, see ilosocpex1.cpp.

  • In the Java API, see SocpEx1.java.

  • In the .NET API, see SocpEx1.cs and SocpEx1.vb.

  • In the Python API, see socpex1.py.

  • In the MATLAB connector, see socpex1.m.
