Fetching the results of distributed concurrent MIP optimization

Fetches remote results in an example of distributed concurrent optimization of a mixed integer program.

In order to access the x vector of the optimal solution, the example (parmipopt.c) simply keeps track of the remote object that reported the best feasible solution. In fact, the argument bestidx is the index of the remote object that reported the best feasible solution.
