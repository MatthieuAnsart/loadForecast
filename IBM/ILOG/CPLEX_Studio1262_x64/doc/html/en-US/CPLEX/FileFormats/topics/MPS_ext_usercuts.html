User-defined cuts in MPS files

CPLEX extensions of MPS file format support user defined cuts.

The advanced feature user defined cuts can be declared in a special section following the ROWS section. The title of this section is USERCUTS . The order of sections must be ROWS USERCUTS. The format of the USERCUTS section is the same as the format of the ROWS section with this exception: the type must be one of E, L, or G; the row must not be ranged. For more information about user-defined cuts, see User-cut and lazy-constraint pools in the CPLEX User’s Manual.
