Additional documentation of the Python API

Introduces a new topic about the Python API.

In the CPLEX User’s Manual, the new topic CPLEX for Python users documents the Python API of CPLEX more fully than the introductory Python tutorial already available in Getting Started. In particular, this new topic shows how to write callbacks for applications of the Python API. It also offers an example where the model is modified and the problem solved again in a sequence of improvements. The new topic also introduces the reporting features of the Python API through short coding examples.
