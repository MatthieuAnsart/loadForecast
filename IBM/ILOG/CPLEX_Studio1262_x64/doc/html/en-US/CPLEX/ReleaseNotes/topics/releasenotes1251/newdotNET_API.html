New features of the Concert .NET API

Introduces new features of the Concert .NET API in CPLEX V12.5.1.

These notes highlight new features of interest to users of the Concert .NET API in CPLEX V12.5.1.

To support the new parameter, calculate QCP dual values, CPLEX V12.5.1 introduces a new class and new constants in the .NET API. This new class Cplex.CalcQCPDuals enumerates possible values of the parameter CalcQCPDuals.
  • QCPDualsNo Do not calculate dual values for the QCP.
  • QCPDualsIfPossible Calculate dual values for the QCP as long as the calculations do not interfere with presolve reductions. default
  • QCPDualsForce Calculate dual values and disable any presolve reductions that interfere with these calculations.
