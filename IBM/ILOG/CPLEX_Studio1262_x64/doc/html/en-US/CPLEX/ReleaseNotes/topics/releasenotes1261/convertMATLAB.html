Conversion notes for users of the MATLAB Connector

Explains changes introduced in the MATLAB Connector by CPLEX V12.6.1.

These notes explain changes introduced in CPLEX V12.6.1 for users of the MATLAB Connector.

Path changed to MATLAB Connector for Linux and OSX users

In order to support your installation of multiple platforms at the same relative location in your file system without over-writing platform-specific files, CPLEX now distinguishes the path to the following platforms:

Platform New Path
OSX cplex/matlab/x86-64_osx
Linux 32-bit cplex/matlab/x86_linux
Linux 64-bit cplex/matlab/x86-64_linux

For Windows users, the default installation path remains the same, as CPLEX already distinguished cplex/matlab/x86_win32 for MATLAB on 32-bit Windows and cplex/matlab/x64_win64 for MATLAB on 64-bit Windows.

Options removed

In order to conform to changes by MathWorks in the MATLAB Toolbox options, the following options have been removed from CPLEX V12.6.1:

  • option.LargeScale
  • option.Simplex
  • option.Diagnose

Use the option.Display (instead of option.Diagnose) to display diagnostic information. Possible values of option.Display are:

  • off
  • on
  • iter

Use one of the possible values of option.Algorithm (instead of option.LargeScale or option.Simplex). Possible values are:

  • auto
  • primal
  • dual
  • interior-point
