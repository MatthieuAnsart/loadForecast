New error codes

Lists error codes newly added in this release.

These new error codes have been added to CPLEX V12.5:

  • 1811 CPXERR_UNSUPPORTED_OPERATION. The user attempted an operation that is not supported in the environment passed to the routine, or the user attempted an operation that is not supported at all by this instance of the library.
  • 1812 CPXERR_PROTOCOL A protocol error occurred during communication with a remote object.
  • 1813 CPXERR_SYSCALL A call to an operating system routine or external library failed unexpectedly.
  • 1814 CPXERR_DYNLOAD A dynamically loadable library could not be found or loaded.
  • 1815 CPXERR_DYNFUNC A dynamically loadable library did not provide a function that it was required to provide.
