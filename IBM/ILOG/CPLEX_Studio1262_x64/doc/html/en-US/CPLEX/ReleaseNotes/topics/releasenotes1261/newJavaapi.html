New features of the Java API

Introduces new features of the Java API in CPLEX V12.6.1.

These notes highlight new features of interest to users of the Java API in CPLEX V12.6.1.

New methods to support the remote object

The Java API of CPLEX V12.6.1 offers these new methods to support the remote object. These methods manage transport control.
  • IloCplex.transportctrl(int ctrl, double[] data)
  • IloCplex.transportctrl(int ctrl, int[] data)
  • IloCplex.transportctrl(int ctrl, long[] data)

Source of the solution and a lazy constraint callback

CPLEX V12.6.1. offers a new method to query the source of a solution to determine whether a lazy constraint callback was invoked because of a solution found during a heuristic or because of a solution found at a node in the search tree. For more detail, see the documentation of this new method and the existing enumeration:

  • IloCplex.LazyConstraintCallback.getSolutionSource
  • IloCplex.IncumbentCallback.SolutionSource
