Location of VMC examples

Explains changes introduced by CPLEX V12.6.1 that apply to users who consult examples of virtual machine configuration (VMC) files.

In CPLEX V12.6.1, the location of VMC files (that is, examples of configuration files for virtual machines for distributed parallel optimization) has changed. Formerly, VMC examples of user projects were located in yourCPLEXhome/examples/data. Now, those VMC examples of user projects are located in yourCPLEXhome/examples/SYSTEM/FORMAT where

  • yourCPLEXhome specifies the folder or directory in your file system where you installed CPLEX as part of IBM ILOG CPLEX Optimization Studio.
  • SYSTEM specifies your combination of computer and operating system.
  • FORMAT specifies your compiler.

This change in organization of the VMC examples allows CPLEX V12.6.1 to provide project files that are port-specific for users.

Environment variables in VMC files

CPLEX expands an environment variable cited in a VMC file at the time that CPLEX reads the VMC file. Reference to an environment variable must be of the form: 
$(name)
In particular, CPLEX does not recognize $name nor ${name}. CPLEX interprets the character sequence $$ as a single $. This convention means that you can produce verbatim $( by writing $$(.
