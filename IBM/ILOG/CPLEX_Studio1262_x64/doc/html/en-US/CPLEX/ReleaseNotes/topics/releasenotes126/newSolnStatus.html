New solution status

Documents a new solution status in CPLEX V12.6.0 when CPLEX solves for a global solution of nonconvex QP and MIQP

As announced in Global solution of nonconvex MIQP in these release notes, CPLEX V12.6.0 can solve to global optimality both QP and MIQP problem types. This extended facility entails a new solution status: CPXMIP_ABORT_RELAXATION_UNBOUNDED, returned in the case where it is not possible to build a relaxation with finite bounds for the original problem and it is not possible to prove that the original problem was unbounded. (That is, both conditions hold.)
