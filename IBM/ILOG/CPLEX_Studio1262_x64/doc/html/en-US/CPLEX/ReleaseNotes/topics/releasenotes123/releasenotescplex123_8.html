Error codes added

Lists error codes new in this release.

The following error codes have been added to CPLEX V12.3:

  • 1810 CPXERR_OVERFLOW 32-bit overflow. The operation requires the return of 64-bit integer data but only 32-bit integer variables were provided by the user.

  • 1235 CPXERR_ENCODING_CONVERSION Code page conversion failed. An attempt to convert a string from or to the encoding chosen by the user to or from the internal encoding failed. This condition occurs when a user submits a string that does not conform to the selected encoding.

  • 1808 CPXERR_LOCK_CREATE Creation of lock failed The operating system could not create (another) lock object. This condition usually means that you do not have the appropriate privileges or the operating system has run out of resources.

  • 1809 CPXERR_SYNCPRIM_CREATE Creation of synchronization primitive failed. The operating system could not create (another) synchronization primitive. This condition usually means that you do not have the appropriate privileges or the operating system has run out of resources.
