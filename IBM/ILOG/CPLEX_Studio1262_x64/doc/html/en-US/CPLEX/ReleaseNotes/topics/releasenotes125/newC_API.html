New features for users of the Callable Library (C API)

Documents new features available to users of the Callable Library (C API).

These notes highlight new features of interest to users of the Callable Library (C API) of CPLEX V12.5.

Attaching user data at a branch from a callback

New functionality supports users who want to attach data to a node in a branch callback. In other words, this new functionality allows your application to branch as CPLEX would at a given node, while also allowing your application to attach data from a callback to the node. To take advantage of this new functionality in the Callable Library (C API), use the new routine CPXbranchcallbackbranchasCPLEX.
