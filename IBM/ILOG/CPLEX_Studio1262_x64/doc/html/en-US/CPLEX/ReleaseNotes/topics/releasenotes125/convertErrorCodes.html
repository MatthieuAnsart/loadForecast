Error codes changed

Lists changed error codes in this release.

The following error code changed in CPLEX V12.5:

1224 CPXERR_NULL_NAME Null pointers were not allowed in names previously. This error code is no longer needed and thus deprecated.

For background information about this change, see also the announcement Names and file formats in these release notes.

See also the new error codes added in CPLEX V12.5, as announced in these release notes New error codes.
