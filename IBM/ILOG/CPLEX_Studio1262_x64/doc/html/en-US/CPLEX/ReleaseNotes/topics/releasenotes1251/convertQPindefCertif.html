Documentation of CPXqpindefcertificate and CPXXqpindefcertificate changed

Error corrected in the documentation of Callable Library (C API) routines CPXqpindefcertificate and CPXXqpindefcertificate

In previous versions, the reference documentation of the Callable Library (C API) routines CPXXqpindefcertificate and CPXqpindefcertificate incorrectly stated that the output array x[] required only the number of elements returned by the routines CPXXgetnumquad and CPXgetnumquad.

That statement in the reference documentation was incorrect.

Reference documentation distributed with CPLEX V12.5.1 has been corrected so that the description of the routines CPXXqpindefcertificate and CPXqpindefcertificate now states correctly that the output array x[] requires the number of elements returned by the routines CPXXgetnumcols and CPXgetnumcols.
