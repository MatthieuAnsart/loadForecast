Simplified threading and parallel search

Describes improvements in user’s control of threads, parallel search, and callbacks.

In CPLEX V12.3, the parallel mode switch (CPX_PARAM_PARALLELMODE, ParallelMode) is independent of the global thread count parameter (CPX_PARAM_THREADS, Threads). The default setting of the parallel mode switch (0 zero) is now equivalent to choosing deterministic (formerly, the value 1 one). Only if you set this parameter to opportunistic search mode (-1) will you get the opportunistic algorithm.

In particular, if the parallel mode switch (CPX_PARAM_PARALLELMODE, ParallelMode) is at its default value (0 zero), then setting the global thread count parameter (CPX_PARAM_THREADS, Threads) to a nondefault value will no longer implicitly make CPLEX switch to opportunistic mode.

At its default value (0 zero), the global thread count parameter (CPX_PARAM_THREADS, Threads) now behaves differently for mixed integer programming problems (MIPs), depending on whether callbacks are registered in your application.

  • If no callbacks are registered, CPLEX uses the maximum number of threads (that is, the number of cores on your machine) up to 32 threads.

  • If only a MIP information callback is registered, CPLEX uses the maximum number of threads (that is, the number of cores on your machine) up to 32 threads.

    For documentation about informational callbacks, see the topic What is an informational callback? in the CPLEX User’s Manual.

  • If other callbacks are registered, such as a MIP callback, an LP callback, a network simplex callback, or one of the MIP control callbacks, then CPLEX uses 1 (one) thread.

    If you want parallel execution of your application in the presence of callbacks other than an information callback, you need to set the global thread count parameter (CPX_PARAM_THREADS, Threads) explicitly to a positive value.

    For documentation of control and query callbacks, see the topics What are MIP control callbacks? and What are query or diagnostic callbacks? in the CPLEX User’s Manual.
