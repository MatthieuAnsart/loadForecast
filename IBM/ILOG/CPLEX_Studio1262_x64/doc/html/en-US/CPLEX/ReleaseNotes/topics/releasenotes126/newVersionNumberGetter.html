New accessor for current version number

A new feature accesses the current version number of your CPLEX installation.

A new feature in CPLEX V12.6.0 access the current version number of your installation.

  • In the Callable Library (C API), use the new routine CPXXversionnumber and CPXversionnumber.
  • In the C++ API, use the method IloCplex::getVersionNumber.
  • In the Java API, use the method IloCplex.getVersionNumber.
  • In the .NET API, use the interface method CplexI.GetVersion.
  • In the Python API, use the method get_versionnumber.
  • In the MATLAB (Cplex class API), use the method Cplex.getVersionNumber.
