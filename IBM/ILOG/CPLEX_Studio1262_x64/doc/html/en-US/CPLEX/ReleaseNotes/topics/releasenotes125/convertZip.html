Compressed file conversion note for all users

CPLEX V12.5 supports compressed file formats; limitations apply.

CPLEX V12.5 supports compressed files on all platforms, including Microsoft Windows, UNIX, and GNU/Linux. In particular, CPLEX can read and write compressed files with these file extensions:

  • .bz2
  • .gz

Limitations apply

  • CPLEX V12.5 does not support files compressed by WinZip.
  • CPLEX V12.5 does not read files that were compressed by the UNIX or GNU/Linux compression tool named "compress" which produces a file with the extension .Z. (Earlier versions of CPLEX that successfully read files with the .Z extension relied on a tool that CPLEX no longer uses.)

Remedy

If you have a file which you want CPLEX to read, but the file was compressed by "compress" or by WinZip, then you must first uncompress the file by means of the appropriate tool, and then pass the uncompressed file to CPLEX. If your situation requires that you pass a compressed file to CPLEX, then you can uncompress that file and compress it with a compression tool known to CPLEX, such as one of these compression tools:

  • bzip2 to produce a compressed file with the file extension .bz2
  • gzip to produce a compressed file with the file extension .gz
