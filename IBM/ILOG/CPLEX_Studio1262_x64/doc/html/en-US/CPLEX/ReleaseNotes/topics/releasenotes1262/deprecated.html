Deprecated

Documents deprecated elements in CPLEX V12.6.2.

These notes announce deprecated elements of CPLEX V12.6.2.

Deprecated parameter name and number

As detailed in the conversion note Optimality target: new name for former solution target parameter, the name of the SolutionTarget parameter has been deprecated. The behavior of this parameter remains the same. The new name of the parameter varies by application programming interface, as documented in optimality target.

  • CPXPARAM_OptimalityTarget in the Callable Library (C API)
  • IloCplex::Param::OptimalityTarget in the C++ API
  • IloCplex.Param.OptimalityTarget in the Java API
  • Cplex.Param.OptimalityTarget in the .NET API
  • parameters.optimalitytarget in the Python API
  • Cplex.Param.optimalitytarget in the MATLAB API
  • optimalitytarget in the Interactive Optimizer

Deprecated value of parameter for barrier crossover algorithm

Formerly, the parameter barrier crossover algorithm accepted the value -1 (minus one) meaning that the user specified to CPLEX to perform no crossover at the end of barrier optimization. In CPLEX V12.6.2, this value has been deprecated and will be removed in a future release.

Other values of this parameter remain valid.

Deprecated channel routines in the Callable Library (C API)

The routine CPXXaddchannel, CPXaddchannel has been deprecated and will be removed from the next release of CPLEX.

The routine CPXXdelchannel, CPXdelchannel has been deprecated and will be removed from the next release of CPLEX.

Deprecated error message: CPXERR_NO_RNGVAL

The error message CPXERR_NO_RNGVAL, number 1216, has been deprecated. The conditions that could raise this error no longer exist. The error message will be removed from CPLEX in a future release.

Deprecated Python method

The Python method Cplex.presolve.get_objective_offset has been deprecated. This deprecated method will be removed from the next release of CPLEX. To access the objective offset, use the method Cplex.objective.get_offset instead.
