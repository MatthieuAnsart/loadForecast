Conversion note for Python users

Explains a change of interest to users of the Python API of CPLEX.

The spelling of these attributes has changed:

  • max_indicator_slack_infeasibilty

  • sum_indicator_slack_infeasibilty

(Notice the missing i between l and ty.)

Now these attributes are spelled correctly:

  • max_indicator_slack_infeasibility

  • sum_indicator_slack_infeasibility

These attributes appear in members of the classes supporting quality metrics of solutions, including solutions in the solution pool, specifically, in these classes:

  • cplex._internal._subinterfaces.SolutionPoolInterface

  • cplex._internal._subinterfaces.QualityMetric

If these attributes appear in your compiled Python applications, you need to recompile. If these attributes appear in your interactive Python sessions, the feature that automatically completes names of attributes as you type supplies them correctly now.
