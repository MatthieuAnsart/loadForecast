New features of the .NET API

Introduces new features of the .NET API in CPLEX V12.6.1.

These notes highlight new features of interest to users of the .NET API in CPLEX V12.6.1.

Source of the solution and a lazy constraint callback

CPLEX V12.6.1. offers a new property: the source of a solution to specify whether a lazy constraint callback was invoked because of a solution found during a heuristic or because of a solution found at a node in the search tree. For more detail, see the documentation of this new property and its possible values:

  • Cplex.LazyConstraintCallback.Source
  • Cplex.IncumbentCallback.SolutionSource
