Callbacks can store user data at nodes

Announces new routines and methods for users to store data at nodes from callbacks.

To store user data at nodes and to access the user data from callbacks, CPLEX offers these new routines and methods, documented in the reference manuals of the APIs:

  • Callable Library (C API)
    • CPXXcallbacksetnodeuserhandle
    • CPXcallbacksetnodeuserhandle
  • Python API NodeCallback.set_node_data
  • C++ API IloCplex.NodeCallback.setNodeData(long,Object)
  • Java API IloCplex.NodeCallback.setNodeData(long,Object)
  • .NET API Cplex.NodeCallback.SetNodeData

    Additionally, the NodeData property in both control callbacks and in incumbent callbacks is no longer read-only. You can now set this property; in other words, you can store user data from callbacks.
