MIP kappa

Introduces computation of condition number for mixed integer programming solutions.

Kappa is already familiar to many practitioners in operations research and related fields as the condition number, a measure of the quality of a simplex linear programming (LP) solution. CPLEX V12.2 implements an extension of this idea to compute MIP kappa, that is, statistics about the condition number of the optimal bases CPLEX encountered during the solution of a MIP model. For more detail about the parameter to invoke MIP kappa computation in your application, see the topic MIP kappa computation in the Parameters Reference Manual.
