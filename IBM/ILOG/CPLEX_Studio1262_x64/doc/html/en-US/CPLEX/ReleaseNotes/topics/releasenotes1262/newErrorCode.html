New error codes in CPLEX V12.6.2

Announces new error code in CPLEX V12.6.2.

CPLEX V12.6.2 introduces new error codes.

  • CPXERR_CPUBINDING_FAILURE which arises when an attempt fails to bind threads to cores in a multithread application on a multicore or multiprocessor platform.

  • CPXERR_MULTIPLE_PROBS_IN_REMOTE_ENVIRONMENT which arises when there is an attempt to create more than one model in the same remote environment. A remote environment supports only one problem object.
