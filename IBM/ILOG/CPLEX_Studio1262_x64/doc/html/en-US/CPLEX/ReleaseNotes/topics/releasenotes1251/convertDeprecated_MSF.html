Deprecated: CPLEX connectors for MSF and Excel

CPLEX connectors for Microsoft Solver Foundation and Excel have been deprecated.

CPLEX connector for Microsoft Solver Foundation (MSF) deprecated

The CPLEX connector for Microsoft Solver Foundation (MSF) is deprecated in CPLEX V12.5.1. It will be removed in a future release. Clients currently using this feature should consider migrating to the OPL data source feature of IBM ILOG CPLEX Optimization Studio or using the .NET interfaces of CPLEX instead.

CPLEX connector for Excel deprecated

The CPLEX connector for Excel was deprecated in CPLEX V12.5.0. It will be removed in a future release. Clients currently using this feature should consider migrating to the OPL data source feature of IBM ILOG CPLEX Optimization Studio or using the .NET interfaces of CPLEX instead.
