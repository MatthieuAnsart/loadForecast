New file format: VMC

Documents a new file format in CPLEX V12.6.0 to support virtual machine configuration for distributed parallel optimization.

As announced in Distributed parallel MIP optimization in these release notes, CPLEX V12.6.0 can solve mixed integer programming (MIP) models in parallel in a distributed computing environment. To support this new feature, CPLEX introduces a new file format, VMC for virtual machine configuration. A file of data formatted by VMC conventions in XML specifies worker configurations for distributed parallel optimization with CPLEX. For samples of this file format, see these topics in the CPLEX User's Manual, where VMC is applied in applications of distributed parallel optimization:
  • Using Open MPI with distributed parallel MIP
  • Using MPICH with distributed parallel MIP
  • Using a process transport protocol with distributed parallel MIP
  • Using TCP/IP as the transport protocol with distributed parallel MIP
See also the reference documentation in VMC file format: virtual machine configuration for distributed parallel optimization in File Formats Supported by CPLEX.
To configure a cluster of workers for distributed parallel optimization, CPLEX introduces new routines and methods to read such a configuration from a formatted VMC file.
  • In the Callable Library (C API), use the routines:
    • CPXXreadcopyvmconfig and CPXreadcopyvmconfig to read a VMC file and install from it the virtual machine configuration,
    • CPXXcopyvmconfig and CPXcopyvmconfig to install a virtual machine configuration specified by a string,
    • CPXXdelvmconfig and CPXdelvmconfig to delete a VMC file and thus un-install that configuration.
  • In the C++ API, use the methods of the class IloCplex:
    • readVMConfig to read a VMC file and install that configuration,
    • copyVMConfig to read a configuration from a string and install it,
    • delVMConfig to delete and thus un-install a virtual machine configuration,
    • hasVMConfig to test whether a VMC has been loaded and installed.
  • In the Java API, use the methods of the class IloCplex:
    • readVMConfig to read a VMC file and install that configuration,
    • copyVMConfig to read a configuration from a string and install it,
    • delVMConfig to delete and thus un-install a virtual machine configuration,
    • hasVMConfig to test whether a VMC has been loaded and installed.
  • In the .NET API, use the methods of the class Cplex:
    • ReadVMConfig to read a VMC file and install that configuration,
    • CopyVMConfig to read a configuration from a string and install it,
    • DelVMConfig to delete and thus un-install a virtual machine configuration,
    • HasVMConfig to test whether a VMC has been loaded and installed.
  • In the MATLAB connector (Cplex class interface), use the new methods:
    • readVMConfig to read a VMC file and install that configuration,
    • copyVMConfig to read a virtual machine configuration from a string and install it,
    • delVMConfig to delete a virtual machine configuration and thus un-install that configuration.
