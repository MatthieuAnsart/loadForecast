Deprecated features removed:solution pool filtering

Features deprecated in a previous version have been removed from CPLEX V12.5.1.

These routines for solution pool filtering have been deprecated and removed from CPLEX V12.5.1:

  • CPXgetsolnpoolfilter
  • CPXgetsolnpoolfilterindex
  • CPXgetsolnpoolfiltername
  • CPXXgetsolnpoolfilter
  • CPXXgetsolnpoolfilterindex
  • CPXXgetsolnpoolfiltername

Remedy

Instead of those deprecated routines, which are no longer available, first query the filter type by means of the routine CPXXgetsolnpoolfiltertype or CPXgetsolnpoolfiltertype. Then consider the type of filter (diversity or range) specified by the query, and use the corresponding routine instead:

  • CPXgetsolnpooldivfilter
  • CPXgetsolnpoolrngfilter

Those two routines return details about the specified filter, such details as the index or name, in their arguments.
