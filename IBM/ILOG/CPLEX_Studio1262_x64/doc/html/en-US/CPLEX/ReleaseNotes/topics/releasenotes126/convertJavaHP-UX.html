For Java users of CPLEX V12.6.0 on HP-UX

Java users on HP-UX have reported internal Java crashes due to increasing stack size. Consequently, Java users of CPLEX on HP-UX may need to set the environment variable PTHREAD_DEFAULT_STACK_SIZE to avoid crashes of CPLEX applications written in Java on this platform. The value PTHREAD_DEFAULT_STACK_SIZE=1052672 has been tested with CPLEX on HP-UX with recent versions of Java to address this problem.
