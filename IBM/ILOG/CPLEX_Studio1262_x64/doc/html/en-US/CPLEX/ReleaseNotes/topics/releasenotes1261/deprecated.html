Deprecated

Documents deprecated elements in CPLEX V12.6.1.

These notes announce deprecated elements of CPLEX V12.6.1 and recommend replacements instead.

Deprecated in a previous release, removed from this release

These parameters of CPLEX were deprecated in a previous release (V10.0); they have been removed from CPLEX V12.6.1.

  • BasInterval, CPX_PARAM_BASINTERVAL Formerly, this parameter specified the frequency in number of iterations for CPLEX to write a backup basis to a file in XXX format. The XXX format is not supported. Consequently, this frequency setting is no longer needed. (CPLEX supports the industry standard for basis files, MPS BAS, documented in the topic BAS file format: advanced basis, the reference manual, File Formats Supported by CPLEX.)
  • ReverseInd, CPX_PARAM_REVERSEIND Formerly, this parameter allowed you to specify manually that CPLEX should reverse the byte order when CPLEX read SAV files. Now, CPLEX automatically detects byte order in data, so there is no longer a need for this manual intervention.
  • XXXInd, CPX_PARAM_XXXIND Formerly, this parameter enabled and disabled the reading of XXX files. The XXX format is not supported. Consequently, this manual switch is no longer needed. (CPLEX supports the industry standard for basis files, MPS BAS, documented in the topic BAS file format: advanced basis, the reference manual, File Formats Supported by CPLEX.)

Deprecated in the C++ API

Recall that the Callable Library (C API) routine CPXXputenv or CPXputenv can be invoked directly in a C++ application. This routine has been deprecated, and it will be removed in a future release. In order to set environment variables in your C++ application, use conventional system calls instead.

Deprecated in the Java API

The method IloCplex.putenv has been deprecated. It will be removed in a future release. In order to set environment variables in your Java application, use conventional system calls instead.

Deprecated in the .NET API

The method Cplex.Putenv of the .NET API has been deprecated and will be removed in a future release. In order to set environment variables in your .NET application, use conventional system calls instead.

Deprecated in the Callable Library (C API)

The routines CPXXputenv, CPXputenv of the Callable Library (C API) have been deprecated. They will be removed in a future release. In order to set environment variables in your application, use conventional system calls instead.
