New features for all users

Documents new features of CPLEX V12.6.0

These notes highlight new features of interest to users of CPLEX V12.6.0.

New names of parameters

New names of all existing parameters are available in CPLEX V12.6.0. These new names are organized hierarchically, for greater ease of use. The reference manual of CPLEX Parameters documents both the new name and the existing, previous name of each parameter in each application programming interface (API) and in each component, such as the Interactive Optimizer or MATLAB connector. See the announcement New parameters for all users in these release notes for more detail.

Solving nonconvex QP and MIQP

CPLEX now finds global solutions of nonconvex quadratic programs (that is, nonconvex QPs) and nonconvex mixed integer quadratic programs (that is, nonconvex MIQPs). In other words, CPLEX solves the minimization of problems with a quadratic objective function and linear constraints in which the objective function is not necessarily convex; or equivalently, CPLEX solves the maximization of a QP or MIQP in which the objective function is not necessarily concave.

Such a problem of linear constraints with possibly one or more quadratic terms in the objective function is also sometimes known as an indefinite QP. Likewise, a problem of linear constraints where some variables are also constrained to be integer and with possibly one or more quadratic terms in the objective function is also sometimes known as an indefinite MIQP.

For an introduction to this new feature, see the announcement Global solution of nonconvex MIQP in these release notes.

For more detail about this new feature, see also the new topic Solving mixed integer programming problems with quadratic terms in the CPLEX User's Manual.

Solving a distributed MIP in parallel

CPLEX now offers more support for the solution in parallel of mixed integer programs (MIPs) in a distributed computing environment. This new feature, known as distributed parallel MIP optimization, is a mode of running CPLEX that harnesses the power of multiple computers or of multiple nodes inside a single computer to achieve better performance on some MIP problems. For an introduction to this new feature, see the announcement Distributed parallel MIP optimization in these release notes, and for a fuller description with examples, see the topic Solving a MIP with distributed parallel optimization in the CPLEX User's Manual.
