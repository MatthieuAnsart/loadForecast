Deprecated properties in the .NET API of CPLEX V12.6.0

For users of the .NET API of CPLEX V12.6.0, these properties have been deprecated.

These two properties of the .NET application programming interface (API) of CPLEX V12.6.0 have been deprecated. They will be removed in a future release of the product. 

Cplex.IncumbentCallback.NodeData(Object data)
Cplex.ControlCallback.NodeData(Object data)

In your .NET applications of CPLEX, use the corresponding accessors (setters) instead to conform to .NET programming conventions (where properties do not raise exceptions, as those deprecated properties did).

Cplex.IncumbentCallback.SetNodeData(Object data)
Cplex.ControlCallback.SetNodeData(Object data)
