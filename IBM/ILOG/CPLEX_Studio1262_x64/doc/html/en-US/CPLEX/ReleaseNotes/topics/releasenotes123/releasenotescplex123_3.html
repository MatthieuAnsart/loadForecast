Conversion notes for all users

Users of prior versions must first apply the conversion notes accompanying previous versions of this product before upgrading to this new version.

  • Changes in existing parameters
    Describes changes in existing parameters of interest to all users.
  • Simplified threading and parallel search
    Describes improvements in user’s control of threads, parallel search, and callbacks.
  • Changes in callbacks
    Describes major changes in callbacks.
  • String comparisons: impact of encoding on existing applications
    Suggests ways to evaluate impact of new encoding parameters on existing applications.
  • Error codes added
    Lists error codes new in this release.
  • Error-message format-string changed
    Documents changes in the strings that format error messages.
  • RMP file format deprecated
    Suggests alternative to deprecated RMP file format.
  • Changed in Callable Library (C API)
    Describes change in behavior of C routines accessing parameters.
  • Deprecated in the Callable Library (C API)
    Lists deprecated routines of the Callable Library and recommends replacements.
  • Deprecated in the C++ API
    Lists deprecated macros and classes of the C++ API and recommends replacements.
  • Deprecated in the Java API
    Lists deprecated elements of the Java API of CPLEX and suggests replacements.
