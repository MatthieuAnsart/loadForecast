New features for users of Microsoft Solver Foundation (MSF)

Describes new features in CPLEX V12.3 for users of Microsoft Solver Foundation (MSF)

CPLEX V12.3 is compatible with Microsoft Solver Foundation (MSF) V3.0. Improvements in MSF V3.0 make it possible for users of CPLEX V12.3 to solve mixed integer quadratic programs (MIQP); that is, mixed integer programs with a quadratic term in the objective function. Because Microsoft Solver Foundation V3.0 requires .NET Framework V4.0, and .NET Framework V4.0 is part of Visual Studio 2010 (but not part of Visual Studio 2008), CPLEX V12.3 requires Visual Studio 2010.

For an introduction to using CPLEX with MSF, see the tutorial MSF tutorial for CPLEX users in Getting Started with CPLEX. For more information about solving MIQPs, see the topic Solving problems with a quadratic objective (QP) in the CPLEX User’s Manual.
