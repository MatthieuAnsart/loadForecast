New parameters

Introduces new parameters, with links to further documentation.

These release notes introduce new parameters to manage deterministic time in a greater variety of circumstances: in probing a model, in polishing a solution, in tuning a model or a suite of models. A new parameter to manage the random seed that CPLEX uses for certain internal operations is also available. More detail about each of these new parameters is available in the IBM ILOG CPLEX Parameters Reference Manual.
