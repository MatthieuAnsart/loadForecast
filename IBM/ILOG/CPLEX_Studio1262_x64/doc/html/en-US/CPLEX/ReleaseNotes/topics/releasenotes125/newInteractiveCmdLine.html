New features of the Interactive Optimizer

Highlights new features of the Interactive Optimizer

You can now direct the Interactive Optimizer to read a file of commands and execute them sequentially as if you had entered the commands individually from the keyboard. To use this command-line option, supply the name of the file when you invoke the Interactive Optimizer, like this: 

cplex -f filename

This new option is documented in greater detail in the topic Using a file of Interactive Optimizer commands from your operating system in the manual of Commands of the Interactive Optimizer.

In fact, if you enclose each of the commands separately in double quotation marks, you are not obliged to collect them in a file. You can simply pass a series of commands (each enclosed in double quotation marks) to the Interactive Optimizer on the command line of your operating system, like this example:

cplex -c "read a_problem_name.lp" "opt" "disp sol var -" "quit"

In that example, a_problem_name represents the name of a file containing formatted input, opt is the acceptable abbreviation of the command to optimize the input, and disp sol var - tells CPLEX to display the variables of the solution that it finds.
