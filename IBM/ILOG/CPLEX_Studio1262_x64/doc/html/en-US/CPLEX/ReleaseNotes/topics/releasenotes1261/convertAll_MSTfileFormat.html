MST file format

Explains changes introduced by CPLEX V12.6.1 that apply to users of the MST file format.

For reading and writing MIP start files, CPLEX supports the MST file format based on XML. CPLEX provides an appropriate style sheet and schema or DTD for this format.

In versions prior to CPLEX V10, an MST file format not based on XML was supported. However, that file format was deprecated in CPLEX V10. That deprecated MST file format has been removed from CPLEX V12.6.1.

If this removal of the deprecated file format poses a problem for your CPLEX application, contact IBM ILOG CPLEX Optimization Studio support for your product.
