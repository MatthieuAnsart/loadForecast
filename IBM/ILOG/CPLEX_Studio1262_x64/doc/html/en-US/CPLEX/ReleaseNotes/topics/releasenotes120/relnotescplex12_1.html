Release notes for CPLEX V12.0

Thank you for installing CPLEX V12.0. Before using it, please review these notes highlighting new features in this version.

  • Welcome to CPLEX V12.0
    Describes what to expect in this version.
  • Conversion notes for all users: parallel optimizers enabled by default
    Explains the new policy making parallel optimizers available by default.
  • Conversion notes for all users: port changes
    Lists additional ports available in this release and ports removed from this release.
  • Conversion notes for Concert Technology users: deprecated methods
    Lists methods deprecated in CPLEX V12.0 and recommends replacements for them.
  • New feature: CPLEX connector for The MathWorks MATLAB
    Describes the new connector of CPLEX to this engineering software.
  • New feature: CPLEX Python API
    Describes the new fully-supported Python API for CPLEX.
  • New feature: multi-commodity flow (MCF) cuts
    Describes a new type of cut available for problems with network flow structure.
  • New feature for Java or .NET users: piecewise linear numeric expression
    Introduces a method for creating a piecewise linear numeric expression in the Java and .NET APIs.
  • New feature: deterministic parallel barrier algorithm
    Announces that deterministic mode for the parallel barrier optimizer is available.
  • New feature: parallel algorithms available to all users
    Announces technical aspects of a change in the licensing policy for parallel applications.
