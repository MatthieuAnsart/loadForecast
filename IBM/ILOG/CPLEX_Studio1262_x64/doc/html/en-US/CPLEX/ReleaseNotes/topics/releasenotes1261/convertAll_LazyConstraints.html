Lazy constraints

Announces changes introduced by CPLEX V12.6.1 that apply to users of lazy constraints.

CPLEX V12.6.1.0 offers an improved interface for lazy constraints. For more detail, see the conversion note or announcement of new features specific to the application programming interface (API) of interest to you.

  • Callable Library (C API)
    • New features of the Callable Library (C API) especially the topic "Lazy constraint callback source" which introduces new values for CPXXgetcallbacknodeinfo and CPXgetcallbacknodeinfo.
    • Conversion notes for users of the Callable Library (C API) especially the topic "CPXXpreslvwrite: changed behavior" with respect to lazy constraints and lazy constraint callbacks.
  • C++ API Lazy constraint callback source
  • Java API Lazy constraint callback source
  • .NET API Lazy constraint callback source
