New Parameters for all users

Documents new parameters of CPLEX V12.5.1.

These notes highlight new parameters available in CPLEX V12.5.1.

Lift-and-project cuts

A new parameter, Lift-and-project cuts switch for MIP and MIQCP, enables lift-and-project cuts. The new parameter (LiftProj, CPX_PARAM_LANDPCUTS) is available in all application programming interfaces (APIs) of CPLEX V12.5.1, as well as the Interactive Optimizer. Lift-and-project cuts are defined briefly in the topic Lift-and-project cuts in the CPLEX User's Manual. That topic also offers bibliographic citations for further reading about lift-and-project cuts.

Calculating dual values for QCP and impact on presolve

A new parameter, calculate QCP dual values, enables a user to control whether CPLEX calculates dual values for a quadratically constrained program (QCP) and whether this calculation impacts presolve reductions. The new parameter (CalcQCPDuals, CPX_PARAM_CALCQCPDUALS) is available in all application programming interfaces (APIs) of CPLEX V12.5.1, as well as the Interactive Optimizer. New symbols appropriate to each API support the possible values of this parameter.
