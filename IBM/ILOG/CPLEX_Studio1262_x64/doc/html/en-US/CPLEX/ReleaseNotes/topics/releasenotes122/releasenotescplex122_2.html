Announcements

Highlights news about the product.

CPLEX now offers an interface to the Microsoft Solver Foundation. For more information about that new feature, see the note MSF interface and the tutorial in Getting Started, MSF tutorial for CPLEX users.

MIP kappa describes a new feature extending the computation of kappa to mixed integer programming (MIP). (In linear programming, kappa is also known as the condition number of a solution.)

Intel processors prior to Pentium 4 are no longer supported. For a complete list of machine types and library formats (including version numbers of compilers and JDKs) see the Detailed System Requirements (DSR) on the IBM customer support site.
