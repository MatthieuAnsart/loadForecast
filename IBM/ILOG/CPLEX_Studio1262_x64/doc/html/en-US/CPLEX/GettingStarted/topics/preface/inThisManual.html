What’s in this manual

Getting Started with CPLEX offers tutorials for the components of CPLEX, including the Interactive Optimizer and the application programming interfaces.

Setting up CPLEX tells how to install CPLEX.

Solving an LP with CPLEX shows you at a glance how to use the Interactive Optimizer and each of the application programming interfaces (APIs): C++, Java, .NET, and C. This overview is followed by more detailed tutorials about each interface.

Interactive Optimizer tutorial explains, step by step, how to use the Interactive Optimizer: how to start it, how to enter problems and data, how to read and save files, how to modify objective functions and constraints, and how to display solutions and analytical information.

C++ tutorial describes the same activities using the classes in the C++ implementation of the CPLEX Concert Technology Library.

Java tutorial describes the same activities using the classes in the Java implementation of the CPLEX Concert Technology Library.

.NET tutorial describes the same activities using .NET facilities.

Callable Library tutorial, describes the same activities using the routines in the CPLEX Callable Library.

Python tutorial covers certain installation considerations plus an introduction to using the CPLEX Python API to model and solve optimization problems.

All tutorials use examples that are delivered with the standard distribution.
