Using a file of Interactive Optimizer commands from your operating system

The Interactive Optimizer accepts a file from your operating system.

You can create a file of Interactive Optimizer commands to use from your operating system. CPLEX accepts this file and executes the commands in a session of the Interactive Optimizer. This strategy can be helpful on an operating system, such as IBM Blue Gene, where it is not appropriate to redirect one command to another. The strategy is also available on other operating systems, such as Microsoft Windows, GNU/Linux, or IBM z/OS.

To use this feature of the Interactive Optimizer, first create the file as you normally create a new file on your operating system. Put each Interactive Optimizer command on a separate line in the file, as you would type commands to enter in the Interactive Optimizer.

To invoke your file of commands, use the following syntax at the command line of your operating system, where cplex is the command to invoke the Interactive Optimizer, -f is the option to specify a filename, and filename is the name of your file:

cplex -f filename

If the name of the file containing commands includes spaces, be sure to enclose the name of the file in double quotation marks when you pass it to the -f option. This technique of quoting file names that include spaces works for the command-line option; however, the read command inside the Interactive Optimizer does not recognize file names including spaces, even when the file names are quoted.

In fact, if you enclose each of the commands separately in double quotation marks, you are not obliged to collect them in a file. You can simply pass a series of commands (each enclosed in double quotation marks) to the Interactive Optimizer on the command line of your operating system, like this example:

cplex -c "read a_problem_name.lp" "opt" "disp sol var -" "quit"

In that example, a_problem_name represents the name of a file containing a formatted model and data, opt is the acceptable abbreviation of the command to optimize the input, and disp sol var - tells CPLEX to display the variables of the solution.
