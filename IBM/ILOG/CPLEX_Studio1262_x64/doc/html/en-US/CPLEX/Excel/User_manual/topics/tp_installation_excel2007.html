Registering CPLEX for Microsoft Excel as an add-in with Excel 2007

Presents the procedure for registering the IBM ILOG CPLEX for Microsoft Excel XLL in Excel 2007.

About this task

Before you can use CPLEX for Microsoft Excel to solve problems, you must first register the XLL (Excel Add-In) file that contains the CPLEX solver as an add-in with Microsoft Excel. This section contains the procedure for doing this with Excel 2007.

Note: CPLEX for Microsoft Excel is supported for Excel 2007 and Excel 2010. For details, please see the detailed system requirements http://www-01.ibm.com/support/docview.wss?uid=swg27019100.

Procedure

  1. Locate the file cplexNNN.xll (where NNN specifies the version number) on your computer.

    By default this file is installed in yourCPLEXhome\excel.

    Note:

    We recommend that you do not move this XLL file from its default location, because there are several “support” files in the same directory that are required to make context-sensitive Help function properly.

  2. Open Excel 2007. Click the round Office button in the upper left of the Excel window.
  3. In the pop-up menu, click the Excel Options button at the bottom right of the window.
  4. When the Excel Options window appears, click Add-Ins in the left pane to display that wizard in the right pane.
  5. At the bottom left of the Add-Ins pane, choose Excel Add-Ins from the Manage drop-down list and click the Go button beside it.
  6. In the popup window, click Browse and navigate to the location of the cplexNNN.xll file, select it, and click OK.
  7. When you return to the Add-Ins pane of the Excel Options window, verify that IBM ILOG CPLEX VN.N Excel Add-in has been added to the list of available add-ins.
  8. Finally, click OK to close the Excel Options window and return to Excel 2007.

    In Excel, click the Add-Ins tab in the Ribbon Bar to verify that a CPLEX menu is displayed below it.

Results

At this point, the XLL for IBM ILOG CPLEX for Microsoft Excel is registered as an add-in of Excel 2007.
