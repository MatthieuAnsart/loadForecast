Accessing help and documentation in CPLEX for Microsoft Excel

How to view the online help in CPLEX for Microsoft Excel.

Procedure

In Excel, press the Help button on any of the IBM ILOG CPLEX for Microsoft Excel popup windows.

Results

The IBM ILOG CPLEX for Microsoft Excel Online Help system will open in its own window, displaying context-sensitive Help for the window you are on.

Note:

In Excel 2007, you can navigate to other portions of the Help using the table of contents in the left panel.

To be able to see the entire contents of the IBM ILOG CPLEX for Microsoft Excel Online Help system, launch the <cplexinstalldir>\excel\cplex121.chm file using Windows Explorer. You can use the Windows Start Menu command Programs > IBM ILOG > IBM ILOG CPLEX Vn.n > Content Folder command to navigate to this file (where n.n specifies the version number of your installation).
