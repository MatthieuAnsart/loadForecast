Overview

Presents an overview of how to start working with IBM ILOG CPLEX for Microsoft Excel.

There is no specific installation procedure for IBM ILOG CPLEX for Microsoft Excel. The program is contained in an XLL (Excel Add-In) file that is installed on your computer when you install CPLEX. You make the XLL available to Excel by registering it as an add-in program. The following section shows how to do this for Excel 2007 (as part of Microsoft Office 2007).

The XLL and all supporting files are located in the folder or directory yourCPLEXhome\excel where you install CPLEX.

Note: CPLEX for Microsoft Excel is supported for Excel 2007 and Excel 2010. For details, please see the detailed system requirements http://www-01.ibm.com/support/docview.wss?uid=swg27019100.
