Using CP Optimizer with C# or Java

This file first describes how to use Microsoft Visual Studio to:

  • build and run the C# .NET examples delivered with CP Optimizer.
  • create a project and link the target with CP Optimizer.

Then, this file describes how to build and run the Java examples.

CP Optimizer Installation Directory

Throughout this document, the CP Optimizer installation folder is referred to as <CPOPTDIR>.

For example, if CP Optimizer has been installed in the folder C:\Program Files\IBM\ILOG\CPLEX_Studio_1262\cpoptimizer (the default location), references to <CPOPTDIR>\bin\x86_win32 represent C:\Program Files\IBM\ILOG\CPLEX_Studio_1262\cpoptimizer\bin\x86_win32.

Libraries

The CP Optimizer .NET library is delivered as DLLs so it may be used with any of the library formats.  All the DLLs are located in <CPOPTDIR>\bin\x86_win32 or <CPOPTDIR>\bin\x64_win64. 

Building and Running CP Optimizer .NET Examples

The CP Optimizer .NET examples have all been gathered in a solution for C#.  The C++ libraries are delivered in several formats, and the .NET project and solution files, although independent of the C++ library formats, are located in each of  the library format folders.  For this discussion, the folder <CPOPTDIR>\examples\x86_windows_vs2012\stat_mda will be used.  Be aware that the order of the instructions below is important.

The following instructions are for using Microsoft Visual Studio 2012. The steps for using Microsoft Visual Studio 2010 and 2013 are similar.

  1. Start Microsoft Visual Studio 2012.
  2. From the File menu, choose Open Solution.
    The Open Solution dialog box appears.
    The default selection in the Files Of Type drop-down list is     Solution Files (.sln).
    • Select the folder <CPOPTDIR>\examples\x86_windows_vs2012\stat_mda.
    • Select the examples.cs.net.sln file and click Open.
  3. To build only one example (for instance, Ppp):
    • Select the Ppp project in the Solution Explorer window.
    • From the Build menu, choose Build Ppp.
      Wait for the completion of the building process.
  4. To build all of the examples:
    • From the Build menu, choose Build Solution
      Wait for the completion of the building process.
  5. To run an example (for instance, Ppp):
    • Start a command prompt window by running the Microsoft Visual Studio 2012 Command Prompt.
      In the window Microsoft Visual Studio 2012 Command Prompt:
      • Type <CPOPTDIR>\examples\x86_windows_vs2012\stat_mda\csbin\Ppp.exe. The result is then displayed.

Building Your Own Project which Links with CP Optimizer

Let's assume that you want to build a target named Test.exe and have:

  • a source file named Test.cs
  • a folder where this file is located and which, for the sake of simplicity, we'll refer to as <MYAPPDIR>.

One way to achieve that is to create a Solution named Test.sln as described here. Be aware that the order of instructions is important.

  1. Start Microsoft Visual Studio 2012.
  2. The first step is to build the Test.sln solution.

From the File menu, select New->, and then Project....

The New Project dialog box appears.

    • In the Project Types pane, select Visual C# Projects.
    • In the Templates pane, select the Empty Project icon.
    • Fill in the project name (Test).
    • If necessary, correct the location of the project (to <MYAPPDIR>)
    • Click OK  

This creates a solution, Test, with a single project, Test. You can see the contents of the solution by selecting Solution Explorer in the View menu.

  1. To add your source file to the project, from the Project menu, choose Add Existing Item...
    • Move to <MYAPPDIR> and select Test.cs.
    • Click Add.
  2. Next, you have to set some options so that the project knows where to find the CP Optimizer components.

From the  Project menu, choose  Add Reference....
The Add Reference dialog box appears. 

    • Browse to the folder <CPOPTDIR>\bin\x86_win32.
      • Select ILOG.CP.dll.
    • Click OK to close the dialog box.
  1. Next, you have to set the default project configuration.

From the Build menu, select Configuration Manager...

    • Select Release in the Active Solution Configuration drop-down list.
    • Click Close.
  1. Finally to build the project, from the Build menu, select Build Solution
      • After completion of the build process, the target is created. The full path of the Test.exe is <MYAPPDIR>Test\bin\Release\Test.exe .
  2. To run the example:
    • Start a command prompt window by running the Microsoft Visual Studio 2012 Command Prompt.
      In the window Microsoft Visual Studio 2012 Command Prompt:
      • Type <MYAPPDIR>Test\bin\Release\Test.exe.

 

Building and running Java examples

The Java examples can be built as follows:

First you have to set your windows environment variable JAVA_HOME to your Java directory.

The examples can be all compiled and executed simply by calling “ant” in directory <CPOPTDIR>\examples\<platform>\stat_mda

 

Building and running a new Java application

For compiling a new file, myExample.java, and after having defined the environment variable CLASSPATH, you can use the following command line:

 

 > javac myExample.java

 

Then the following command line runs the file myExample.java (your CLASSPATH environment variable must contain ".", the current directory, and the .jar file for CP Optimizer: <CPOPTDIR>\lib\ILOG.CP.jar

 

> java -Djava.library.path=<CPOPTDIR>\bin\x86_win32 myExample.java
