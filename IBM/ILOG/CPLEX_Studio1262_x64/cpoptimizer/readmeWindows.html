IBM ILOG CPLEX Optimization Studio 12.6.2 (CP Optimizer component)

This is the CP Optimizer 12.6.2 distribution, including Concert Technology for C++, Java, and .NET users. Please read the CP Optimizer 12.6.2 Release Notes for news on this version of CP Optimizer.

Directory Structure

The following sub-directories (and this file), are contained in the directory where you installed CP Optimizer 12.6.2.

In the cpoptimizer directory:

 bin\x86_win32:
       CP Optimizer DLLs for Java and .NET

 bin\x64_win64:
       CP Optimizer DLLs for Java and .NET

 examples\src\cpp:
       Source files for the C++ examples.

 examples\src\java:
       Source files for the Java examples.

 examples\src\csharp:
       Source files for the C# examples.

 examples\<compiler>\<libformat>:
       Visual Studio workspace and project files to support your particular platform.

 include\ilcp:
       Header files, such as cp.h and cpext.h,
       that are needed to compile C++ applications.

 lib:
       The ILOG.CP.jar file containing the Java classes 
       needed to use the Java version of CP Optimizer.

 lib\<compiler>\<libformat>:
       CP Optimizer library file for C++ users (cp.lib).

In the concert directory:

 include\ilconcert:
       Header files, such as ilomodel.h, 
       that are needed to compile C++ applications.

 lib\<machine>\<libformat>:
       Concert Technology libraries for C++ users (concert.lib).

Compiling and Executing examples

IMPORTANT: Before compiling or executing CP Optimizer examples, and before launching Visual Studio, you must add to your PATH variable the appropriate directories for finding DLLs (if you did not choose to do so at installation time). If CPLEX Optimization Studio was installed in C:\Program Files\IBM\ILOG\CPLEX_Studio_1262 (the default location), then you should add to the PATH variable C:\Program Files\IBM\ILOG\CPLEX_Studio_1262\cplex\bin\x86_win32 and C:\Program Files\IBM\ILOG\CPLEX_Studio_1262\cpoptimizer\bin\x86_win32 (for 32-bit Windows) or C:\Program Files\IBM\ILOG\CPLEX_Studio_1262\cplex\bin\x64_win64 and C:\Program Files\IBM\ILOG\CPLEX_Studio_1262\cpoptimizer\bin\x64_win64 (for 64-bit Windows). To do this, use Start > Control Panel > System > Advanced > Environment Variables.

To compile and execute the C++ examples, or to write a new project using CP Optimizer from C++, please consult the file cpp.html. To compile and execute the Java examples, compile and execute the C# examples, or to build a new application using Java or C#, please consult the file javadotnet.html.
